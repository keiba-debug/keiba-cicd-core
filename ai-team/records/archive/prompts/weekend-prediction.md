# 週末競馬予想プロンプトテンプレート

## 📊 土曜朝: 分析プロンプト

```
本日のレース分析を行います。

対象レース: [競馬場] [R番号] [レース名]
取得済みデータ: Z:/KEIBA-CICD/data/

分析項目:
1. 出走馬の過去成績分析
2. 調教評価（A/B/C判定）
3. 騎手の当該コース成績
4. 血統のコース適性
5. 展開予想（逃げ・先行・差し・追込）

期待値計算:
- 推定勝率の算出
- 現在オッズとの比較
- 期待値 = オッズ × 推定勝率

出力形式:
- 推奨馬（◎○▲）
- 期待値上位5頭
- 推奨買い目
```

## 🏇 レース直前: 最終確認プロンプト

```
レース[X]分前の最終確認

1. 最新オッズの確認
2. パドック情報の確認（可能なら）
3. 天候・馬場状態の確認

調整項目:
- オッズ変動による期待値再計算
- 馬場状態による適性再評価
- 最終的な買い目確定

記録項目:
- 予想内容
- 購入金額
- 期待回収額
```

## 📈 レース後: 結果記録プロンプト

```
レース結果の記録と分析

結果入力:
- 1着: [馬番] [馬名]
- 2着: [馬番] [馬名]
- 3着: [馬番] [馬名]

的中確認:
- 予想: [的中/不的中]
- 配当: [金額]
- 収支: [±金額]

1. 予想の的中/不的中要因
2. 見落とした要素

4. 次回への改善点

データ保存:
Z:/KEIBA-CICD/data/results/YYYY-MM-DD-[race_id].json
```
