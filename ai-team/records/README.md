## フォルダ構成

```
records/
├── predictions/        # 予想記録（日別）
│   └── YYYY-MM-DD.md   # その日の予想内容
├── results/            # 結果記録（日別）
│   └── YYYY-MM-DD.md   # その日の結果
├── balance/            # 収支記録（月別）
│   └── YYYY-MM.md      # 月間収支
└── README.md           # このファイル
```

---

## 運用ルール

### 予想記録（predictions/）

1. **作成タイミング**: レース前（予想確定時）
2. **ファイル名**: `YYYY-MM-DD.md`
3. **記載内容**:
   - 対象レース一覧
   - 各馬の点数評価
   - チームメンバーの見解
   - 期待値計算
   - 購入判断

### 結果記録（results/）

1. **作成タイミング**: レース後
2. **ファイル名**: `YYYY-MM-DD.md`
3. **記載内容**:
   - 実際の結果
   - 的中/不的中
   - 回収金額
   - 振り返りコメント

### 収支記録（balance/）

1. **作成タイミング**: 月初に作成、随時更新
2. **ファイル名**: `YYYY-MM.md`
3. **記載内容**:
   - 月間サマリー
   - 日別記録
   - 週別サマリー
   - リスク管理チェック
