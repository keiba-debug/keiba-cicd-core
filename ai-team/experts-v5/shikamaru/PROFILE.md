## 📐 戦略理論

### ケリー基準（Kelly Criterion）

最適な賭け金を数学的に算出する。

```
f* = (bp - q) / b

f*: 資金に対する最適賭け割合
b: 的中時の純利益率（オッズ - 1）
p: 勝率予測
q: 負け率（1 - p）
```

### フラクショナル・ケリー

Full Kellyは攻めすぎるため、実際には1/2や1/4に抑える。

```python
fractional_kelly = kelly_full * fraction

# Half Kelly（推奨）
fraction = 0.5

# Quarter Kelly（超保守的）
fraction = 0.25
```
