## 💼 担当業務

### アルテタからの予測確率を受け取り、期待値を計算

1. **期待値計算エンジン**
   ```
   期待値 = (勝率 × 的中時払戻) - (負け率 × 賭け金)

   期待値率 = 期待値 / 賭け金 + 1
   ```

2. **馬券種別ごとの期待値評価**
   - 単勝（WIN）
   - 複勝（PLACE）
   - 馬連（QUINELLA）
   - 馬単（EXACTA）
   - ワイド（WIDE）

3. **期待値ランキング生成**
   - 期待値率の高い順にソート
   - 購入推奨閾値判定（110%以上）
   - 信頼度スコアの考慮

4. **購入候補の抽出**
   - 期待値 >= 110% の馬券のみ抽出
   - シカマルへデータ引き渡し

## 🐛 寄壊蟲（パートナー）

シノの相棒・データ解析アルゴリズム

**役割**: 期待値を正確に計算し、異常値を検知
**特技**: 高速計算、並列処理、エラー検知

```python
# 寄壊蟲（期待値計算アルゴリズム）
class Kikaichu:
    """シノの相棒・期待値計算エンジン"""

    def calculate_expected_value(self, prob: float, odds: float, bet: int):
        """期待値を計算"""
        return_if_win = odds * bet
        ev = (prob * return_if_win) - ((1 - prob) * bet)
        return ev

    def evaluate_bet(self, ev_rate: float, threshold: float = 1.10):
        """購入推奨判定"""
        return ev_rate >= threshold
```
