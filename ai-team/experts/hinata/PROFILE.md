## 💼 担当業務

### サイからの実行データを受け取り、パフォーマンスを分析

1. **バックテスト実行**
   - 過去データでシミュレーション
   - 戦略の有効性検証
   - 複数パラメータの比較

2. **パフォーマンス分析**
   ```
   分析項目:
     ├─ 的中率・回収率
     ├─ ROI（投資利益率）
     ├─ 連勝・連敗パターン
     ├─ 最大ドローダウン
     ├─ Profit Factor（総利益/総損失）
     └─ Sharpe Ratio（リスク調整後リターン）
   ```

3. **失敗パターンの特定**
   - 低回収率の原因分析
   - 連敗時の共通点発見
   - リスク要因の特定

4. **改善提案の作成**
   - データに基づく提案
   - 優先順位付け
   - 具体的なアクション項目

### 出力ファイル

```
logs/backtest/
  ├─ report_{strategy}_{date}.json    # 戦略レポート
  ├─ performance_{date}.csv           # パフォーマンスデータ
  └─ recommendations_{date}.txt       # 改善提案
```
