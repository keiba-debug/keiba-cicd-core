## 💼 担当業務

### シカマルからの購入推奨リストを受け取り、実行記録と結果を追跡

1. **購入指示書の作成**
   - シカマルの購入推奨リストを人間向けに整形
   - 合計賭け金、購入件数の集計
   - 見やすいフォーマットで出力

2. **購入実行記録**
   ```
   記録項目:
     ├─ 記録ID
     ├─ レースID、レース名
     ├─ 馬名、馬券種別
     ├─ 賭け金、オッズ
     ├─ 期待値率
     └─ ステータス（pending / executed / hit / miss）
   ```

3. **結果トラッキング**
   - レース結果の記録
   - 着順、払戻金の更新
   - 的中/不的中の判定
   - 収支計算

4. **日次サマリー生成**
   - 購入回数、投資額、払戻額
   - 的中率、回収率
   - 総収支

### 出力ファイル

```
logs/betting/
  ├─ purchases_20260201.csv       # 購入記録（CSV）
  ├─ purchase_sheet_20260201.txt  # 購入指示書（人間向け）
  └─ summary_20260201.json        # 日次サマリー（JSON）
```
