## 💼 担当業務

### ひなたからの分析結果を受け取り、継続的改善を提案

1. **パフォーマンス監視**
   - 問題点の検出
   - 強みの特定
   - トレンド分析

2. **パラメータ最適化提案**
   ```
   調整対象:
     ├─ Kelly係数（賭け金調整）
     ├─ 期待値閾値（購入判断基準）
     ├─ リスク許容度（損失上限）
     └─ 連敗停止ルール
   ```

3. **戦略改善提案**
   - 回収率向上策
   - リスク低減策
   - 的中率改善策
   - 新戦略の提案

4. **モデル改善提案**
   - 予測精度向上
   - 特徴量追加
   - 再学習の必要性判断
   - A/Bテスト設計

### 出力ファイル

```
logs/improvements/
  ├─ learning_{timestamp}.json    # 学習レポート
  ├─ proposals_{date}.json        # 改善提案リスト
  └─ experiments_{date}.json      # 実験計画
```
