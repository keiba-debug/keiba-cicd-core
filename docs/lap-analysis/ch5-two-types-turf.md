# 第5章 分析: 競走馬は2つのタイプに分けられる（芝編）

> 出典: 『永遠の初心者を脱する！ための競馬思考 第1巻』（まく著, 2023改訂版）
> 分析日: 2026-02-20
> 目的: KeibaCICD のラップ分析v2設計のための知識ベース

---

## 1. 核心理論: 2つの適性

全ての芝競走馬は本質的に **2つの適性タイプ**（+ 万能型）に分類できる。

| タイプ | 定義 | 求められる能力 |
|--------|------|-------------|
| **瞬発戦タイプ** | スローペース・高速馬場・長い最終直線 → ラスト3Fの加速力勝負 | トップスピード、加速力 |
| **持続戦タイプ** | ハイペース・タフな馬場・短い直線 → 脚を長く使う勝負 | スタミナ、パテない脚、加速力よりも持続力 |
| **万能タイプ** | どちらにも対応できる（圧倒的能力 or 両方の適性を持つ） | 例: ディープインパクト、オルフェーヴル |

**著者の主張**: 「この分類方法なくして芝の重賞は予想できない」

---

## 2. 瞬発戦と持続戦の定義

### 瞬発戦
> スローペースや仕掛けの遅さによって、速い上がり3ハロンが要求されるレースのこと。

**特徴**:
- レース上がり3Fが速い（レベルによるが34秒台〜35秒台前半）
- ラスト3ハロン付近から急加速
- 「速い上がりを使えない馬」の出番がない

### 持続戦（3つのサブタイプ）

| サブタイプ | 定義 | 特徴 |
|-----------|------|------|
| **ロンスパ型** | ペースアップが早く（ラスト4〜5F）、長い脚が必要 | 中盤からじわじわ加速、持続力勝負 |
| **ハイペース型** | 前半からハイペースで大きくバテない脚が必要 | 前傾ラップ、消耗戦 |
| **道悪型** | 道悪馬場で相対的に馬場の重さを苦にしない脚が必要 | 全体時計がかかる、スタミナ消耗 |

**共通点**: どの持続戦の型においても「**脚を持続させる必要がある**」

---

## 3. 判定方法

### 方法1: レース上がり3F（主要指標）

| 条件 | 判定 |
|------|------|
| レース上がり ≤ 34.4秒 | **瞬発戦**（確定） |
| レース上がり ≤ 35.5秒（1800m以上） | **瞬発戦**（本書の閾値） |
| レース上がり ≥ 35.5秒（1800m以上） | **持続戦** |

**例外・注意**:
- 東京・新潟外回り: 持続戦の目安は「レース上がり **35.1秒** 以上」（高速馬場補正）
- **1600m以下は上がりだけでは判定不十分**: 道中ペースが速くても上がりが速くなりやすい
- 上級条件ほど未勝利の方が多少上がりが遅くても判別可能

### 方法2: RPCI（レースペースチェンジ指数）

> RPCI = レース上がり3F / (前半3F + 上がり3F) × 100

- TARGET の PCI は馬ごとの通過タイムベース
- RPCI はレースラップベース（先頭を走った馬が基準）

**持続戦の目安（RPCI）**:

| コース | 距離 | 持続戦目安 |
|--------|------|---------|
| 東京・新潟外回り | 1600m | RPCI ≤ **50** |
| その他全コース | 1600m | RPCI ≤ **48** |
| 全コース | 1200m | RPCI ≤ **45** |
| 全コース | その他 | おおむね **45以下** |

### 方法3: 補助判定ルール（境界レース用）

レース上がりやRPCIだけでは判定困難な場合、以下2条件で瞬発戦寄りと判断:

1. **ラスト4ハロン区間で0.5秒以上の加速がある**
2. **ラスト1ハロンでの減速が少ない**

> 注意: 中山・阪神は急坂があるためラスト1Fは構造的に減速しやすい。
> 福島も坂の影響あり。全体的に時計がかかるレースではラスト1Fの減速は目立たない。

### 方法4: 大逃げがいるレースの判定

大逃げ馬がいるとレースラップが参考にならない。

**代替手法**: 上位馬の上がり3Fを、各馬の「過去同条件での上がり3F」と比較
- 上位馬の上がりが普段より遅い → 持続戦
- 上位馬の上がりが普段通り → 瞬発戦

> この手法は下級条件（未勝利・1勝クラス）でも活用可能。
> 各馬の過去PCIと比較する方法も同様に有効。

## 4. 1600m以下の特殊性

マイル以下の上級条件では:
- スタートからずっと速いラップを刻んでも前が止まらない現象が起きる
- レース上がり3Fだけでは瞬発戦か持続戦か判定できない場合がある

**具体例**: 2022年ヴィクトリアマイル
```
ラップ: 12.5-10.8-11.4-11.6-11.7-11.1-11.3-11.8
ペース: 12.5-23.3-34.7 -46.3-58.0-1.09.1-1.20.4-1.32.2  (1:32.2  34.7-34.2)
```
- 初めの1Fを除いて全て11秒台 → 持続戦とも言える
- しかし著者は**瞬発戦**と判定:
  - 理由: 「ラスト4F区間で0.5秒以上の加速」+「ラスト1Fの減速が少ない」

## 5. 具体例による分類

### 瞬発戦の例: 2022年安田記念（ソングライン）
```
ラップ: 12.2-11.0-11.5-12.0-12.0-11.2-11.0-11.4    RPCI: 54.8
ペース: 12.2-23.2-34.7 -46.7-58.7-1.09.9-1.20.9-1.32.3  (1:32.3  34.7-33.6)
```
- 4-5ハロン目が12.0秒と大きく緩む → スローの瞬発戦

### 持続戦（ロンスパ型）の例: 2021年ジャパンカップ（コントレイル）
```
ラップ: 12.7-11.5-12.8-12.6-12.6-12.3-11.6-11.6-11.7-11.6-11.5-12.2
ペース: 12.7-24.2-37.0-49.6-1.02.2-1.14.5-1.26.1-1.37.7-1.49.4-2.01.0-2.12.5-2.24.7
        (2:24.7  37.0-35.3)
```
- 6ハロン目から7ハロン目で0.7秒ペースアップ → 「ラスト5ハロンのロンスパ」
- レース上がり35.3秒 → 持続戦の目安に当てはまる

### 持続戦（ハイペース型）の例: 2022年宝塚記念（タイトルホルダー）
```
ラップ: 12.5-10.4-11.0-12.1-11.6-12.1-11.9-11.8-11.9-12.0-12.4
ペース: 12.5-22.9-33.9-46.0-57.6 -1.09.7-1.21.6-1.33.4-1.45.3-1.57.3-2.09.7
        (2:09.7  33.9-36.3)
```
- 前3F 33.9秒 = 1200m並みの超ハイペース
- 1000m通過57.6秒 → 超ハイペース
- ペースアップ不可能 → 後半はどれだけ減速しないかの勝負

### 持続戦（道悪型）の例: 2019年ジャパンカップ（スワーヴリチャード）
```
ラップ: 12.8-11.3-12.3-12.2-11.7-12.2-12.1-12.0-12.1-12.4-12.6
ペース: 12.8-24.1-36.4-48.6-1.00.3 -1.12.5-1.24.6-1.36.6-1.48.7-2.01.1-2.13.3-2.25.9
        (2:25.9  36.4-37.2)
```
- 後半部の時計がかかるのが特徴（上がり37.2秒）
- 前半はそこまで遅くない → 「ロンスパ型」「ハイペース型」とも重複しにくい純粋な道悪型

## 6. 馬の適性分類の実践方法

### 手順
1. 各レースを瞬発戦/持続戦に分類
2. 各馬の過去レースを「好走」「凡走」に分類
3. どちらのタイプで好走/凡走が偏るかを分析
4. **瞬発戦タイプ** / **持続戦タイプ** / **万能タイプ** に分類

### 2020年マイルチャンピオンシップの分類例

| 着順(人気) | 馬名 | 適性 | 根拠 |
|-----------|------|------|------|
| 1 (1) | グランアレグリア | **瞬発** | 32秒台の上がりを繰り出せる超一流の瞬発力 |
| 2 (3) | インディチャンプ | **万能** | 安田記念+マイルCS両勝ち、どちらかといえば瞬発寄り |
| 3 (5) | アドマイヤマーズ | **瞬発** | ダイワメジャー産駒の中では瞬発寄り |
| 5 (2) | サリオス | **持続** | ハイペースの朝日杯を先行押し切り、毎日王冠レコード勝ち |
| 6 (8) | ヴァンドギャルド | **持続** | 条件戦・富士Sで上がりのかかる展開で勝ち上がり |
| 8 (4) | レシステンシア | **持続** | 阪神JF逃げ切り、35.2秒の上がり最速がこの馬の本質 |

**ポイント**: レースが瞬発戦になれば瞬発戦タイプがセットで来る場合が非常に多い

## 7. 瞬発戦/持続戦を決定づける要因（5+1）

| 要因 | 持続戦寄り | 瞬発戦寄り |
|------|---------|---------|
| 初角までの距離 | 長い | 短い |
| 最終直線の長さ | 短い | 長い |
| コーナーの角度 | 急カーブ | 緩いカーブ |
| 坂の有無 | 上り坂あり | 平坦 |
| 馬場状態 | 重い・タフ | 良い・高速 |
| 出走頭数 | 多頭数 | 少頭数 |

## 8. 芝の重賞で特に有効な理由

1. **上級条件ほど「適性 > 能力」**: クラスが上がるほど得意条件と苦手条件の差が明確化
2. **G1は「適性 > 能力」が最たる構図**: 馬齢戦・定量戦が多くフェアな条件
3. **適性の偏った馬が多い**: オープンクラスでは極端な適性の馬が生き残る
4. **回収率を重視するなら「適性」以外のファクターの入り込む余地が少ないG1が有利**

**階級別の傾向**:
- G1 > G2 = G3 の順に適性が重要
- G3は荒れやすい（ハンデ戦・別定戦が多く、予想ファクターが多様化）

---

## 9. 道悪の巧拙

- **持続戦タイプ → 道悪に強い傾向**
- **瞬発戦タイプ → 道悪に弱い傾向**（ただし例外あり）
- 走法: ピッチ走法（小股走り）→ 道悪得意 / ストライド走法（大股走り）→ 道悪苦手
- 蹄: 立蹄 → ベタ踏（苦手）/ 寝蹄 → ねつなぎ（苦手）
- 血統面: ステイゴールド産駒 = タフなレースに強い / ディープインパクト産駒 = 瞬発力特化だが加齢で持続戦にシフトする個体も

## 11. 開発への示唆

### 現状の問題点
- 現在の `race_trend` は RPCI閾値のみで5分類 → **上がり3Fを使っていない**
- 持続戦のサブタイプ（ロンスパ/ハイペース/道悪）を区別していない
- ラスト4F加速・ラスト1F減速の補助判定がない
- 馬ごとの適性分類を行っていない

### 実装すべき改善

#### A. レース分類の改善
1. **主判定**: レース上がり3F閾値（距離・コース別）
2. **副判定**: RPCI（コース別閾値）
3. **補助判定**: ラスト4F加速（0.5秒以上）+ ラスト1F減速パターン
4. **持続戦サブタイプ**: ロンスパ/ハイペース/道悪 の3分類
5. **大逃げ検出**: 先頭馬と2番手以降のギャップ → 代替判定法への切り替え

#### B. 馬の適性分類
1. 過去レースの瞬発戦/持続戦成績を集計
2. 各タイプでの好走率・着順平均を算出
3. 瞬発戦タイプ / 持続戦タイプ / 万能タイプ に分類
4. 信頼度（サンプル数）付き

#### C. ML特徴量の追加候補
- `race_type_match`: 当該レースの予想タイプと馬の適性のマッチ度
- `horse_shunpatsu_rate`: 瞬発戦での好走率
- `horse_jizoku_rate`: 持続戦での好走率
- `race_predicted_type`: コース/距離/馬場/出走頭数から予測されるレースタイプ

#### D. UIへの反映
- 馬詳細ページ: 過去レースに「瞬/持」列追加 + ペース傾向バッジ
- レース結果ページ: ラップチャートに瞬発戦/持続戦の判定結果表示
- 予測ページ: 適性マッチ度を表示

## 12. 要注意ポイント

1. **1600m以下はラップ形状の補助判定が必須**: 上がり3FだけではNG
2. **大逃げレースは馬別上がりで代替判定**: レースラップが信頼できない
3. **馬場状態は持続戦度に大きく影響**: 道悪は持続戦度を大幅に上げる
4. **血統・走法も適性に関連**: ただし本システムでは血統情報をまだ特徴量化していない
5. **「瞬発戦タイプの馬は道悪で危険視」が実戦で有効**: 人気馬の凡走予測に直結
