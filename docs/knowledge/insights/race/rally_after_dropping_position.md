# 位置取りを下げてから盛り返す馬 — 戦意と充実度の指標

> 起点: 「位置取り下げながら盛り返した馬は、戦意喪失せずに盛り返すことが充実してる証拠ではないか」

---

## 1. この着眼点が鋭い理由

```
通常、レース中に位置取りが下がる（通過順位が悪化する）馬は:
  - スタミナ切れで失速している
  - 不利を受けてポジションを失った
  - 折り合いを欠いて引っ掛かった後に止まった
  → いずれも「ネガティブな理由」

しかし、位置取りが下がった後に「盛り返す」馬は:
  - 下がったにもかかわらず、再度加速できた
  - 戦意を失わずに最後まで走り切った
  - 「もう一脚」使えるスタミナと精神力がある

→ 一度下がってから盛り返す = 単なる「差し馬」とは異なる現象。
  差し馬は最初から後方にいて直線で伸びる。
  盛り返し馬は一度ポジションを「失ってから」再加速する。

★ この「盛り返し」ができること自体が:
  (a) 身体的に余力がある証拠
  (b) 精神的に戦意を維持している証拠
  (c) 次走でも好走する可能性が高い「充実のサイン」
  ではないか？という仮説。

→ これは非常に説得力のある仮説。検証する価値が高い。
```

## 2. 「盛り返し」のパターン分類

### パターンA: 中盤失速 → 直線盛り返し（最も典型的）

```
通過順位の例: 3-3-8-5 → 着順4着

解釈:
  1-2角: 3番手で好位追走
  3角: 8番手に後退（何かあった or ペースについていけなかった）
  4角: 5番手に盛り返し
  ゴール: 4着

考えられる原因:
  ① 中盤でペースが上がり一時的についていけなかった
     → しかし直線で再加速できた = スタミナに余力
  ② 不利を受けてポジションを落とした
     → しかし立て直して盛り返した = 戦意旺盛
  ③ 前が壁になり一旦下げて外に出した
     → 騎手の判断で下げた後にスパート = 実力は高い

充実度の判断:
  3角で下がった後に4角〜ゴールで盛り返している
  → 「最後まで諦めていない」= 充実の証拠
  → 次走に期待できる走り
```

### パターンB: 序盤好位 → 中盤後退 → ゴール前粘る

```
通過順位の例: 2-2-6-6 → 着順5着

解釈:
  1-2角: 2番手で先行
  3-4角: 6番手に後退
  ゴール: 5着（6番手から巻き返してはいないが、それ以上下がっていない）

これは「盛り返し」というより「粘り」:
  完全に止まっておらず、下がるスピードが遅い
  → スタミナはギリギリだが、精神力で粘っている
  → 距離短縮や負担重量減で次走好走の可能性

充実度の判断:
  完全に止まる馬（2-2-10-15）とは根本的に異なる。
  粘れていること自体が「戦意がある」証拠。
```

### パターンC: 不利後の盛り返し（最も価値が高い）

```
通過順位の例: 5-5-10-6 → 着順3着

解釈:
  1-2角: 5番手で好位追走
  3角: 10番手に急落（不利発生？出遅れではないので中盤のトラブル）
  4角: 6番手に盛り返し
  ゴール: 3着（さらに盛り返し）

考えられる原因:
  3角での急落 = 何らかの不利（接触、前が壁、進路が塞がれた等）
  その後の盛り返し = 実力が高い + 戦意旺盛

充実度の判断: 最も高い。
  不利を受けながら3着まで盛り返している。
  → 不利がなければ1-2着の可能性が高い。
  → 次走で実力通り走れれば勝ち負け。
  → [前走不利検出](smooth_trip_without_simulation.md) との組合せで
    強力な巻き返し候補の検出が可能。
```

### パターンD: 意図的な下げ→仕掛け（騎手の作戦）

```
通過順位の例: 6-6-9-4 → 着順2着

解釈:
  1-2角: 6番手で追走
  3角: 9番手に後退（意図的に下げた）
  4角: 4番手にスパート
  ゴール: 2着

考えられる原因:
  騎手が「このペースでは前が止まる」と判断
  → 一旦下げて脚を溜めた → 4角から仕掛けてスパート
  → [マクリ](pace_crasher_and_makuri.md) に近い動き

充実度の判断:
  これは「充実」というより「騎手の好判断」。
  馬の状態が良いから騎手が積極的に動けたとも解釈できる。
  → 次走も同じ騎手なら再現性あり。
```

## 3. なぜ「盛り返し」は充実の証拠と言えるか

### 3-1. 身体的な根拠

```
レース中の馬のエネルギー消費:
  序盤: グリコーゲン（好気性代謝）→ 効率が良い
  中盤: 乳酸蓄積開始 → 筋肉疲労が始まる
  終盤: 乳酸濃度が閾値超え → 急激に失速

一度位置取りが下がる = 乳酸蓄積が一時的に閾値に達した状態。

盛り返す = その後に再加速できた = 以下のいずれか:
  (a) 乳酸耐性が高い（フィットネスが良い状態）
  (b) 一時的にペースを落とすことで乳酸を処理できた
  (c) そもそもスタミナに余力があった

→ いずれも「体調が良い」「仕上がっている」ことを示唆。
→ 充実度が低い馬（疲労、故障気味）は
  一度下がったらそのまま止まる。盛り返せない。
```

### 3-2. 精神的な根拠

```
馬の戦意（Fighting Spirit）:
  競走馬には「勝ちたい」という本能的な闘争心がある。
  しかし、疲労やストレスで戦意を失う場合がある。

戦意喪失の兆候:
  - ポジションが下がり続ける（下がるだけで盛り返さない）
  - 上がり3Fが極端に遅い（もう走る気がない）
  - 着差が大きく開く（ゴールまで流している）

戦意維持の兆候:
  - 一度下がっても再加速する（まだ走る気がある）
  - 上がり3Fが水準レベル以上（最後まで走っている）
  - 下がったが着差が大きく開かない（粘っている）

→ 盛り返しは「この馬はまだ戦える状態にある」という
  精神的な充実の客観的な証拠。

★ 特に連敗中や長期休養明けの馬が盛り返した場合:
  「まだ終わっていない」サイン。
  → 次走での巻き返しに期待が持てる。
```

### 3-3. 次走予測への応用

```
前走で「盛り返し」パターンが観測された馬は:
  ① 体調が良い可能性が高い → 次走も好走する確率UP
  ② 戦意がある → 次走でも最後まで粘る可能性
  ③ 不利がなかった可能性 → 実力通りの走りが期待できる
    （※ 不利が原因で下がった場合は、不利がなければさらに上位）

→ 「前走で盛り返した馬」は、
  着順だけでは見えない「隠れた好走サイン」。
  特に着順が悪い場合（7-10着）でも、盛り返しパターンがあれば
  次走で大きく巻き返す可能性がある。

★ これは [着順バイアス](../model/finishing_position_vs_margin_bias.md の別角度:
  着順は「7着」だが、通過順位のパターンを見ると
  「一度12番手まで下がって7着まで盛り返した」→ 実力は高い。
  着順だけ見ると軽視されるが、盛り返しを検出すれば正しく評価できる。
```

## 4. JRA-VANデータからの検出方法

### 4-1. 盛り返しの定義（通過順位パターン）

```
基本定義:
  レース中に通過順位が「下がった後に上がる」パターン。

具体的な判定条件:

■ 盛り返し判定（強い版）:
  条件1: max(通過順位) - min(通過順位の前半) ≥ 4
    （前半の最良ポジションから4つ以上下がった）
  条件2: max(通過順位) - 着順 ≥ 3
    （最悪ポジションから着順まで3つ以上戻した）

  例: 3-3-8-5 → 着順4
    下落: 8 - 3 = 5 ≥ 4 ✓
    盛り返し: 8 - 4 = 4 ≥ 3 ✓
    → 盛り返し判定: YES

■ 盛り返し判定（弱い版 = 粘り込み）:
  条件1: 3角順位 - 1角順位 ≥ 3（中盤で3つ以上ポジション下落）
  条件2: 着順 - 4角順位 ≤ 0（4角以降はポジション維持 or 改善）

  例: 2-2-6-6 → 着順5
    中盤下落: 6 - 2 = 4 ≥ 3 ✓
    4角以降: 5 - 6 = -1 ≤ 0 ✓
    → 粘り込み判定: YES

★ 閾値は過去データで最適化する必要がある。
  上記は仮の値。
```

### 4-2. 盛り返し強度の定量化

```
盛り返し強度:
  = (最悪ポジション - 着順) / (最悪ポジション - 最良ポジション)

  0: 盛り返しなし（下がったまま）
  0.5: 半分戻した
  1.0: 完全に戻した（下がる前と同じポジション）
  >1.0: 下がる前より上位でゴール（超盛り返し）

例1: 3-3-8-5 → 着順4
  強度 = (8-4) / (8-3) = 4/5 = 0.8 → 強い盛り返し

例2: 2-2-6-6 → 着順5
  強度 = (6-5) / (6-2) = 1/4 = 0.25 → 弱い盛り返し（粘り込み程度）

例3: 5-5-12-7 → 着順3
  強度 = (12-3) / (12-5) = 9/7 = 1.29 → 超盛り返し（下がる前より上位）

★ 強度が高いほど「充実度」が高いと推定。
  特に強度 > 0.7 は注目に値する。
```

### 4-3. 上がり3Fとの組合せ

```
盛り返しパターン × 上がり3F → 充実度の確度

盛り返し + 上がり3F速い:
  → 確実に充実。盛り返しの後にさらに速い脚を使えた。
  → 次走で大きく期待できる。

盛り返し + 上がり3F普通:
  → おそらく充実。盛り返すのに脚を使ったので上がりは平凡。
  → 次走で枠順や展開が向けば好走。

盛り返し + 上がり3F遅い:
  → 判断保留。盛り返したが最後は止まっている。
  → 充実しているが距離が長すぎた可能性。
  → 距離短縮なら次走好走の可能性。

盛り返しなし + 上がり3F遅い:
  → 戦意喪失 or 体調不良。次走は割引。
```

## 5. 特徴量設計

```
■ F30: 盛り返し強度（前走）
  定義: (最悪ポジション - 着順) / (最悪ポジション - 最良ポジション)
  範囲: 0〜1.5+（0=盛り返しなし、1=完全回復、>1=超盛り返し）
  条件: 最悪ポジション - 最良ポジション ≥ 3 の場合のみ計算
        （そもそも下がっていない場合は0 or null）
  用途: 前走の「充実度」の指標

■ F31: 盛り返し率（過去5走）
  定義: 過去5走で盛り返し判定（強い版）に該当した割合
  用途: この馬は「盛り返すタイプ」かの指標
        高い = 戦意が強い、スタミナに余力がある型
        低い = 下がったらそのままの型

■ F32: 盛り返し強度 × 上がり3F順位
  定義: F30 × (1 / 上がり3F順位)
  用途: 盛り返し+速い上がり = 最も充実度が高い
        盛り返し+遅い上がり = 充実度はやや割引

■ F33: ポジション下落度（中盤）
  定義: max(3角順位, 4角順位) - min(1角順位, 2角順位)
  用途: 中盤でどれだけポジションが下がったかの絶対値。
        これ単独では悪い指標だが、F30（盛り返し強度）と
        組合せることで「下がったけど盛り返した」を検出。
        下落度が大きい × 盛り返し強度が高い = 最も充実。
```

## 6. 「戦意喪失」の反対指標としての位置づけ

```
盛り返しの概念を広げると、
「馬の現在の精神的・身体的状態」を推定する枠組みになる。

  充実度 高:
    ↑ 盛り返し強度が高い + 上がりが速い
    ↑ 着差が小さい（僅差で負けている）
    ↑ 過去数走で着順が安定 or 上昇傾向

  充実度 低:
    ↓ 一度下がったらそのまま止まる（盛り返しなし）
    ↓ 上がり3Fが極端に遅い（最後方グループ）
    ↓ 着差が大きく開く
    ↓ 過去数走で着順が悪化傾向

→ これは [パフォーマンススコア](../model/alternative_target_variables.md の
  Layer 2補正（ルール補正）に組み込める:
  「盛り返しパターンがある走は、着順の割にスコアを上方修正」
  「戦意喪失パターンがある走は、着順に関わらずスコアを下方修正」
```
