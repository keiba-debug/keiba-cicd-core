# 環境物理学: 空気抵抗・気温・風・含水率が競走馬に与える影響

空気抵抗、気温、風、馬場含水率は「データ化しにくいが確実に存在する物理的影響」。将来的にこれらを特徴量に取り込めれば、他のAI競馬が見落としている構造的な優位性を得られる可能性がある。

> 起点: 「空気抵抗・空気密度・気温・風・含水率の変化はとても難しいテーマだが将来的には考える必要がある。暑い時期に馬体重が小さい馬のほうが好走するのか？風の影響は大きい馬のほうが受けないのか？」

## 2. 空気抵抗の物理学

### 基本方程式

```
空気抵抗（ドラッグ）の式:

  F_drag = 0.5 × ρ × v² × Cd × A

  ρ  = 空気密度 (kg/m³)
  v  = 速度 (m/s)
  Cd = 抗力係数（馬+騎手の形状で決まる。≈ 0.8-1.0）
  A  = 前面投影面積 (m²)（馬の正面から見た面積。≈ 1.0-1.5 m²）

具体的な数値:
  馬の速度: スプリント ≈ 60 km/h = 16.7 m/s
             中長距離   ≈ 55 km/h = 15.3 m/s
  空気密度: 20°C, 海面 ≈ 1.204 kg/m³
  Cd × A: ≈ 0.9 × 1.2 = 1.08 m² （推定）

  F_drag = 0.5 × 1.204 × 16.7² × 1.08
         ≈ 181 N（ニュートン）
         ≈ 18.5 kgf（キログラム重）

  馬の体重: ≈ 480 kg
  ドラッグによる減速: 181 / 480 ≈ 0.377 m/s²

  → これは無視できない大きさ。
  → 馬は常に空気抵抗と戦いながら走っている。
  → 先行馬が風を受け、後方の馬はスリップストリームの恩恵を受ける。
```

### 速度の二乗に比例する意味

```
F_drag ∝ v²

  速度が10%上がると、空気抵抗は21%増加する。
  速度が20%上がると、空気抵抗は44%増加する。

含意:
  - スプリントレース（速度が速い）ほど空気抵抗の影響が大きい
  - 長距離レース（速度が遅い）ほど空気抵抗の影響は小さい
  - ハイペースで飛ばす逃げ馬は、空気抵抗の「代償」を最も多く払っている
  - 後方の馬はスリップストリーム効果で空気抵抗が軽減される

  → 「逃げ馬が最も空気抵抗のコストを払う」は物理的事実。
  → ただし、この効果はペースや脚質の効果に比べて小さい可能性がある。
```

## 3. 空気密度: 気温・湿度・気圧・標高の影響

### 空気密度の計算

```
空気密度ρは気温・気圧・湿度で決まる:

  ρ = P / (R_specific × T)

  簡略版（乾燥空気）:
    ρ = P / (287.05 × T)

    P = 気圧 (Pa)
    T = 絶対温度 (K) = 摂氏 + 273.15

  湿潤空気の補正:
    水蒸気は窒素・酸素より軽い（分子量: H2O=18, N2=28, O2=32）
    → 湿度が高いほど空気密度は「下がる」
    → 直感に反するが物理的事実。

  ★ 湿度が高い日は空気が「軽い」→ 空気抵抗が小さい
```

### 季節変動の具体的な数値

```
■ 日本の競馬場での気温変動:

  冬（1月、東京/中山）: 気温5°C,  ρ ≈ 1.269 kg/m³
  春秋（4月/10月）:     気温18°C, ρ ≈ 1.213 kg/m³
  夏（8月、小倉/新潟）: 気温32°C, ρ ≈ 1.153 kg/m³

  冬→夏の空気密度変化: 1.269 → 1.153 = 約9.1%減少

■ 空気抵抗への影響:
  F_drag ∝ ρ なので、夏は冬に比べて空気抵抗が約9%小さい。

  数値例（中距離レース、v = 15 m/s）:
    冬: F_drag = 0.5 × 1.269 × 225 × 1.08 = 154.2 N
    夏: F_drag = 0.5 × 1.153 × 225 × 1.08 = 140.1 N
    差: 14.1 N → 馬体重480kgに対して 0.029 m/s² の減速差

  2000mレースでの影響（概算）:
    走行時間 ≈ 120秒
    速度差の累積 ≈ 0.029 × 120 × 0.5 ≈ 1.7 m/s の速度差蓄積...
    → これは過大推定。実際にはエネルギー収支で計算すべき。

  エネルギーベースの推定:
    空気抵抗のエネルギー = F_drag × 距離
    冬: 154.2 × 2000 = 308,400 J
    夏: 140.1 × 2000 = 280,200 J
    差: 28,200 J ≈ 28 kJ

    馬の出力: ≈ 15,000 W（スプリント時）
    28 kJ / 15,000 W ≈ 1.9秒分のエネルギー

    ★ 2000mレースで約1-2秒の差に相当する可能性。
    → これは無視できない大きさ！
    → ただし、夏場の暑さによる体力低下がこれを相殺する。
```

### 標高の影響

```
日本の主要競馬場の標高:
  東京:  約40m
  中山:  約30m
  阪神:  約20m
  京都:  約40m
  小倉:  約10m
  新潟:  約5m
  福島:  約70m
  札幌:  約25m
  函館:  約35m

  → 日本の競馬場は全て低地。標高差は最大70m程度。
  → 標高100mごとに空気密度は約1.2%低下。
  → 最大でも0.8%程度の差 → 影響は非常に小さい。
  → 日本では標高の影響は無視してよい。

  （参考: 米国のデンバー競馬場は標高1,600m → 空気密度が約17%低い → 大きな影響）
```

## 4. 風の影響: 最も大きな環境要因

### 向かい風と追い風の物理

```
空気抵抗は「馬と空気の相対速度」で決まる:

  F_drag = 0.5 × ρ × (v - v_wind)² × Cd × A

  v_wind > 0: 追い風（抵抗減少）
  v_wind < 0: 向かい風（抵抗増加）

■ 向かい風 5 m/s の影響（馬の速度 v = 16 m/s）:
  無風:    F = 0.5 × 1.2 × 16² × 1.08 = 166 N
  向かい風: F = 0.5 × 1.2 × 21² × 1.08 = 286 N
  → 空気抵抗が72%増加！

■ 追い風 5 m/s の影響:
  無風:    F = 0.5 × 1.2 × 16² × 1.08 = 166 N
  追い風:  F = 0.5 × 1.2 × 11² × 1.08 = 78 N
  → 空気抵抗が53%減少！

■ 2000mレースでの影響（概算）:
  向かい風5m/s: 追加エネルギー ≈ (286-166) × 2000 = 240,000 J
               → 追加所要時間 ≈ 240,000 / 15,000 ≈ 16秒分のエネルギー
               → 実際にはペース調整が入るので、タイム影響は2-4秒程度

  ★ 風の影響は気温の影響より桁違いに大きい。
  ★ 5 m/sの風（そよ風程度）でもレース結果を変えうる。
```

### 風向きとコースの関係

```
■ 直線コースでの風:
  - 向かい風: 全馬に等しく影響するが、先頭の馬が最も影響を受ける
  - 追い風: 全馬に等しく恩恵があるが、先頭の馬の恩恵が最も大きい
  - → 向かい風は差し有利、追い風は逃げ有利の傾向

■ 周回コースでの風:
  - 一定方向からの風が、コースの位置によって向かい風にも追い風にもなる
  - ゴール前の直線が向かい風 → 最後のスパートが効きにくい → 前残り
  - ゴール前の直線が追い風 → スパートが効く → 差し有利

  東京競馬場の例:
    直線が東西方向。北風なら横風、西風なら向かい風（直線で）。
    → 風向きによって有利な脚質が変わる。

■ カーブでの横風:
  - バランスを崩すリスク
  - 外を回る馬が横風の影響をより受ける
  - 小柄な馬は特に影響を受けやすい可能性
```

### 風と馬体重の関係: 大きい馬は風に強いか？

```
物理学的分析:

  空気抵抗力: F_drag ∝ A ∝ (体高)² ∝ mass^(2/3)
  （前面投影面積は体格の二乗に比例。体格は体重の1/3乗に比例）

  空気抵抗による減速: a_drag = F_drag / mass ∝ mass^(2/3) / mass = mass^(-1/3)

  つまり:
    大きい馬（520kg）: a_drag ∝ 520^(-1/3) = 0.1240
    小さい馬（460kg）: a_drag ∝ 460^(-1/3) = 0.1296

    比率: 0.1296 / 0.1240 = 1.045

  ★ 小さい馬は大きい馬に比べて空気抵抗の減速効果が4.5%大きい。
  ★ つまり「大きい馬の方が風の影響を受けにくい」は物理的に正しい。

  ただし:
    - 4.5%の差は、460kg vs 520kgという60kgの差での話
    - 実際の馬体重差の多くは440-520kgの範囲（80kg差）
    - 最大でも5-7%程度の差
    - レース全体のタイムへの影響は0.1-0.3秒程度

  結論:
    物理的には「大きい馬の方が風に強い」は正しい。
    しかし影響は小さく、他の要因（体調、展開）に埋もれる可能性。
    → 風が強い日（10 m/s以上）にのみ有意な差が出るかもしれない。
```

## 5. 気温と馬体重: 暑い時期に小さい馬が有利か？

### 体温調節の物理学

```
馬の体温調節:
  - 馬は主に発汗（蒸発冷却）で体温を調節する
  - 運動時の代謝熱は非常に大きい
  - 高温環境では放熱が追いつかず体温が上昇 → パフォーマンス低下

スケーリング法則:
  代謝熱産生:  Q_produce ∝ mass^(3/4)  （Kleiberの法則）
  放熱能力:    Q_dissipate ∝ mass^(2/3)  （体表面積に比例）
  放熱余裕度:  Q_dissipate / Q_produce ∝ mass^(2/3 - 3/4) = mass^(-1/12)

  → 大きい馬ほど「体表面積あたりの熱産生が多い」
  → 大きい馬ほど体温調節が不利

具体的な計算:
  520kg馬 vs 460kg馬:
    放熱余裕度の比: (520/460)^(-1/12) = (1.130)^(-0.083) = 0.989
    → わずか1.1%の差

  ★ 体温調節のスケーリング効果は非常に小さい（指数が-1/12）。
  ★ 「暑い時期に小さい馬が有利」は物理的根拠としては弱い。
```

### しかし、実際のデータでは差が出る可能性がある

```
スケーリング法則だけでは捉えられない要因:

1. 体脂肪率の違い
   - 脂肪は断熱材。体脂肪が多い馬は放熱しにくい
   - 馬体重が大きい馬は脂肪量も多い傾向（ただし個体差大）
   - → 「馬体重」より「体脂肪率」が本質だが、計測できない

2. 毛の量と質
   - 夏毛 vs 冬毛で放熱効率が大きく変わる
   - 暑さに強い血統（サウスペイパス系など暖地品種）が存在
   → 血統が気温適性の代理変数になりうる

3. 発汗効率
   - 馬の発汗量は個体差が大きい
   - 発汗が多い馬は冷却効率が高いが、脱水リスクも高い
   → 直接測定できない

4. 体重と距離の交互作用
   - 暑い日の長距離レースでは体温調節の問題が累積する
   - 1200mスプリントでは体温上昇の前にレースが終わる
   - 2400m以上の長距離で差が出やすい可能性
   → 気温 × 距離 × 馬体重の三重交互作用

5. 実は「暑い日は軽い馬が走る」の因果が逆かもしれない
   - 夏場は上位クラスの馬が休養中 → 出走メンバーが弱い
   - 軽い馬が多いのは「夏場のメンバーレベル」の影響？
   - → 馬体重そのものの効果なのか、メンバー構成の効果なのか、分離が必要
```

## 6. 馬場含水率: 気温と湿度から予測する

### 馬場状態の物理

```
JRAの馬場状態区分: 良 / 稍重 / 重 / 不良
→ これは「含水率」を4段階に離散化したもの。

含水率に影響する要因:
  - 降水量（直接的）
  - 気温（蒸発速度に影響）
  - 湿度（蒸発速度に影響）
  - 風速（蒸発を促進）
  - 日照（蒸発を促進）
  - 散水（JRAが開催前に散水することがある）
  - 排水性（コースの構造。東京は水はけが良い等）

含水率の推定モデル（概念）:
  含水率(t) = 含水率(t-1) + 降水量 - 蒸発量
  蒸発量 = f(気温, 湿度, 風速, 日照)

  → Penman-Monteith方程式（農業で使われる蒸発散量の推定式）を応用可能。
```

### なぜ4段階の馬場状態では不十分か

```
問題1: 「良」の中にも差がある
  - 「パンパンの良馬場」（高速馬場）
  - 「やや水分を含んだ良馬場」（標準的なタイム）
  → 同じ「良」でも0.5-1.0秒のタイム差が出る
  → 連続値の含水率があれば精密な補正が可能

問題2: レース中に馬場が変わる
  - 1Rと12Rで馬場状態が変わることがある
  - 朝は「重」→ 昼に乾いて「稍重」→ 夕方に雨で「重」
  → レースごとの馬場状態を推定する必要がある

問題3: コースの位置による差
  - 内側は馬に踏まれて荒れる → 含水率が高い部分がある
  - 外側は比較的良好
  → 「馬場バイアス」（内有利/外有利）は含水率分布の問題
  → これは気象データからは推定困難。当日のレース結果から推定。

活用方法:
  馬場状態（良/稍重/重/不良）の4段階に加えて、
  気象データから含水率の「連続値推定」を作る。
  → 「良だが水分多め」vs「良だがカラカラ」を区別できる。
```

### 含水率と馬体重の関係

```
物理学的分析:

  重い馬場（含水率が高い）では:
  - 蹄が地面に沈む → エネルギーロス増加
  - 沈み量 ∝ 荷重 / 接地面積 ∝ mass / (蹄の面積)
  - 蹄の面積は馬体重に比例しない（体重が2倍でも蹄は2倍にならない）
  → 重い馬ほど地面に沈みやすい → 重い馬場で不利

  軽い馬場（含水率が低い、パンパンの良馬場）では:
  - 地面が硬い → 蹄への衝撃が大きい
  - 重い馬ほど蹄にかかる衝撃が大きい → 脚への負担
  - しかし硬い地面は反発力も大きい → 速い馬が有利

  → 重馬場では軽い馬が有利、良馬場では重い馬でも問題ない
  → これは一般的に知られた傾向であり、既存研究でも確認されている

特徴量化:
  - 含水率推定値 × 馬体重 の交互作用
  - → 「重馬場で重い馬」のペナルティをモデルに学習させる
```

```
    ❌ 気温、湿度、風速、風向

```

```
  → CSVでダウンロード可能

  → レース日時 × 競馬場 → 最寄り観測所の該当時間帯のデータ

Step 4: 馬体重との交互作用
  → 気温 × 馬体重
  → 風速 × 馬体重
  → 含水率推定 × 馬体重
  → LightGBMに入力
```

## 8. 具体的な特徴量設計

### Phase 1: 低コストで即実装可能

```
気象庁データがなくても、JRA-VANのデータだけで部分的に対応:

■ 馬場状態 × 馬体重
  - 良馬場での馬体重: 影響なし（基準）
  - 稍重での馬体重: やや重い馬が不利
  - 重/不良での馬体重: 重い馬が明確に不利
  → 馬場状態 × 馬体重の交互作用項

■ 季節（月）× 馬体重
  - 夏場（6-9月）× 重い馬: やや不利（暑さの影響）
  - 冬場（12-2月）× 重い馬: 影響なし or やや有利
  → 月 × 馬体重の交互作用項

■ 天候 × 脚質
  - 雨 × 逃げ馬: 前残り傾向（重馬場で差し届かず）
  - 雨 × 差し馬: やや不利
  → JRA-VANの天候情報で対応
```

### Phase 2: 気象庁データとの統合

```
■ 気温（レース時の実測値）
  - 連続値として使用（月よりも精密）
  - 気温 × 馬体重
  - 気温 × 距離（長距離で暑いほど不利）

■ 湿度
  - 湿度が高い → 発汗による冷却効率が下がる
  - 湿度 × 気温 = 「不快指数」的な指標
  - 不快指数 × 馬体重 × 距離

■ 風速
  - 風速（m/s）を直接特徴量に
  - 風速 × 距離カテゴリ（スプリントでの影響が大きい）
  - 風速 × 馬体重（大きい馬は風に強い）

■ 風向 × コースの直線方向
  - 各競馬場の最終直線の方角を固定値として持つ:
    東京: 概ね西→東
    中山: 概ね北西→南東
    阪神: 概ね西→東
    ...
  - 風向と直線方向の角度差から「直線での向かい風成分」を計算
  - cos(θ) × 風速 = 直線での向かい風/追い風の強さ
  - → 逃げ馬には直線の向かい風が不利、差し馬には有利

■ 空気密度
  - ρ = f(気温, 湿度, 気圧) で計算
  - 直接特徴量に入れるか、スピード指数の補正に使う
```

### Phase 3: 馬場含水率モデル

```
■ 含水率推定値
  - 直近48時間の降水量の累積
  - 蒸発量の推定（気温 × 風速 × 日照で概算）
  - 含水率 ≈ 直近降水量 - 蒸発量推定
  - → JRAの馬場状態よりも精密な連続値

■ 含水率 × 馬体重
  - 含水率が高い × 馬体重が大きい → 不利（沈み込み）
  - 含水率が低い × 馬体重が大きい → 中立

■ 含水率の変化率
  - 「乾いてきている」vs「湿ってきている」
  - レース開始前の数時間の含水率変化
  - → 1R時点の馬場と12R時点の馬場の違いを予測
```

## 9. 仮説と検証計画

### 検証可能な仮説

```
H-30: 風速が強い日は馬体重が大きい馬が相対的に有利
  検証方法:
    1. 気象庁データで各レース時の風速を取得
    2. 風速を「弱（0-3m/s）」「中（3-7m/s）」「強（7m/s以上）」に分類
    3. 各群で馬体重と着順の相関を比較
    4. 風が強い群でのみ「重い馬が有利」の傾向が出るか確認
  成功基準: 強風群で馬体重と着順の相関係数が有意に異なる
  必要データ: 気象庁アメダスデータ + JRA-VANデータ

H-31: 夏場の長距離レースでは馬体重が小さい馬が有利
  検証方法:
    1. 6-9月の2000m以上のレースを抽出
    2. 馬体重別の勝率・連対率を比較
    3. 同条件の冬場（12-2月）と比較
    4. 夏場のみ軽量馬が有利な傾向があるか確認
  成功基準: 夏場×長距離で軽量馬の優位性が冬場より統計的に有意に大きい

H-32: JRAの馬場状態（4段階）より含水率推定値の方が予測精度が高い
  検証方法:
    1. 気象データから含水率を推定
    2. モデルA: 馬場状態（4カテゴリ）を使用
    3. モデルB: 含水率推定値（連続値）を使用
    4. AICまたはクロスバリデーションでB > Aかを確認
  成功基準: モデルBが有意に精度向上

H-33: 直線の向かい風は差し馬に有利、追い風は逃げ馬に有利
  検証方法:
    1. 風向 × 各競馬場の直線方向から向かい風成分を計算
    2. 向かい風が強いレースで差し馬の好走率を確認
    3. 追い風が強いレースで逃げ馬の好走率を確認
  成功基準: 風向き×脚質の交互作用が統計的に有意
```

## 10. 実装のロードマップ

```
Phase 0（今すぐ）: JRA-VANデータのみで簡易対応
  - 馬場状態 × 馬体重
  - 月 × 馬体重
  - 天候 × 脚質
  → features.mdに追加済みの特徴量で部分的に対応可能

Phase 1（データ基盤構築）: 気象データの統合
  - 気象庁アメダスデータのダウンロードスクリプト
  - レースデータとの結合パイプライン
  - 気温・湿度・風速の基本特徴量
  → 1-2週間の実装期間

Phase 2（探索的分析）: 仮説検証
  - H-30〜H-33の検証
  - 「本当に影響があるのか」をデータで確認
  - 有意な効果が見つかった項目のみモデルに組み込む
  → 影響が小さければPhase 3に進まない判断も重要

Phase 3（精密モデル）: 含水率推定・空気密度計算
  - 含水率推定モデルの構築
  - 空気密度の計算と補正
  - 風向 × コース方向の計算
  → Phase 2の結果次第

注意:
  005の教訓（過学習）を常に意識する。
  「物理的に正しい」ことと「予測に有用」は別の問い。
  気温・風の影響が統計的に検出できるほど大きいかは、
  仮説検証で確認してから先に進む。
```

## 11. この考察が示す本質: 「他が見ていないもの」に価値がある

```
環境物理学の特徴量は、単独では大きな改善をもたらさないかもしれない。
しかし:

1. 市場が織り込んでいない情報 = EV機会
   → 一般ファンは気温や風を馬券に反映していない
   → 少なくとも「風が強い日のスプリント」等の特定条件では
     市場の非効率性がある可能性

2. 小さな改善の積み重ね
   → roi_volume_tradeoff.md で論じたように、
     回収率を0.5%上げるだけでも年間では大きな差
   → 環境物理学が0.5%の改善に寄与するなら、取り組む価値がある

3. 独自の競争優位性
   → 他のAI競馬が実装していない特徴量 = 独自の情報源
   → 012で松風が「独自データへの投資」を強調した文脈に合致

4. 「なぜ」を理解していることの価値
   → 「重馬場で軽い馬が走る」を経験則として知っているのと、
     「蹄の接地圧が低いから沈み込みが小さい」と理解しているのは違う
   → 理解があると、新しい条件（例: 新コース）への汎化が効く
```
