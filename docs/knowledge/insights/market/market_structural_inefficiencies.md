# 馬券市場の構造的非効率性

一般ファンの馬券購入行動に起因する、AIが突ける構造的な歪みの考察。

> 起点: 「三連単フォーメーションで、確率が全く違う組合せに全部100円ずつ賭けている。そこに脆弱性がある」

---

## 1. 三連単フォーメーションの均等配分問題 ★★★

### 一般ファンの買い方

```
フォーメーション例:
  1着: A, B
  2着: A, B, C
  3着: A, B, C, D, E

→ 展開すると20通り、全て100円ずつ = 2,000円

しかし各組合せの的中確率は全く異なる:
  A→B→C: 5.0%（本命決着）
  A→B→E: 0.3%（3着が穴）
  B→C→E: 0.05%（荒れ決着）
  ...
```

### 何が歪みを生むか

| 組合せ | 真の的中確率 | ファンの賭け金 | あるべき賭け金(確率比) |
|---|---|---|---|
| A→B→C | 5.0% | 100円 | 1,000円 |
| A→B→D | 1.5% | 100円 | 300円 |
| A→B→E | 0.3% | 100円 | 60円 |
| B→C→E | 0.05% | 100円 | 10円 |

- 高確率の組合せに対して**過少投資**
- 低確率の組合せに対して**過剰投資**
- 結果としてパリミュチュエル方式のオッズプールに歪みが生じる

### パリミュチュエルでの帰結

```
パリミュチュエル:
  組合せXのオッズ = (全体のプール × (1-控除率)) / Xへの投票額

フォーメーション買いが大量にあると:
  → 低確率組合せに不釣り合いな金額が集まる
  → 低確率組合せのオッズが「真のオッズ」より低くなる
  → 高確率組合せのオッズが「真のオッズ」より高くなる

AIの戦略:
  → 高確率組合せに集中投資（オッズが割高＝EV > 1.0になりやすい）
  → 低確率組合せは見送り（オッズが割安＝EV < 1.0になりやすい）
```

### AIの優位性

```
AIの買い方:
  A→B→C: 5.0% × オッズ50倍 → EV=2.50 → 500円投資
  A→B→D: 1.5% × オッズ120倍 → EV=1.80 → 150円投資
  A→B→E: 0.3% × オッズ400倍 → EV=1.20 → 30円投資
  B→C→E: 0.05% × オッズ2000倍 → EV=1.00 → 見送り

ファンの買い方:
  全部100円ずつ（EVを考慮していない）
```

**核心**: AIは確率に比例した金額配分ができる。一般ファンはフォーメーションの構造上、均等配分を強いられる。

## 2. 同類の構造的歪み一覧

### 2-1. ボックス買いの歪み ★★★

フォーメーション以上に非効率な買い方。

```
三連単5頭ボックス = 60通り × 100円 = 6,000円

しかし:
  1番人気→2番人気→3番人気: 確率5%
  5番人気→4番人気→3番人気: 確率0.01%

60通りが全て等確率のはずがない。
ボックス買いは「順番がわからないから全通り」だが、
AIは順番ごとの確率を推定できる。
```

**歪みの度合い**: フォーメーションより大きい。ボックスは着順の情報を完全に放棄しているため。

### 2-2. ながし（流し）買いの歪み ★★

```
三連単 軸1着A → 相手B,C,D,E,F → 3着全頭

A→B→C: 3.0% (2番人気が2着)
A→B→F: 0.8% (6番人気が3着)
A→E→F: 0.1% (人気薄同士)

全て100円ずつ → AとBの組合せにもっと厚く張るべき
```

### 2-3. 馬連・ワイドでの均等買い ★

複数の組合せを100円ずつ買うパターン。三連単ほど組合せ数が多くないため歪みは小さいが、同じ構造。

```
ワイド: A-B, A-C, A-D, A-E を100円ずつ
→ A-B（1-2番人気）の確率 >> A-E（1-5番人気）の確率
→ A-Bにもっと張るべき
```

### 2-4. Favorite-Longshot Bias（本命-大穴バイアス）★★★

学術的に最も研究されている馬券市場の非効率性。

```
現象:
  大穴馬（高オッズ）は過剰に買われている → オッズが実力より低い
  本命馬（低オッズ）は過少に買われている → オッズが実力より高い

原因:
  - 人間の「夢を買う」心理（万馬券への過度な期待）
  - リスク愛好的な選好（小さな確率を過大評価）
  - 「100円で100万円」のギャンブル的魅力

三連単での増幅:
  三連単は組合せ数が膨大 → 個々の組合せの確率推定が困難
  → ファンは「なんとなく」で買う → バイアスが増幅される
  → 三連単市場はFavorite-Longshot Biasが最も大きい馬券種の可能性
```

**AIの優位性**: 全組合せの確率を数値で推定できる。人間には4,896通り（18頭立て）の確率を直感で把握することは不可能。

### 2-5. 条件付き確率の無視 ★★★

一般ファンの思考回路:

```
ファンの思考:
  「Aが1着になりそう」（P(A)=30%）
  「Bは2着候補」（P(B)=25%）
  「Cは3着に来そう」（P(C)=20%）
  → A→B→C が来る！

ファンが無視していること:
  P(A1着 かつ B2着 かつ C3着) ≠ P(A1着) × P(B2着) × P(C3着)

  正しくは:
  P(A1着,B2着,C3着) = P(A1着) × P(B2着|A1着) × P(C3着|A1着,B2着)

  「Aが勝った場合」のBの2着確率は、Aが負けた場合とは異なる。
  例: AとBが同脚質（逃げ馬同士）なら、Aが勝つとBは潰れやすい
  → P(B2着|A1着) < P(B2着)
```

**AIの優位性**: 条件付き確率を正しくモデリングできる。特に三連単では着順の条件付き依存関係が複雑で、人間の直感では対応不可能。

### 2-6. 応援馬券・思い入れ馬券の歪み ★★

001（ショート戦略）で考察済みだが、三連単では影響が拡散する。

```
馬主クラブ会員のAファンが三連単を買う場合:
  必ずAを1着に入れる → A1着の組合せが過剰に買われる
  → A1着のオッズが「真のオッズ」より低い
  → A以外が1着の組合せは相対的に割高（EV > 1.0の可能性）
```

### 2-7. 控除率の馬券種間格差 ★

| 馬券種 | 控除率 |
|---|---|
| 単勝・複勝 | 20.0% |
| 枠連・馬連・ワイド | 22.5% |
| 3連複 | 25.0% |
| 3連単 | 27.5% |

- 三連単は控除率が最も高い（27.5%）
- しかし**市場の非効率性（歪み）が最も大きい**
- 歪みが控除率を上回れば、三連単が最もEVの高い馬券種になりうる
- 松風も2020年Q3に三連単を解放し、+1億65万の爆発（012）

```
EV = 真の的中確率 × オッズ
   = 真の的中確率 × (プール × (1-0.275)) / 投票額

控除率27.5%のハンデがあっても、
市場の歪み（均等配分バイアス等）が27.5%以上あれば正のEV
```

### 2-8. 締切時刻の異なる購入タイミング ★★

007（スマートマネー）との関連。

```
タイムライン:
  発売開始 ← カジュアルファンが早めに購入（フォーメーション等）
  ...
  最終オッズ ← AIが確率計算を完了
  ...
  締切直前 ← スマートマネー（情報優位者）が購入
  確定オッズ

カジュアルファンの購入:
  - 早い段階で均等額フォーメーションを購入
  - オッズプールの「下地」を形成

AIの購入:
  - 最終オッズ付近でEVを計算
  - 確率に比例した金額で購入
  - カジュアルファンが作った歪みを突く
```

### 2-9. 多頭数レースでの推定困難性 ★★

```
出走頭数と三連単の組合せ数:
  8頭: 336通り
  12頭: 1,320通り
  16頭: 3,360通り
  18頭: 4,896通り

多頭数ほど:
  - 人間の確率推定の精度が下がる
  - フォーメーションの「漏れ」が増える
  - 市場の非効率性が増大
  → AIの優位性が拡大
```

## 3. 歪みの体系的分類

### 分類マトリクス

| 歪みの種類 | 原因 | 影響する馬券種 | AIの対応 |
|---|---|---|---|
| **均等配分バイアス** | フォーメーション/ボックスの構造 | 三連単、三連複、馬連 | 確率比例の金額配分 |
| **Favorite-Longshotバイアス** | 夢買い心理 | 全馬券種（三連単で最大） | 大穴を避け、本命寄りに |
| **条件付き確率の無視** | 認知限界 | 三連単、三連複 | 正確な条件付き確率モデル |
| **応援馬券バイアス** | ファン心理 | 全馬券種 | 馬主クラブ分析(001) |
| **スマートマネー効果** | 情報の非対称性 | 全馬券種 | オッズ変動予測(007) |
| **多頭数の推定困難** | 認知限界 | 三連単で最大 | 全組合せの網羅的計算 |
| **控除率の見落とし** | 無知 | 三連単(27.5%)で最大損失 | EV計算で控除率を織り込み |

### 歪みの大きさ（推定）

```
歪みが大きい ← → 歪みが小さい

三連単 >> 三連複 > 馬連 > ワイド > 枠連 > 複勝 > 単勝

理由:
  - 組合せ数が多いほど、人間の推定精度が下がる
  - 組合せ数が多いほど、均等配分バイアスが大きくなる
  - 控除率も高い馬券種ほど高いが、歪みの方が上回る可能性
```

## 4. 三連単戦略への統合

### 松風の2020年Q3爆発の再解釈

012で記録された三連単導入後の爆発（Q3: +1億65万）は、上記の歪みを突いた結果と解釈できる。

```
松風の三連単戦略（推定）:
  1. 全出走馬の着順確率分布を推定（Poseidon/予測モジュール）
  2. 全三連単組合せのEVを計算（条件付き確率で正確に）
  3. EV > threshold の組合せのみ、確率に比例した金額で購入（Themis）
  4. 1日2,700点（→9,000点）の購入上限まで買う

vs 一般ファン:
  1. 「来そうな馬」を3-6頭選ぶ
  2. フォーメーション or ボックスで組む
  3. 全て100円ずつ

→ AIは「どの組合せに、いくら」を最適化できる
→ 一般ファンは「組合せの選択」はするが「金額配分」を放棄している
```

### 三連単EV計算の定式化

```
三連単(A→B→C)のEV:

  EV = P(1着A, 2着B, 3着C) × オッズ(A→B→C)

  P(1着A, 2着B, 3着C)
    = P(1着A) × P(2着B | 1着A) × P(3着C | 1着A, 2着B)

  計算量:
    n頭レースで n × (n-1) × (n-2) 通り
    18頭: 4,896通りの全てでEVを計算

  購入判断:
    EVが高い順に並べ、threshold以上を全て購入
    金額はKelly基準で確率に比例配分
```

### 三連単で特に効くバイアスの組合せ

```
最も歪みが大きいパターン（仮説）:

  1. 本命馬が1着の組合せ
     → ファンがフォーメーションの1着に入れがち
     → 実は2-3着の組合せで割高オッズが発生しうる

  2. 中穴馬が絡む組合せ（5-8番人気）
     → ファンのフォーメーションから漏れやすい
     → オッズが過大（= EV > 1.0の可能性）

  3. 着順の入れ替わり
     → ファンは「A→B→C」は買うが「A→C→B」を軽視
     → 2着と3着の入れ替えでオッズが大きく変わる
     → AIは条件付き確率で2着3着の順序を正確に推定

  4. 多頭数レース（16頭以上）
     → 人間の推定限界 × 組合せ爆発 → 歪みが最大化
```

## 6. 検証可能な仮説

### 仮説1: 三連単の本命決着組合せはEVが高い
```
検証: 1-3番人気の三連単組合せのオッズ × 的中確率を過去データで計算
予想: 均等配分バイアスにより、本命決着のオッズが割高
```

### 仮説2: 着順入れ替えでEVが変わる
```
検証: A→B→CとA→C→Bのオッズ比 vs 確率比を比較
予想: ファンが「なんとなく」で着順を決めるため、一方が過大評価/過小評価
```

### 仮説3: 多頭数レースほど三連単のEVが高い
```
検証: 出走頭数別に三連単の回収率を計算
予想: 多頭数ほど市場の非効率性が増大し、EV > 1.0の組合せが増える
```

### 仮説4: フォーメーション「端」の組合せはEVが低い
```
検証: フォーメーションの3着候補の末尾（= 低確率候補）のオッズ × 確率
予想: 均等配分バイアスにより、低確率組合せのオッズが実力より低い
```

### 仮説5: 三連単市場のFavorite-Longshot Biasは単勝より大きい
```
検証: 人気別の三連単回収率と単勝回収率を比較
予想: 三連単では大穴組合せの回収率がさらに低い（夢買いバイアスの増幅）
```
