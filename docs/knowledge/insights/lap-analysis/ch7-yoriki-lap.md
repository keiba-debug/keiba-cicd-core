# 第7章: 2歳馬の能力をラップで見分ける方法 — 「余力ラップ」理論

> 出典: 「ラップで読み解く勝ち馬の見つけ方」第7章 (pp.176-205)

---

## 章の前提: なぜ2歳戦でラップが特に有効か

- 2歳戦は情報が少ない（キャリア浅い、レース数少ない）
- 時計やタイムランクだけでは「勝ち時計が重視」→ 中身（ラップ構造）が軽視されがち
- **若駒はラップで余力を見極められる** = 本章の核心テーマ

---

## ラップに影響を与える「5+1要素」

ラップはコースとヒットで分けて考える（コース要因 + 展開要因）。

### コース要因 5つ:

| # | 要素 | 影響 |
|---|------|------|
| 1 | **初角までの距離** | 長い→スローペースになりやすい（先行争い収まる）。東京・新潟外回り等は直線長くスロー化。小倉は初角まで短くHペース化 |
| 2 | **最終直線の長さ** | 長い→スパート位置が遅い→ペースアップ遅い。短い→仕掛け早い→加速持続が必要 |
| 3 | **コーナーの角度** | 急→ペース落ちる（阪神・京都ダート1200m等）。緩→速度維持。内回り/外回りで差 |
| 4 | **坂の有無** | 上り坂→ペース落ち。下り坂→ペースup。中山2000mは3コーナー手前にアップダウン。阪神2000mはラスト1Fの急坂で減速 |
| 5 | **馬場状態** | タフ/高速で全体に大きく影響。各競馬場ごとに基準が異なる |

### +1（展開要因）:
- **頭数**: 少頭数→スロー化、多頭数→ハイペース化（競り合い増）

### 各競馬場の馬場タフ度基準（芝2000m, 良馬場との差）:

| 競馬場 | 差(秒) | 特徴 |
|--------|--------|------|
| 東京 | -1.5 | タフ度高い。水はけ良いが時計かかる |
| 中山 | -0.7 | 馬場悪化頻度は減。芝張替後は高速 |
| 阪神 | -1.1 | 2019年から高速馬場化。タフになりにくい |
| 京都 | -0.9 | 雨が降ると一気に低速化。改修後の馬場に注目 |
| 新潟 | -0.7 | 夏開催は超高速。夏の後半・秋・春開催はタフ |
| 福島 | -0.6 | 年を通してほぼタフ |
| 中京 | -0.4 | 2019年から高速馬場化 |
| 小倉 | -0.5 | 夏開催は高速。開催後半〜秋はタフ化 |
| 札幌 | -1.0 | 元からタフな洋芝。-1.0秒でもレースは上がりかかる |
| 函館 | -1.0 | 札幌同様タフ。馬場設定で年によって変化 |

## 「余力ラップ」理論の核心

### 基本概念
**余力ラップ = 勝ち馬がレースの最後（ラスト1-5ハロン）で見せた、能力の余裕度を示すラップ**

- 2歳戦は能力差が大きい → 強い馬は余力を残して勝つ
- その余力がラップの終盤区間に現れる
- **時計の絶対値よりも、ラストの加速/減速パターンが重要**

### なぜ余力ラップが有効か
- 多くの予想家が「レーススケールで馬のスケールや余力を見極める」と言うが、その基準は人によって曖昧
- 本書は **定量的な5つの基準** を提示してこれを体系化

## 芝編: 若駒の余力を見極める5つの基準

### 基準1: ラスト1ハロン 11.4秒以内

- **新馬戦・未勝利戦を勝利した馬のラスト1ハロンが11.4秒以内なら余力がある（強い）と見る**
- 2021年の2歳馬（芝のみ）: 勝ち上がった延べ402頭中、29頭該当（2022年末時点で11頭が重賞勝ち）
- 2022年: 65頭がクリア
- キラーアビリティ（ホープフルS勝ち）のラスト1ハロン10.8秒等の具体例

### 基準2: ラスト2ハロン 22.7秒以内

- 夏目耕四郎氏の「ラスト2ハロンが23秒5以内/11秒5以内→11秒5以内」を修正した基準
- 著者は **ラスト2ハロン22.7秒以内** がより正確と判断
- 2021年: 延べ53頭がクリア、2022年末時点で重賞勝ち馬15頭
- 2022年: 89頭がクリア
- **注意**: 新馬戦のラップで、ラスト1ハロンが速くても2ハロン通しで見ないと判断を誤る
  - 例: ドルチェモアの新馬戦（ラスト2ハロンはクリアだがラスト1ハロンは遅い = ペース先行で粘った形）

### 基準3: 上がり最速との差（相対比較）

- ラスト3ハロンは公知情報で馬券的妙味は低い
- ただし **上がり2位の馬に上がり最速をマークした時は価値が高まる**
- 超スローの上がり最速は価値低い（全馬脚が溜まっている）
- 上がり差0.5秒以上 = 瞬発力の質的差

### 基準4: ラスト4ハロン 46.0秒以内（マイル以上）

- 2歳のマイル以上の距離において、**ラスト4ハロン46.0秒以内のレースで勝った馬** = 総合力が高い
- ラスト4ハロンは「トップスピード + ギアチェンジ能力に秀でた馬」のペース変動で速い時計が出る
- ラスト4ハロンが速い = **トップスピードを長く持続** + **緩急に対応**する総合的な力
- 改定: 2021年以前は「阪神外回り・新潟外回り」限定だったが、2022年以降は「阪神外回り」を「阪神」に拡大
- 2021年以前の基準クリア馬: サリエラ、セリフォス等の重賞馬多数
- 2022年: ラスト4ハロン基準クリア馬リスト掲載

### 基準5: ラスト5ハロン 58.2秒以内（マイル以上）

- 「マイル以上の距離でラスト5ハロン58.2秒以内」
- ラスト5ハロンのクリアは非常に難しい → 過去クリア馬は少数精鋭
- **ラスト4ハロンとラスト5ハロンの違い**:
  - 4ハロン = トップスピード＋ギアチェンジ
  - 5ハロン = **持続力の証明**（より長い区間でのスピード維持）
- コントレイル、サリオス等がクリア

### 基準の運用方法

1. まず基準1・2で余力を見る（若い順 = ラスト1F→2F）
2. **タイムランクBランク以上で勝ち時計も良い馬が、さらにラスト2ハロン基準もクリア** = 時計＋ラップ両方◎ = 鬼に金棒
3. 基準4・5は古馬のレースレベルを測る尺度にも転用可能
4. 基準をクリアしなかった馬でも、「次走の平均着順」（Victory Roadというサイトのデータ）で追跡して価値を検証

### 基準クリアの「2着馬」の扱い

- **基準をクリアしたレースの2着馬をどう扱うか** は重要な論点
- 2着でも「タイム差なし」なら評価は同等
- 2着だったレベンスティールは今後の成功が期待された例
- ただし、減速ラップ or 加速ラップ不足の2着は余力がない可能性あり

## 若駒の芝・短距離戦の攻略法

### 2歳短距離（芝1400m以下）の特徴
- 逃げ切り勝ちが多い = 次走は逃げ馬有利バイアスに見える
- 函館2歳ステークス等では **前走逃げた馬が危険な人気馬** になりやすい
  - 理由: 新馬戦のペースが緩く逃げやすい → 本番でHペース化すると脆い

### 函館2歳ステークスの前走脚質別成績（2012-2021）:

| 前走脚質 | 勝率 | 連対率 | 複勝率 | 単勝回収 | 複勝回収 |
|---------|------|--------|--------|---------|---------|
| 逃げ | 5.2% | 12.1% | 17.2% | 30% | 99% |
| 先行 | 9.0% | 15.4% | 24.4% | 100% | 98% |
| 中団 | 0.0% | 12.5% | 12.5% | 0% | 42% |
| 後方 | 0.0% | 0% | 0% | 0% | 0% |
| マクリ | 0.0% | 0% | 0% | 0% | 0% |

- **前走逃げの複勝回収率と他が高く見えるが、人気馬の特性として対穴を開ける例が少ないから**
- 前走先行が勝率最高

### 短距離の余力指標
- 2歳芝1400m以下の新馬・未勝利: **前走脚質が「前走競馬・先行」の馬の次走成績が良い**
- 集計: 勝率14.3%、連対率26.6%、複勝率38.9%、単回収114、複回収109
- ペタペタ逃げ（余裕のない逃げ）vs 先行して余力を残す競馬の差

## ダート編: 若駒の余力を見極める

### ダート2歳戦の特徴
- ダート馬の方が芝馬より成長するまで時間がかかる → 新馬は「上がり4ハロン」で判定有効
- 新馬は上位馬でもペースが遅くスローからの加速 → 時計より加速パターンが重要

### ダート余力基準: 上がり4ハロン 50.0秒以内

- **ダート1700m以上では「上がり4ハロン50.0秒以内」のレース勝ち馬に注目**
- 1600m以下は中距離カテゴリーに入れるか著者は悩んでいるが、結論として1600m以下も含める
- 馬場問わず **上がり4ハロン50.0秒以内** のレースでの勝ち馬の次走を見る

### ダート1700m以上 上がり4ハロン50.0秒以内の勝ち馬 次走成績:

| 条件 | 勝率 | 連対率 | 複勝率 | 単勝回収 | 複勝回収 |
|------|------|--------|--------|---------|---------|
| 条件合致 | 27.8% | 44.4% | 47.2% | 123% | 91% |
| その次走 | -% | -% | -% | -% | -% |

- 勝率27.8%、単勝回収123% = かなり高い的中率
- 上がり4ハロン50.0秒は「出にくい数字」→ 基準クリア馬は73頭(集計期間中)のみ

### ダート短距離の余力基準

- ダートの短距離で有効なのは **タイムランクA・Bの馬** （先走の結果タイムのタイムランクで判定）
- タイムランクA-Bの次走成績: 勝率18.1%、連対率26.8%、複勝率39.1%、単回収101%、複回収73%
- ダート短距離は体が未完成な馬が多い → 減速が少ないor加速ラップの馬は余力がある

### ダートの加速ラップ事例
- カルネアサーデ（阪神1R, 3歳未勝利, ダート1200m）: 加速ラップ出現 → 即注目
- ゲンパチフォルツァ（中山ダート1200m）: 加速ラップ出した勝ち馬 → 次走好走

### ダート新馬・未勝利のラストハロン基準

| 条件 | 勝率 | 連対率 | 複勝率 | 単勝回収 | 複勝回収 |
|------|------|--------|--------|---------|---------|
| L1H基準合致 | 26.0% | 36.9% | 52.1% | 85% | 108% |

- ダートのラスト1ハロン基準: **12.3秒以内**（良馬場）/ **12.2秒以内**（重・不良）
- 頭数体格問わず = 出馬頭数少ない短距離でも有効
- 条件に合致した馬のエンジェルキャリー等の具体例

## 実用上の重要ポイント

### 1. 余力ラップは「絶対値」ではなく「レースの中での構造」を見る
- 同じラスト1ハロン11.3秒でも、スローペースの11.3秒と、ハイペースの11.3秒では価値が全く違う
- **基準はあくまで「入口」**。レース全体のラップ構造と合わせて判断する

### 2. 馬場状態の補正が必須
- タフな馬場では基準をクリアしにくい → 馬場差を考慮する必要あり
- 著者は各競馬場の馬場差テーブルを提供（上記参照）

### 3. 「基準をクリアした馬を追いかける」だけでは収支プラスにならない
- 基準は **注目馬のフィルタ** であって、馬券の自動購入ルールではない
- ラスト1ハロン基準馬 → 人気に反映されることが多い → 単勝回収率は100%前後
- 複勝回収率が安定的に100%超 = **穴馬を拾う指標としてはやや弱い**
- ただし **VBシステムと組み合わせれば、余力ラップ馬で市場が過小評価しているケースを検出可能**

### 4. ダートは芝以上にラップの見方にメリットがある
- ダートは芝に比べてラップを見る人が少ない → 市場に織り込まれにくい
- その分、ダートでラップを見ておくことは大きなメリット（著者の結論）

## KeibaCICD への応用可能性

### すぐ使える要素

| 理論 | 実装アイデア | データ有無 |
|------|------------|----------|
| **ラスト1Fハロン基準** (11.4秒) | SE_DATAの各馬last_3fから逆算は不可 → レースラップのラスト1F利用 | JRA-VAN SR_DATA lap_times ✅ |
| **ラスト2Fハロン基準** (22.7秒) | lap_timesの最後2つの合計 | 同上 ✅ |
| **ラスト4Fハロン基準** (46.0秒, マイル以上) | lap_timesの最後4つの合計 | 同上 ✅ |
| **ラスト5Fハロン基準** (58.2秒, マイル以上) | lap_timesの最後5つの合計 | 同上 ✅ |
| **ダート上がり4F基準** (50.0秒) | lap_timesの最後4つの合計（ダート限定） | 同上 ✅ |
| **ダートL1H基準** (12.3秒良 / 12.2秒重不良) | lap_timesの最後1つ | 同上 ✅ |
| **加速ラップ検出** | lap_times配列で末尾が加速しているかチェック | 同上 ✅ |
| **上がり最速との差** | SE_DATAのlast_3f順位 + タイム差 | SE_DATA ✅ |
| **馬場タフ度テーブル** | 競馬場 × 馬場状態 → 補正係数 | 著者のテーブル + race_*.json ✅ |

### ML特徴量候補

1. **yoriki_l1f**: 勝ちレースのラスト1ハロンが基準クリアか (binary)
2. **yoriki_l2f**: 勝ちレースのラスト2ハロンが基準クリアか (binary)
3. **yoriki_l4f**: 勝ちレースのラスト4ハロンが基準クリアか (binary, マイル以上)
4. **yoriki_l5f**: 勝ちレースのラスト5ハロンが基準クリアか (binary, マイル以上)
5. **accel_lap_flag**: 勝ちレースのラストが加速ラップだったか (binary)
6. **l1f_value**: ラスト1ハロンの生タイム (continuous)
7. **l2f_value**: ラスト2ハロンの合計タイム (continuous)
8. **best_l1f_career**: キャリア中最速のラスト1ハロン (continuous)
9. **baba_toughness**: 馬場タフ度補正済みの基準判定

### 注意点
- これらは「レース単位」のラップ → 馬別のラスト1Fではない（馬別は上がり3Fのみ利用可能）
- 2歳限定の基準だが、3歳以上にも転用可能と著者が言及
- Model Bに入れるとVB ROI悪化する可能性（市場が織り込んでいる場合）→ まずModel Aで検証
