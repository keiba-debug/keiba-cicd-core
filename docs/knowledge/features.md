# AI特徴量候補リスト

ブログ考察から抽出した、競馬AI予測モデルに活用できる特徴量の候補を管理する。

## 凡例

- **実装難度**: 低 / 中 / 高
- **データソース**: JRA-VAN / スクレイピング / 計算値
- **出典**: 考察ドキュメント番号

## 特徴量一覧

（ブログ考察に応じて追記）
