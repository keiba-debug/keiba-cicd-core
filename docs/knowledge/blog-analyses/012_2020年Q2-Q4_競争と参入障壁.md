## 要点

### Q2: 馬券種別収支の罠 ★★

#### +1,300万 → -1,300万の週次変動

- ある週に+1,300万、翌週に-1,300万 → 2週合計は-39万
- 「割とどうでも良くなりますね。皆さんもぜひやってみてください」
- 複利運用の高ベットレートによる変動の大きさ

#### ワイドの収支が悪い → ワイドを止めるべきか？

> これを見て「ワイドを止めれば良い」という結論を導くのは拙速です

- レートが変動するため、高レート時の結果に引きずられる
- **馬券種を事後的に選択する = サイコロを100個振って偏った目を選ぶ**のと同じ
- 005の「平均回帰」、009の「生存者バイアス」と同じ論点
- ただしMatsukaze氏の印象: ワイド3頭目が来ない → 同時的中2つ分を逃している

#### 無観客競馬の影響

- 外的環境の変化（無観客競馬によるオッズ傾向の変化）の可能性に言及
- 応援馬券の減少 → 001のショート戦略の前提が変わる可能性

### Q2: 競合戦略とAlphaImpact提携 ★★★

> 最近、競合の競馬AIerが増えてきました

#### AlphaImpact社との提携

- 以前「某社で某Mamba」というAIを共同開発した関係
- 両者とも競馬AI業界の先行勢と自負
- 「ここで我々が喧嘩するべきではない」→ チームマージの方向

#### 後続への対抗戦略

> 期待値の高い馬券のオッズをしっかり下げ、後続の複利での資産形成を阻害する事です

- **高EV馬券のオッズを自分たちが消費して下げる** → 後続のEV低下
- 回収率が数%下がるだけで、複利では1-2年で桁が変わる差になる
- 一部のレース・馬券種で「上限コツン」（購入上限に到達）
- **規模こそが最大の参入障壁**

### Q3: 三連単の解放 ★★

- Q3で一気に+1億65万 → 三連単の導入が大きな要因
- PAT購入点数の上限問題:
  - 当時の上限: 1口座あたり2,700点/日
  - 三連単は買い目が多い → 上限超過
  - **対策: PAT口座を3つ用意し、切り替えるシステムを構築**
  - 3週間後にPATの上限が9,000点に緩和 → システム不要に
- 三連単は高配当×低的中率 → 期待値があれば大きな利益を生む

### Q3: 独自データへの投資 ★★★

> 郊外に家が土地付きで立つくらいの金額で、過去20年のレースの目視データを発注・購入

- **「目視データ」**: 推定でレース映像の解析データ（パトロールビデオ等から人手で作成？）
- 投資額: 数千万円規模（「郊外に家が土地付きで立つ」）
- **投資回収: 1ヶ月**
- 「完全に新たなステージに突入」
- この意思決定は「経営力」= AI/競馬スキルとは別の資本力の問題

> データはパワー

### 2020年総まとめ: スケールの限界

> 来年はさすがにスケールしなくなってきて、それでも10億くらい勝てるかなと思っています

- 2020年の好調要因: **三連単の解放 + 独自データの蓄積・投入**
- 年間25億円の購入 → 市場への影響が無視できないレベルに
- 「金利も結構大きい」→ 銀行を変えるほどの預金規模
- 定期的な収支報告はこの回で実質終了

## AI開発への示唆

### 1. 馬券種の事後選択バイアス ★★

馬券種別収支を見て「負けている馬券種を止める」判断の危険性:

```
正しい判断プロセス:
  1. 各馬券種のEV計算ロジックに理論的欠陥はないか？
  2. サンプル数は十分か？（数十レースでは不十分）
  3. レート変動の影響を除いた均等ベースの回収率は？
  4. 馬券種間の相関を考慮したポートフォリオ効果は？

誤った判断プロセス:
  × 直近の収支が悪い → 止める
  × 特定期間で回収率が低い → その馬券種は相性が悪い
```

- 008のポートフォリオ理論と矛盾しない: ワイドの理論的EVが正なら、短期の下振れで止めるべきではない
- 005のサイコロの比喩の直接適用

### 2. 競合戦略: オッズの食い合い ★★★

競馬AI市場の競争構造:

```
競合が増えると:
  → 同じ高EV馬券に複数のAIが集中
  → オッズが下がる
  → EVが低下
  → 回収率が低下
  → 複利では年単位で桁違いの差に

対抗策:
  1. 規模の優位性（先に大きく買ってオッズを下げる）
  2. 独自データによる予測精度の差別化
  3. 提携による資金力の統合
  4. 新しい馬券種（三連単）への展開
```

- 002の「ブレークスルーはコードの中にない」の実証: 競争優位は技術だけでなく資本・データ・規模
- **我々への含意**: 後発であることの不利を認識した上で、独自のエッジを見つける必要がある

### 3. 三連単への展開と購入インフラ

三連単の追加が示す設計上の要件:

```
馬券種の拡張に必要なインフラ:
  1. 買い目の爆発的増加への対応（三連単: 最大4,896通り/レース）
  2. PAT購入上限への対応（口座分割 or 上限管理）
  3. EV計算の拡張（3着内の全組合せの確率推定）
  4. Themis（011参照）への新馬券種の統合
```

- 008で「複勝/ワイド/馬連/3連複」のポートフォリオだったが、三連単が加わると:
  - 高分散・高リターンの馬券が加わる
  - ポートフォリオの効率的フロンティアが拡大
  - ただしEV計算の精度要求が格段に上がる

### 4. データ投資のROI ★★★

目視データ投資の示唆:

```
投資対効果:
  投資: 数千万円（推定）
  回収: 1ヶ月
  ROI: 1200%+/年（仮に3000万投資、年間でそれ以上の利益増）

含意:
  - 独自データ = 最強の参入障壁
  - 「データはパワー」= 公開データだけでは差別化困難
  - JRA-VAN等の公開データはライバルも使える → 同質化
  - 独自データの例:
    - レース映像からの解析データ（脚質詳細、位置取り、馬群の密度等）
    - 調教映像の解析
    - パドック映像の解析
    - 独自収集の馬場データ
```

### 5. スケールの天井

年間25億円購入で見えてきた制約:

- **流動性制約**: 馬券種×レースごとの売上に対する自分の購入比率が上がる → オッズへの影響大
- **PAT上限制約**: 口座あたりの購入点数/金額に上限
- **市場規模制約**: JRAの年間売上約3兆円 → 25億はその0.08%、特定レースでは数%に達する可能性
- 我々のレベルでは当面問題にならないが、設計時に考慮する価値はある

## 独自考察

### 2020年の成長曲線の分析

```
Q1: +343万    (6ヶ月累計)
Q2: +329万    (9.5ヶ月累計、Q2単体は-13万)
Q3: +1.04億   (12ヶ月累計、Q3単体は+1.01億)
Q4: +4.37億   (15ヶ月累計、Q4単体は+3.3億)

パターン:
  - 上半期は耐える期間（バンクロールを減らさない）
  - 下半期で爆発的に稼ぐ
  - 2019年も同パターン（上半期ドローダウン → 下半期で1,739万）
```

- 「流れを掴んだらほんの数週間で2倍3倍」= パレートの法則（008）の再現
- **含意**: 年間の収支は少数の「当たり期間」が支配する → 耐える期間にバンクロールを毀損しないことが最重要

### 「目視データ」の正体の推測

Matsukaze氏が「過去20年のレースの目視データ」に数千万円投資:

```
推測される内容:
  1. レース映像からの手動ラベリング（脚質、位置取り、不利、進路等）
  2. パトロールフィルムからの詳細データ
  3. 馬場の内外差・コース取りの数値化
  4. 各馬のレース中の加速/減速パターン

なぜ高価か:
  - 20年分 × 年間約3,500レース = 約70,000レース
  - 1レースあたりの目視解析コスト × 70,000
  - 人手による解析は自動化困難（2020年時点）

なぜ有効か:
  - JRA-VANの着順・タイムだけでは分からない情報
  - 「内を突いて勝った」「前が壁で追えなかった」等の文脈情報
  - 004の「特徴量が全て」の極地
```

### 競馬AIの競争構造モデル

```
フェーズ1（〜2019）: ブルーオーシャン
  - 競合少数、オッズへの影響軽微
  - 公開データ + 基本モデルで十分な利益

フェーズ2（2020〜）: 競争激化
  - 競合増加、オッズへの影響が出始める
  - 独自データ + 規模の経済が差別化要因
  - 提携・統合が始まる

フェーズ3（将来）: レッドオーシャン
  - 高EV馬券のオッズが効率化
  - 限られたエッジの奪い合い
  - 資本力 + データ + 速度が勝負を分ける

我々の参入タイミング（2025年）:
  - フェーズ2後半〜フェーズ3の境界
  - 公開データでのエッジは小さい可能性
  - 独自の視点・データ・モデルが不可欠
```

### 無観客競馬とオッズ構造

- 2020年はCOVID-19で無観客開催が続いた
- 無観客 → 現地のカジュアルファン（応援馬券購入者）が減少
- 001のショート戦略の前提（馬主クラブの応援馬券によるオッズ歪み）が弱まる可能性
- **環境変化に対するモデルの頑健性が問われる**
