# 競馬2.0の準備は出来ているか

- **出典**: Matsukaze.AI「競馬2.0の準備は出来ているか」 http://matsukaze.ai/?p=406
- **日付**: 2025-02-11（元記事: 2018-03-10）

## 要点

- Matsukaze氏が競馬予想において**決定的なブレークスルー**を達成したことを宣言した記事
- 「プログラムにしてたった10文字程度、この10文字に半年間かかった」
- 「設定6のアイムジャグラーを2000回転回して勝てるか」という勝負 → **数学的に有利な状況を作れたが、短期的な分散は避けられない**ことを自覚
- 「競馬予想には、その先がある。それを私は解いた」
- 電脳賞（AIによる競馬予想コンペ）に過去参戦し最下位 → 悔しさをバネに開発を継続

## AI開発への示唆

### 1. ブレークスルーは「シンプルな10文字」

- 半年の試行錯誤の末に辿り着いた本質は、たった10文字のコードで表現できるほどシンプルだった
- **複雑なモデルが正解とは限らない** → 本質的な洞察・特徴量・計算式こそが価値
- 考えられる「10文字」の候補:
  - ある特定の計算式（例: 期待値の算出方法）
  - 特定のフィルタリング条件
  - オッズと予測確率の関係式
  - 前記事(001)の「オッズ乖離度」に関連する閾値や変換式

### 2. 「設定6のジャグラー」の比喩が意味するもの

- パチスロの設定6 = 長期的に期待値がプラスだが、短期的には負けることもある
- つまり: **正の期待値を持つ賭けを、大量に繰り返す** ことで利益を得るモデル
- これは重賞レースの一発勝負ではなく、**平場レースも含めた大量のレースに賭ける**ことを示唆
- Matsukaze氏の後の実績（年間購入額17.5億円）はまさにこの戦略の具現化

### 3. コンペでの敗北から学ぶ

- 「電脳賞」で最下位 → 開発を継続 → ブレークスルー
- AIコンペの予測精度と実際の馬券収益は別物
- **予測精度の最大化 ≠ 回収率の最大化** という重要な示唆

## 独自考察

### 「10文字のブレークスルー」の正体を推理する

001の考察と合わせると、Matsukaze氏のブレークスルーは以下のいずれかに関連する可能性が高い:

1. **期待値ベースの馬券購入判断**
   - 予測確率 × オッズ > 1 の場合に購入（≒ 「EV = p * odds」）
   - 単純だが、正確な勝率予測モデルがあれば強力

2. **オッズを使った確率の補正**
   - 市場オッズ自体を特徴量として使い、自分の予測を補正する
   - 「市場の知恵」と「AIの知恵」の融合

3. **賭け金の最適配分（ケリー基準に類するもの）**
   - `f = (p * odds - 1) / (odds - 1)` のようなベットサイズ最適化
   - これは「たった10文字」に収まりうる

### 予測精度 vs 回収率の乖離

- 電脳賞で最下位 = 着順予測精度は低かった
- しかし後に年間+4.3億円 = 回収率は極めて高い
- **着順を当てることと、馬券で儲けることは本質的に異なる問題**
  - 着順予測: 分類/ランキング問題
  - 馬券収益: 期待値最大化問題（オッズ × 確率）
- AI開発の方向性として、「的中率」ではなく「期待値」を最適化すべき

### 大数の法則に基づく戦略

- 「設定6のジャグラーを2000回転」= 十分な試行回数で期待値に収束させる
- 競馬に当てはめると:
  - **年間3,000レース以上**に賭ける（中央競馬は年間約3,456レース）
  - 1レースあたりの期待値は小さくても、大量に賭けることで複利的に利益が積み上がる
  - リスク管理: 1レースに集中投資しない → 資金管理が重要
