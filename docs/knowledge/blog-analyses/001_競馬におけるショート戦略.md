# 競馬におけるショート（空売り）戦略

- **出典**: Matsukaze.AI「競馬でショートで勝てるのか」 http://matsukaze.ai/?p=308
- **日付**: 2025-02-11（元記事: 2018-02-16）

## 要点

- 投資の「ショート（空売り）」概念を競馬に応用する考察
- **核心アイデア**: "危険な人気馬"を特定し、その馬が負けた時に儲かるポジションを取る
- **具体的手法**: 対象馬以外の全馬の単勝を、払い戻しが均等になるように購入する
- **シミュレーション結果**:
  - 2017宝塚記念キタサンブラック（オッズ1.4倍）→ 投資6,300円 / 払戻10,890円 = **回収率173%**
  - DMMバヌーシー所有馬5戦のシミュレーション → 勝率は高いがオッズ6倍台では利益が出にくい
- **結論**: 「6倍台じゃ空売れない」 → オッズが極端に低い（1倍台〜2倍台）馬でないと成立しない

### オッズ歪みの発生要因（ブログが指摘するもの）

1. **有名人効果**: YouTuberの参戦→ファンが応援馬券を購入→オッズが実力以上に下がる
2. **大規模馬主クラブ効果**: DMMバヌーシー（会員約10,000人）の所有馬に応援馬券が集中
   - 全員が100円ずつ買っても100万円のインパクト
   - 1割が1万円買えばさらに大きな歪み

## AI開発への示唆

### 1. オッズの歪み検出モデル

- AIの予測確率とオッズ（市場の暗黙的確率）の乖離を検出するのは非常に有効
- **過剰人気馬の検出**が回収率向上の鍵
  - AIが算出した勝率 vs オッズから逆算した市場の勝率
  - 乖離が大きい馬 = 「危険な人気馬」or「妙味のある穴馬」

### 2. 馬券戦略の最適化

- 「ショート」を直接実装するのではなく、**オッズの歪みを利用した期待値ベースの馬券購入**が現実的
- 具体的には:
  - 過剰人気馬を「買わない」（消し馬として活用）
  - 過小評価馬を「積極的に買う」
  - ワイドや三連複で過剰人気馬を外した組み合わせを構築

### 3. オッズ歪みの要因を特徴量化

- 馬主クラブの会員規模
- SNS上のバズ度合い（話題性）
- 有名人・インフルエンサーとの関連

## 独自考察

### ショート戦略の数理的限界

- 控除率（JRA単勝: 約20%）が存在するため、全馬買いは本質的に期待値がマイナス
- ショートが成立するのは **特定の馬のオッズが市場全体の控除率分以上に歪んでいる** 時のみ
- つまり、オッズ1.4倍の馬の「真の勝率」が71%未満であれば、他馬全部買いが期待値プラスになり得る
- **定量化**: ある馬のオッズをO、控除率をcとした時、ショートの期待値は `(1 - c) * (1 - 1/O) / (1 - 1/O) - cost` で近似できる

### AIへの応用: 「逆張りモデル」の可能性

- 多くの競馬AIは「勝つ馬を当てる」ことに注力するが、**「負ける人気馬を当てる」**方がアルファが大きい可能性
- 理由: 人気馬は情報が多く市場効率が高いはずだが、感情的なバイアス（ファン心理・応援馬券）で歪む
- **実装案**:
  1. 通常の勝率予測モデルに加え、「オッズ乖離度」を計算
  2. 乖離度が閾値を超える人気馬を「消し馬」としてフラグ
  3. その馬を除外した馬券（ワイド・三連複）を購入

### Matsukazeの実績との関連

- このブログの著者（Matsukaze.AI）は2020年時点で年間+4.3億円の利益を計上している大規模AI馬券購入者
- 馬券種はワイド・枠連・複勝が中心（関連記事より）
- おそらく本記事の「オッズの歪み検出」のコンセプトを発展させたモデルを運用していると推測

### 馬主クラブ効果の現在（2025年時点での考察）

- 2018年時点のDMMバヌーシーに加え、現在はさらに多くの大規模クラブが存在
- シルクレーシング、キャロットファーム、サンデーレーシングなどの大手クラブ馬も同様の歪みが発生しうる
- ただし市場が学習しているため、以前ほど極端な歪みは減少している可能性
- **検証すべき仮説**: 大規模馬主クラブ所有馬のオッズは実力に対して系統的に低い（過剰人気）か？
