## 注目ポイント

1. **「データ加工が精度の8割」** — 特徴量エンジニアリング最重要の認識はうちと一致
2. **Optunaでハイパーパラメータ最適化** — 回収率120%→144%→151%と大幅改善。うちは未導入、検討価値あり
3. **回収率151%達成** — 特徴量追加 + Optuna。どの特徴量が効いたか後続章で確認したい
4. **更新履歴から見る進化** — 2020/11〜2024/11、約4年の継続開発。上がりタイム追加、血統データ追加等
