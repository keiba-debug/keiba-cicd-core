## 実運用のフロー

```
1. 事前準備（週初め〜前日）
   - 全過去データで学習（train/test分割なし）
   - 当日出走馬の過去成績を再スクレイピング
   - 血統データ更新

2. レース直前（馬体重発表後）
   - 出馬表スクレイピング
   - preprocessing → merge_horse_results → merge_peds → process_categorical
   - predict_proba → レース内標準化スコア

3. 購入判定
   - スコア1.3超えが3頭以上 → 三連複BOX
   - スコア1.3超えが2頭以下 → 単勝
```

## 実績（2021/5/30 東京・中京）

### サマリー
- **22レース**に参加
- **的中10レース** / 不的中12レース
- **投入**: 約10,200円（推定）
- **払い戻し**: 16,120円
- **回収率**: 約158%
- **ハイライト**: 日本ダービー 4番人気シャフリヤール単勝的中（600円→7,020円）

### 購入戦略の内訳
| 馬券種 | レース数 | 備考 |
|--------|---------|------|
| 単勝（1〜2頭） | 18R | 基本戦略 |
| 三連複BOX | 2R | スコア1.3超え3頭以上時 |
| 三連単BOX | 2R | 高スコア3頭以上時 |

### 注目ポイント
- **人気馬を着実に当てる** + **たまに中穴を拾う**パターン
- 中京5R: 7番人気単勝的中（200円→3,460円）が大きい
- 単勝中心の戦略が結果的に安定

## 参考になるポイント

1. **本番時は全データで再学習** — うちはtrain期間を固定しているが、直近データも学習に使う方がエッジが出る可能性
2. **単勝中心が最も安定** — Ch.05の理論（シャープレシオ）と一致。うちのVB戦略も単複中心で正しい方向
3. **スコア閾値による馬券種の切り替え** — 高確信→単勝、中確信→三連複/単は合理的
4. **中穴（4〜7番人気）の単勝が利益の源泉** — うちのGap戦略と同じ。市場が過小評価している馬を拾う
