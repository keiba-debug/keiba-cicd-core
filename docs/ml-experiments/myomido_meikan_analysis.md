## 1. 書籍の核心概念

### 1.1 「妙味度」とは

**定義**: 人気バイアスを除去し、均等拡尺方式で算出した5年重み付け平均の補正回収率指標。基準値100。

- **100超** = 実力の割に人気がない（配当的に妙味がある）
- **100未満** = 買われすぎ（避けるべき）
- **120超** = 単勝・複勝で単独勝負可能
- **110超** = 軸馬として適切
- **100以上** = 紐（相手馬）に適切

### 1.2 妙味度の算出プロセス（3ステップ）

#### Step 1: 人気バイアス除去 → 「補正複勝配当」
- 複勝オッズと複勝回収率には約10%の構造的偏りが存在
  - 低オッズ（人気馬）ほど複勝回収率が高い
  - 原因: 「1倍台の複勝なら単勝を買う」心理 + JRAプラス10
- 複勝オッズに応じた係数を乗算して補正
  - 低オッズ馬 → より小さな係数（回収率を下げる方向に補正）
  - 高オッズ馬 → 補正配当を低くする（まぐれ当たりの排除）

#### Step 2: 均等拡尺方式 → 「補正回収率」
- 通常の「均等買い」（全て同額）ではなく、**オッズに反比例する金額**で購入したと仮定
- 人気馬 → 購入金額大 / 人気薄 → 購入金額小
- これにより人気薄の大当たりに左右されにくい安定した回収率を算出

#### Step 3: 重み付け平均 → 「妙味度」
- **年度重み**（直近重視）: 2021年=0.2, 2022年=0.4, 2023年=0.6, 2024年=0.8, 2025年=1.0
- **騎乗数重み**: 騎乗数が多い年度ほど重みを大きくする
- 定数+21（騎手の場合）を加算して分布を調整

### 1.3 「利益度」とは

**定義**: 直近1年間の騎乗数 × (妙味度 - 100)

- その騎手/種牡馬/厩舎の馬券を1年間買い続けた場合の期待利益の近似値
- 妙味度が高くても騎乗数が少なければ利益度は低い
- **利益度が高い = 実際に儲かる可能性が高い**（妙味度×ボリュームの組み合わせ）

### 1.4 評価の3次元構造

本書では各対象（騎手/種牡馬/厩舎）を以下の条件軸で多次元評価：

| 軸 | 騎手 | 種牡馬 | 厩舎 |
|----|------|--------|------|
| 馬場 | 芝/ダート | 芝/ダート | 芝/ダート |
| 距離 | 短/中/長 | 短/中/長 | 短/中/長 |
| 回り | 左/右 | 左/右 | 左/右 |
| 競馬場 | 10場 | 10場 | 10場 |
| クラス | 新馬/未勝利/1-3勝/OP/重賞 | 同左 | 同左 |
| 馬場状態 | 芝道悪/ダ道悪 | 同左 | 同左 |
| 年齢 | - | 2/3/4/5歳以上 | 2/3/4/5歳以上 |
| ローテ | - | 距離短縮/延長 | 休明け1-3走/連闘 |

## 2. 本書から得られた主要な知見

### 2.1 市場非効率性の構造的パターン

#### パターン1: ダート血統は常に過小評価される
- 「**ダート血統を制するものが競馬を制す**」が本書のキーメッセージ
- 利益度Top3はダート系種牡馬が独占：
  - ルヴァンスレーヴ（利益度13,866 / 9条件で1位）
  - シニスターミニスター（9,446）
  - モズアスコット（8,135）
- 有名芝系種牡馬は軒並み大幅マイナス：
  - エピファネイア（-5,708）、モーリス（-4,407）、ドゥラメンテ（-3,634）
  - ハービンジャー（-2,687）、ハーツクライ（-2,071）

#### パターン2: 知名度上昇が妙味度を下げる
- 坂井瑠星騎手：知名度向上→過剰人気化→妙味度低下
- 杉山晴紀厩舎：TVドラマ出演→2025年単年妙味度86に急落

#### パターン3: 厩舎の妙味度は最も気づかれにくい
- 「この厩舎だから買おう」という層が少ないため妙味が持続
- 5年連続プラスの厩舎多数（千田輝彦、大根田裕之、北出成人等）

#### パターン4: 若手騎手はデビュー初期に妙味度が高い
- デビュー当初は「ダート」「短距離」で特に高い
- 人気が出ると「芝」の妙味度から下がっていく

## 3. KeibaCICDへの応用可能性

### 3.1 ★★★ 条件別妙味度特徴量の導入

本書の手法を参考に構築可能な特徴量：
```
jockey_myomido_surface     # 騎手×馬場の妙味度
jockey_myomido_course      # 騎手×競馬場の妙味度
sire_myomido_surface       # 種牡馬×馬場の妙味度
sire_myomido_dist_change   # 種牡馬×距離変更の妙味度
trainer_myomido_surface     # 厩舎×馬場の妙味度
trainer_myomido_rotation    # 厩舎×ローテーションの妙味度
```

実装方針: 過去5年の複勝回収率→人気バイアス除去→均等拡尺→年度重み付け

### 3.2 ★★★ 種牡馬×距離変更の交互作用

- エピファネイア距離短縮: -6,399 / キズナ距離延長: +3,684
- 現在のrotation_featuresに種牡馬交互作用なし → 追加で効果大

### 3.3 ★★ 人気バイアス補正（bet_engine）

- 複勝オッズの構造的偏り約10%をKelly計算に反映
- 低オッズ帯の複勝→微増、高オッズ帯→微減

### 3.4 ★★ 人気度変化率特徴量

- 騎手/厩舎の直近vs過去の平均人気比較
- 人気急上昇=妙味度低下中 → ベット控えめ

### 3.5 ★★ 利益度3000超えパターンDB

- 高妙味度の[対象×条件]をDB化→VBフィルター補助指標

### 3.6 ★ 均等拡尺方式バックテスト

- VB戦略など高オッズ狙いの評価安定化

### 3.7 ★ 厩舎×ローテーション交互作用

- 休み明け1-3走目の厩舎別差異を特徴量化
