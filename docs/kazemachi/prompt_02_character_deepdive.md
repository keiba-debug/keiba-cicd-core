## 深掘りしてほしい項目

### 1. 口調サンプル（各キャラ10パターン以上）
以下のシチュエーションでの発言を書いてください：
- 自分の本命が1着になった時
- 自分の本命が大敗した時
- 他のキャラの予想が当たった時（素直に認める？悔しがる？）
- 全員外れた時
- 大穴が来た時
- 雨で馬場が変わった時
- 出走取消が出た時
- 進行役にツッコまれた時の返し

### 2. 弱点の具体的エピソード
各キャラの弱点が発動する具体的な場面を3つ書いてください。
「こういう時にこうなる」のレベルで。

### 3. 関係性の温度感
以下の場面で各ペアがどう反応するか：
- 2人だけになった時（進行役がトイレに行った等）
- 片方が3連敗してる時、もう片方はどう接する？
- 一緒に馬券を買いに行ったら何が起きる？

### 4. バックストーリーの断片
各キャラについて、番組内で「チラッと匂わせる」セリフを5つ書いてください。
直接説明するのではなく、会話の中で自然に出る一言で。
例：「……昔、同じことを言ったやつがいたな」

### 5. 進行役の「実は詳しい」匂わせ
進行役が普段は予想しないのに、思わず本音が漏れてしまうシーンを5つ。
すぐに取り繕うが、予想家3人が「今の何？」と反応する瞬間。
