# [Phase 0・レガシー] 予想TVキャラクター ブレストプロンプト① — キャラ原案

> **注意**: このプロンプトは旧「予想TV」時代（Phase 0）のもの。
> プロジェクトは `kazemachi` に進化し、作家チーム4人体制に移行済み。
> 現在のプロンプトは `prompt_04_character_factory.md` 以降を参照。
> このファイルは履歴として残す。

以下のプロンプトをClaude / ChatGPT / Gemini 等に投げて、キャラクター案を出してもらってください。
各AIの回答を比較して、刺さる要素をピックアップしていくイメージです。

---

## プロンプト

```
あなたはキャラクターデザイナー兼脚本家です。
以下の番組企画に登場するキャラクター4人（予想家3人＋進行役1人）の原案を作ってください。

## 番組概要

AIアニメの競馬予想番組「予想TV」。
毎週のレースについてキャラクターたちが議論・予想する。
ターゲットは競馬に詳しくない人がラジオ感覚で楽しめるレベル。
予想に関しては熱い対立が生まれる番組。

## キャラクター構成

- 予想家3人：「データ派」「直感派」「経験派」の3タイプ
- 進行役1人：アレクシア（ハンドイットオーバー）のような冷静ツッコミ型

## 予想スタイルの対立構造

| タイプ | 予想根拠 | 強み | 弱み |
|--------|---------|------|------|
| データ派 | 数字・統計・確率 | 安定した回収率 | 想定外に弱い |
| 直感派 | 馬の雰囲気・第六感 | 荒れるレースで化ける | 外れると言い訳できない |
| 経験派 | 過去の記憶・パターン認識 | 特定条件で異常に強い | 記憶が美化されがち |

## 各キャラに必要な要素

1. **名前**（日本語でもカタカナでもニックネームでも可）
2. **性別・年齢感**（声のトーンが全員違うように）
3. **性格と口調**（具体的なセリフ例を3つ以上）
4. **予想スタイルの詳細**（何を見て、どう判断するか）
5. **弱点・欠陥**（予想以外の人間的な弱さも含む）
6. **他の3人それぞれとの関係性**（好き嫌い、リスペクト、ライバル心など）
7. **バックストーリーのコア**（1〜2行。この人は何を背負っているか）

## 重要な設計方針

- 弱点は「予想が外れる」だけでなく、人間的な欠陥も入れる
  （例：他人に流されやすい、自信がない、馬名を間違える、すぐ脱線する）
- 進行役は予想をしない。しかし視聴者が「この人、実は一番詳しいのでは？」と
  感じる瞬間がたまにある。なぜ予想しないかは謎として残す
- 3人の関係は単純な対立ではなく、場面によって味方が変わる三角構造
- 全員がラジオで声だけ聞いても区別できるように、声のトーンや話し方を変える
- キャラ同士のバックストーリーには隠された繋がりがある（後々明かされる伏線）

## 出力形式

キャラクターごとに上記7項目を書いてください。
その後、4人の関係性マップと、第1話の冒頭3分の会話サンプルも書いてください。
会話サンプルでは各キャラの口調・性格・対立構造が伝わるようにしてください。
```

---

## 使い方

1. 上のプロンプトをそのまま複数のAIに投げる
2. 各AIの回答で「このキャラのこの部分がいい」をメモ
3. いいとこ取りで合成キャラを作る
4. → プロンプト②（深掘り）へ進む
