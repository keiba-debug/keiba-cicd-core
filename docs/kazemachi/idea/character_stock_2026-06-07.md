## 自動セリフ生成パイプライン（2026-06-07 追加）

### コンセプト
レース結果をAIエージェントに送り、キャラのペルソナに合ったセリフを生成→音声出力する仕組み。
毎週の開催後にキャラが自動で喋る = コンテンツ更新コストがほぼゼロになる。

### パイプライン
```
KeibaCICD（結果取得）
    ↓
Claude API（キャラペルソナ + レース結果 + 馬券結果 → セリフ生成）
    ↓
VOICEVOX（セリフ → 音声ファイル）
    ↓
YouTube / X に投稿
```

### セリフのイメージ
- **エミ**（複勝転がし3回成功）：「あらあら〜、3着でいいって言ったでしょ。でもまあ…もう一回だけね」
- **アキヒコ**（IDM上位が4着）：「…想定内のドローダウンだ」（実は凹んでる）
- **ソラ**（応援馬が外れ）：「……合ってると思ったんだけどな」（出馬表の色ペンを持ち直す）

### 技術的な実現性
- KeibaCICD：自動投票・結果取得 → 実装済み
- Claude API：キャラペルソナ + 結果 → セリフ生成 → APIで呼び出すだけ
- VOICEVOX：セリフ → 音声ファイル → 実装済み
- 繋ぐだけで動く。自動化パイプラインの最初の部品として優先度高い

### 実装優先度
このパイプラインを最初に作ることで、コンテンツ更新が自動化される。
モデル・キャラ設定・ビジュアルが揃う前でも「喋るキャラ」は先に動かせる。
