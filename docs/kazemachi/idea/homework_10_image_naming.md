> **画像フォルダ**: `C:\KEIBA-CICD\キャラクター\オーディション\`

## プロンプト（ユキ宛て）

```

■ 宿題⑩ — 確定キャラ画像の整理・命名

コア5人が確定した。各キャラのフォルダに画像が大量にあるが、
ファイル名がハッシュ値やツール名のままで中身がわからない。

あなたの仕事は、全画像を見て以下を行うこと。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ やってほしいこと
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### Step 1: 全画像を見て分類

添付された画像を全て見て、各画像について以下を記録：

1. **シチュエーション**（どこで何をしている場面か）
   - 例: パドック、スタンド、テーブル、食堂、帰り道、自宅、WINS
2. **ポーズ・構図**
   - 例: バストアップ、立ち姿、座り、横顔、2ショット
3. **表情・感情**
   - 例: 伏し目、困り顔、驚き、笑顔、無表情、考え込み
4. **記号アイテム**（画像に映っているもの）
   - 例: 双眼鏡、出馬表、缶コーヒー、ノートPC、ノート
5. **画風・ツール**
   - 例: SD4.5、SD4.0、ChatGPT、Gemini
6. **品質評価**（S/A/B/C）
   - S: そのまま使える。ベスト候補
   - A: 良い。微調整で使える
   - B: 方向性は合ってるが修正必要
   - C: 落選候補

### Step 2: リネーム案を出力

以下の命名規則でリネーム案を出力してほしい：

```
{キャラ名}_{シチュエーション}_{表情}_{ツール}_v{番号}.{ext}
```

例：
```
ソラ_パドック_双眼鏡_伏し目_sd45_v01.jpg
ソラ_テーブル_出馬表書込_困り顔_sd45_v01.jpg
ソラ_ゲンさんと2shot_驚き_sd40_v01.jpg
ソラ_立ち姿_正面_微笑_cgpt_v01.png
ゲン_スタンド_双眼鏡_横顔_sd45_v01.jpg
ゲン_ベンチ_BLACK缶2本_夕暮れ_sd45_v01.jpg
マキ_クリップボード_名札_半笑い_sd40_v01.jpg
ナルミ_PC前_腕組み_考え込み_sd45_v01.jpg
氷河期_アパート_ThinkPad_夜_sd45_v01.jpg
```

ツール略称：
- sd45 = SeedDream 4.5
- sd40 = SeedDream 4.0
- cgpt = ChatGPT/DALL-E
- gmni = Gemini

### Step 3: 用途タグの提案

各画像に「いつ使えるか」の用途タグを1つ以上つけてほしい：

- **プロフィール**: キャラ紹介用のメイン画像
- **予想回**: レース前の予想シーン
- **結果回**: レース後のリアクション
- **日常**: 平日投稿用
- **関係性**: 2人以上のやり取りが見える
- **感情**: キャラの内面が見える瞬間
- **X投稿**: SNS映えする1枚
- **Note**: 長文記事の挿絵

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 出力形式
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

キャラごとにテーブルで出力：

| 2 | ... | ... | ... | ... | ... |

（全キャラ分）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 出力ルール
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 冒頭に【ユキ】
- 全画像分を出力すること（省略禁止）
- 品質Cの画像には「落選理由」を必ず書くこと
- Markdown形式
```

## 使い方

1. `writers/yuki.md` のシステムプロンプトを前置き
2. 上記プロンプトを投げる
3. 添付: 各キャラフォルダの画像を全て添付
   - ソラ36枚は多いので、2回に分けて投げてもOK
4. 返ってきたリネーム案をもとにスクリプトで一括リネーム

### リネーム実行（カカシが担当）
ユキの出力をもとに、リネームスクリプトを生成して実行する。
