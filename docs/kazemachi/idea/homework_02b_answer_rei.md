# 宿題②b 回答 — 「競馬×キャラクター×物語」の軸を探す

`homework_02b_brainstorm.md` に対する回答。思いつくままに書く。

---

## 1. 「物語」とは何か？

**答え：「15年前のあの日が、今も続いている」という物語。**

これはストーリーとしては「一本の物語」だが、日常の積み重ねの中でゆっくり明かされていく。

### 実装イメージ

- 毎回の予想投稿 → 何の変哲もないように見える
- 3ヶ月続けると「ゲンさんが何度も『あのコース』について言及している」に気づく
- さらに3ヶ月すると「その全員の発言が、実は1つの日付を指してる」に気づき始める
- 最終的には「あの日、この人たちに何が起きたのか」という問いが浮かぶ

**「ストーリー性を感じる」= 単発の予想や雑談ではなく、少しずつ全体像が浮かぶ快感。**

---

## 2. このプロジェクトの根幹（最重要）

> フィクションの連載ドラマは「確定した物語」を追うだけ。  
> でも競馬は「毎週現実が起きる」。

キャラクターの記憶・感情・秘密と、実際のレース結果が絡まることで、毎週が宝探しになる。

- 作者が決めた結末に向かって読者が「追いかける」のではない
- **誰も結末を知らない。現実が毎週更新される**
- キャラが「このコースは昔…」と言った翌週に、本当にそのコースで何かが起きる
- それが偶然なのか計算なのか、視聴者にはわからない
- **「どっちかわからない」が最大の快感**

→ `project_plan.md` の根幹として追記済み。

---

## 3. 「軸」とは何か？

**軸は「競馬という同じ趣味で集まった6人が、実は1つの秘密を共有している」という暗黙の了解。**

- キャラたちを集めている力：**「昔のあの日」**
- 読者がこの世界に戻ってくる理由：**「その秘密が何なのか」という謎**
- 差別化ポイント：**「実際のレース結果が物語を動かす」という仕組み**

具体的には：
- ゲンさんが「このコースで15年前に見た馬に似た馬」が出走する
- その馬がどう走るかで、謎解きが一歩進む
- つまり**現実の競馬が、物語の章立てを決める**

---

## 4. キャラクターについて

### 今の6人の役割（新方向性で再整理）

| キャラ | 「あの日」との関与 | 物語上の機能 |
|--------|------------------|------------|
| マキ（MC） | 謎の進行役。何を知ってるか不明 | 視聴者の目線代理 |
| ゲンさん | 「あの日」を一番長く知ってる人 | 記憶の語り部 |
| テツ | 別視点で「あの日」を知ってる | 詩的に断片を語る |
| ナルミ | データで「あの日」を検証しようとしてる | 科学的な突破口 |
| ソラ | 「あの日」を感覚で覚えてる（自覚なし） | 無自覚な証言者 |
| フジさん | 全員の成績・行動を記録している | **実は一番知ってる可能性** |

**全員が「あの日」への関与度が違う。これで複数角形の関係が成立する。**

### 人数は適切か

今の6人で十分。多すぎない。

### 主人公は誰か

**群像劇だが、実質の中心はフジさん。**

- 全員の記録を握っている
- 一番冷静に「あの日」を俯瞰できる立場
- 他の5人は断片的に思い出してるが、フジだけが「全体を知ってる可能性がある」
- 表面的には「マキが進行役」に見える → **実はフジが黒幕**という二層構造

## 5. アウトプットの形

### X投稿（毎日）

各キャラの「今日の一言」＋画像。

例：
- ゲンさん：「このコース、水が出た日のことを思い出す。今週も楽しみだ。」
- ソラ：「あの馬、なんか懐かしい顔してた。なんでだろ。」
- フジさん：「今月の成績、振り返りました。ご確認ください。」（タブレット画像に極小の数字）

### Note（週1〜月1）

- ゲンさんの昔話・完全版（3000字）
- フジさんの月間成績分析（統計＋意味深な一言）
- テツのエッセイ：「あの馬のことを思い出した」

### 短尺動画（毎週）

TikTok/YouTube Shorts：30秒〜1分。**言葉ではなく映像で物語を進める。**

例：
- ゲンさんがパドックで「あ、昔見た…」と止まる（言葉にしない）
- ソラが双眼鏡をふと覗いて、表情が一瞬固まる
- フジがタブレットに「15年前」という日付を入力する一瞬

### 予想コンテンツと物語の比率

- **土日：予想（60%）＋ その予想に絡まる「あの日」への言及（40%）**
- 月〜金：物語・日常（80%）＋ 競馬知識（20%）

## 6. 「伏線と回収」を実際のレース結果と絡める仕組み

**これがこのプロジェクト最大の独自性。**

### 仕組みの設計

```
1. 設定段階
   「15年前のあの日」で、特定のコースで「ある馬」が出走予定だったことを設定

2. 積み重ね段階（1〜3ヶ月）
   ゲンさんが「このコースは昔…」と何度も言及
   視聴者は「何の話？」と思ってる

3. 実装段階（月1程度）
   「そのコースに、その時代の血を引く馬が出走する」という実際のレース

4. 回収段階
   その馬の走りで「謎が一つ明かされる」
```

### 具体例

- 1月：ゲンさんが「このコース、水が出てた…」と言及
- 2月〜3月：その言及が何度も出てくる
- 4月のレース：そのコースに「15年前の馬の孫世代」の馬が出走
- その馬がどう走ったか → ゲンさんが「そっか、お前もか」と言う

**この瞬間、視聴者が「あ、全部つながってる」と気づく快感。**

---

## 7. 収益との接続

### 何にお金を払うのか

**「謎を解きたい」という欲求に金を払う。**

| 区分 | 内容 |
|------|------|
| 無料 | 毎日のX投稿、Noteの無料部分（表層の断片） |
| 有料・月額 | ゲンさんの完全版昔話、フジの全データ分析、マキの日記（伏線満載） |
| 有料・単発 | 「15年前のあの日」についてのトークセッション音声 |
| グッズ | 双眼鏡（ソラ）、ノート（ゲンさん）、タブレット（フジ）などキャラ記号グッズ |

**「謎が気になる」が動機 → 「もっと知りたい」が課金につながる。**

---

## 8. 「これは絶対やめたほうがいい」リスト

- **謎を解き明かしすぎる**：20話で全部明かしたら、その後の展開がなくなる
- **「あの日」をただのトラウマにする**：悲劇的すぎるとダメ。「良い思い出とも言える何か」くらいのバランスが欲しい
- **競馬の知識がないと楽しめない設計**：予想の詳細は「わからなくていい」くらいのスタンスを維持
- **伏線のためにキャラを動かす**：ユキの指摘通り、伏線はキャラの行動の結果として自然に生まれるもの

## まとめ（レイ）

**このプロジェクトの強みは「実際のレース結果が物語を動かす」という仕組み。**

フィクションの連載ドラマは「確定した物語」を追うだけ。  
でも競馬は「毎週現実が起きる」。

その現実を、キャラたちの記憶・感情・秘密と絡ませることで、**毎週が宝探しになる**。  
視聴者が「あ、これのことか！」と気づく瞬間がある。

この快感を中心に設計すれば、長く続く世界が作れる。
