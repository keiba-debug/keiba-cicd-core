## 6. 「魅力的な何か」を狙う（地味になりがちなとき）

49・SE・氷河期という型は、生成モデル側では「記号の平均」に寄りやすい。**イケメン争奪ではなく**、**絵の中で人が立つ／同じ人と通る／感情が一瞬伝わる**を「魅力」と定義すると運用しやすい。

### 6.1 何に効くか（顔だけに載せない）

| レバー | 例 |
|:---|:---|
| **光** | 蛍光灯一辺倒にしない。窓の残り光・サイドからの柔らかい光・目に小さな反射（読める文字は出さない） |
| **所作** | 缶の握り、赤ペンの角度、新聞との距離感。P06 のように**手元と紙を主役**にすると性格が乗りやすい |
| **目だけ** | 口角はほぼ動かさず、**目元だけわずかに柔らかい**。深い皺は付けない前提は維持 |
| **造形の具体語** | 「魅力的」より **1〜2語だけ**：やや幅広の穏やかな目／落ち着いた眉の角度／直線寄りの鼻筋 など。多すぎると別人化 |

### 6.2 年齢スイープより先に試すこと

- **添付同一人物は維持**したまま、**シチュと光**だけ変える（§1・§2）。年齢語をいじるより再現性が高いことが多い。  
- 抽象的な「ハンサム」「色気」単体は**平均顔化**しやすいので、上表のどれかに言い換える。

### 6.3 共通ロックに足す「魅力パス」用（任意・コピペ）
