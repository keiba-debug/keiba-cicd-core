# 番外編 × 伏線構造

> 本編は予想バトル。番外編にこっそりヒントを仕込む。

---

## 番外編の種類

### ① キャラ紹介ショート（新規視聴者向け…のはずが）
```
「データ派の◯◯を紹介します」
- 趣味: スプレッドシート作成
- 好きな食べ物: カレー（辛口）
- 好きな馬券: 複勝
- 座右の銘: 「数字は嘘をつかない」
- 苦手なもの: ————（一瞬ノイズが入る）
```
最後だけ不自然。普通の視聴者は「演出かな」で流す。

### ② 回想ミニエピソード（1-2分）
- 「◯◯が競馬を始めたきっかけ」
- 穏やかなBGMで語られる思い出話
- でも1つだけ辻褄が合わない描写がある
  - 「あの頃は3人でよく競馬場に行ってた」→ 今のメンバーは4人なのに？
  - 「初めて万馬券を当てた日、隣にいたのは…」→ そこで映像が切れる

### ③ 休日エピソード（日常回）
- キャラの私生活をちょっとだけ見せる
- データ派の部屋 → 壁にびっしり数式…の中に1枚だけ手書きの手紙
- 経験派の本棚 → 競馬本に混じって1冊だけ背表紙が見えない本がある
- 直感派の冷蔵庫 → なぜか2人分の食材がある（1人暮らしのはず）

### ④ Q&Aコーナー（視聴者の質問に答える）
```
Q:「◯◯さんはなぜ競馬予想家になったんですか？」
A:「なったんじゃなくて……なるしかなかった、かな。
　 あ、冗談冗談。好きだからだよ（笑）」
```
笑って誤魔化すけど、最初の一言が本音。

### ⑤ 進行役の独白（ED後のCパート、ごく稀に）
- 本編が終わって数秒の暗転後
- 進行役の声だけが入る
- 「……今週も、全員残りましたね」
- 3秒で終わる。次の週は何事もなく始まる

## 仕込み方の原則

### やること
- **99%は普通のコンテンツ**。面白い紹介、楽しい日常回
- 1%だけ「ん？」ってなる要素を入れる
- 気づかなくてもコンテンツとして成立する
- 気づいた人が「他の番外編も見返そう」ってなる

### やらないこと
- 毎回入れない（月1回あるかないか）
- 直接的に怖い描写はしない
- 答えを出さない（考察の余地を残す）
- 本編の予想バトルの邪魔はしない

---

## 伏線の散らし方イメージ（半年スパン）

```
月1: キャラ紹介（普通）
月2: キャラ紹介（普通）…最後に一瞬ノイズ
月3: 回想「競馬を始めたきっかけ」（普通に見える）
月4: Q&A（笑って流す本音が1つ）
月5: 休日回（部屋の背景にヒント）
月6: ED後Cパート、進行役の独白（初めて明確な「異変」）

→ ここで考察班が起動する
→ 月1-5を見返すと「あ、あの時の…」が繋がり始める
```

## プラットフォーム別の仕掛け

| 番外編タイプ | 掲載先 | 狙い |
|-------------|--------|------|
| キャラ紹介 | Instagram/X | 新規獲得＋伏線 |
| 回想ミニ | Note/YouTube | 世界観の深掘り＋伏線 |
| 休日回 | Instagram Stories | 消えるコンテンツに伏線→見逃した人が悔しがる |
| Q&A | X/Note | 参加型＋伏線 |
| 進行役独白 | YouTube本編のED後 | 最後まで見た人だけのご褒美 |

Instagramの24時間で消えるStoriesに伏線入れるの、えぐいけど面白い。
「あの日のStoryに何か映ってたらしい」って噂が広がる構造。
