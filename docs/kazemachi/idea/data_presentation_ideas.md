# データの見せ方アイデア集

> 生データは出せない。でも分析結果と抽象化された結論で、
> むしろ生データより面白く見せる方法を考える。

---

## 基本方針

**制約を強みに変える**:
- 数値の羅列 → キャラの言葉で語る → 初心者にも伝わる
- 専門用語 → 日常語に翻訳 → ラジオ感覚で聞ける
- 指数そのまま → ランク/比喩/ビジュアル → 一目で直感的にわかる

## キャラ別の語り口の違い（同じデータ、違う表現）

### 例: ある馬がVB候補（P% 30, オッズ 12倍, AR 1.4）の場合

**データ派**:
「確率30%でオッズ12倍。期待値的に明らかにプラス。
　こういう馬を淡々と買い続ければ長期的に勝てる」

**直感派**:
「12倍ってことは皆この馬を舐めてるってことだろ？
　俺はこの馬、来ると思う。雰囲気が違う」

**経験派**:
「このパターン、前にも見たことある。
　人気が足りない実力馬ってのは、来る時は一気に来る」

**進行役**:
「珍しく3人の意見が一致してるわね。…逆に怖いけど」

→ **同じ結論でも語り口が違うからキャラが立つ**

## 核心

**「数字を翻訳してエンタメにする」のが予想TVの本質的な価値。**

生データをそのまま出すサイトは山ほどある。
でも「その数字が何を意味するのか」を面白く伝えられるコンテンツは少ない。
制約があるからこそ、ここで差別化できる。
