# kazemachi キャラクター・オーディション素材集

## 前置き：設計思想

今回のキャラ出しで意識したのは3つ。

1. **「根拠の違い＝人格の違い」を最大化する配置**――同じ予想スタイルの人間を2人作らない
2. **「15年前のあの日」に対する距離のグラデーション**――直接関わった者、間接的に影響を受けた者、何も知らない者、を混ぜる
3. **考察班が「この人とこの人、繋がってない？」と疑えるフックを全員に仕込む**

ソラは固定。残りは全部白紙のつもりで書いた。既存キャラの面影が残っている者もいるが、全員「別人として成立する」ように設計してある。

### 裏の設計

- **生活の時間割**：平日6:30起床→7:30出勤→21:00帰宅→YouTube流し見→0:00就寝。競馬のことは日曜しか考えない

- **続けている理由**：パドックで馬を見ている時間だけ、仕事のことを完全に忘れられる
- **一番記憶に残っているレース**：2018年エリザベス女王杯。リスグラシューの馬体を見て「今日は来る」と確信して当てた。あの「見えた」感覚を追い続けている

- **面白くなる組み合わせ**：ソラとの「直感 vs 直感」対話。同じ感覚派なのに結論が真逆になる回が、視聴者に一番刺さる

**伏線フック**：双眼鏡の傷。いつついたのか、本人は「覚えてない」と言う。→ 回収予定：中盤以降。15年前のあの日との接続候補。ただし直接ではなく「誰かから貰ったもの」の可能性を残す。

- **面白くなる組み合わせ**：ナルミ的なデータ派の若者との対比。世代間の「勝てない理由」の違いが浮き彫りになる

**伏線フック**：25年分の手帳。その中に「15年前のあの日」の日付のページがあるはず。そこに何が書いてあるかは本人も忘れている、と言う。→ 設置時期：序盤で「手帳」の存在だけ見せる。回収予定：終盤。手帳のそのページを開いた時、書いてあるのは収支ではなく、たった一行の別の言葉。

- **続けている理由**：レース映像を見ていると「正解がある」と感じる。仕事の映像編集は正解がない（クライアントの好み次第）。競馬は結果が出る。その明確さが救い
