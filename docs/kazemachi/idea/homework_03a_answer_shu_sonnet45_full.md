【シュウ】（モデル名: Claude Sonnet 4.5）

> 取材ノート — kazemachi の世界にいる人間、10人分
> 作成日: 2026-03-27
> 取材者: シュウ（フィールドリサーチ・ルポルタージュ担当）

> **前置き（シュウより）**
>
> 設計書を読んだ。よく作られていると思う。ただし、コア6人はまだ「タイプ」で動いている。経験派、データ派、直感派——これは属性であって、人間ではない。人間は属性から生まれない。属性は人間の後から来る。
>
> 今回俺がやるのは「この世界にいそうな人間」を現場から引っ張ってくること。作家への使い方の指示はしない。それは専門外だ。ただ「こういう人がいる」という報告だけする。
>
> 10人出す。

## ◼ キャラ01｜コジマ アキオ（小島 昭文）

**「第10レースだけ買う男」**

### 表の設計

| 項目 | 内容 |
|------|------|
| **名前** | コジマ アキオ（小島 昭文） |
| **年齢・性別** | 49歳・男 |
| **競馬との関わり方** | 毎週土曜の第10レースだけ買う。それ以外は観ない |
| **一言で言うとどういう人か** | 「ここでは履歴書がいらない」と思っている男 |
| **口調・話し方** | 短文。断定しない。「〜かなあ」「まあね」。笑いで終わらせる癖がある |
| **記号** | くたびれたリュックサック、500円玉だけ入った小銭入れ |

### 裏の設計

**職業（過去・現在）**
2000年に地方の国立大（法学部）を卒業した。バブルはとっくに終わっていた。就職活動は45社受けて4社受かったが、内定先が1社潰れた。残りの中で唯一「潰れなそう」な会社に入った理由が「潰れなさそう」というだけだった。
2012年、その会社がリストラで彼を放出した。その後は派遣社員として3社を転々とした。現在は中小の部品メーカーで在庫管理の派遣社員。時給1,400円。月収22万円前後。

**生活の時間割（平日何してる？）**
6:40起き。バス25分・電車40分。倉庫の中で一日ほぼ一人作業。終わると同じルートで帰る。夜は米を炊いて、買い置きの惣菜を食べながらYouTubeを見る。11時就寝。金曜夜だけ、翌日の競馬新聞をコンビニで買う。これが一週間で唯一の「楽しみに向かう行動」だ。

**競馬を始めたきっかけと続けている理由**
27歳のとき、先輩に誘われて初めて競馬場に行った。最初の1レースで3,000円が16,800円になった。それより鮮烈だったのは「この場所では俺が誰かを知らない」という感覚だった。社歴も肩書も関係ない。全員が同じ窓口で同じ券を買っている。
続けている理由は当てたいからではない。「土曜日に行く場所がある」ということだ。

**一番記憶に残っているレース**
2009年の天皇賞・秋。会社がまだあった頃。同僚と4人で行って、全員で本命馬券を買った。本命が来た。みんなで笑って、ビールを飲んだ。その後リストラで3人と疎遠になった。今でもそのレースを思い出すと、当たったのか負けたのかわからない気持ちになる。

**人に言えない習慣や癖**
競馬場でだれかが大声で叫んでいるのを遠くから見るのが好き。自分は声を出さない。他人が「うわあ！」と叫ぶのを見て、少しだけ何かが軽くなる。代理呼吸みたいなものだ、と思っている。

**kazemachi に来る理由（居場所としての意味）**
マキのコミュニティを見て「場所がある人の顔をしている」と感じた。自分には場所がなかった。入るかどうかを3週間迷っていた。最終的に入ったのは「第10レースだけ買う人間でも排除されない場所かもしれない」と思ったからだ。今も中心にはいない。端っこに立って、時々うなずく。

**他キャラとの面白い組み合わせ**
ゲンさんとが一番おいしい。ゲンさんは60代でコジマを「若い」と見る。コジマは49歳でゲンさんを「終わった人」ではなく「まだいる人」として見る。この2人が競馬場で隣になったとき、ゲンさんが「2000年卒か」と言う一言で全部が変わる瞬間がある。

### 一番おいしいエピソード候補

ある土曜の第10レース、コジマが買った馬が最後の直線で3番手から差してきた。スタンドで彼は声を出さなかった。ゴール板を過ぎた後、隣にいたソラが「来ましたね」と小声で言った。コジマは一瞬だけ、ガッツポーズをしかけた手を止めて「うん」とだけ言った。その夜、帰り道のバスで窓の外を見ながら「来ましたね」という声を思い出していた。自分でも理由はわからない。

## ◼ キャラ02｜ハナ（大野 花音）

**「推し馬しかない子」**

### 表の設計

| 項目 | 内容 |
|------|------|
| **名前** | ハナ（大野 花音） |
| **年齢・性別** | 22歳・女 |
| **競馬との関わり方** | 推し馬を作るタイプ。馬券よりも馬が好き |
| **一言で言うとどういう人か** | 競馬を「推し活」として入ってきたが、本物になりかけている子 |
| **口調・話し方** | 語尾が上がる。「え、それってどういうこと！？」系。Xで使う言葉をそのまま口に出す |
| **記号** | スマートフォン（常に横画面）、推し馬のコラージュ壁紙 |

### 裏の設計

**職業（過去・現在）**
大学4年生（または既卒1年目）。文学部。就活は継続中。バイトはカフェ週3。月8万円くらい。実家暮らし。

**生活の時間割（平日何してる？）**
午前は就活関係のWebセミナーをパジャマで受ける。昼からバイト。夜は競馬関連アカウントのXを3時間スクロールする。寝る前に推し馬の動画を見る。これを週5でやっている。

**競馬を始めたきっかけと続けている理由**
1年前、Xでソラのアカウント（あるいはkazemachi類のアカウント）が流れてきた。「なんかかわいい絵と競馬の話してる人いる」と思ってフォローした。2ヶ月後、気になった馬（芦毛の牝馬）が次走で人気薄の勝利を収めた。その馬を事前に「なんかかわいい」と言っていた自分のツイートが当たっていた。これで火がついた。続けている理由は「推し馬が走り続けているから」だが、最近は馬券の配当計算も少しずつ覚えてきた。本人は気づいていないが移行期にある。

**一番記憶に残っているレース**
推し馬が初めて重賞を勝ったレース。最後の直線で「あ、来る」と思ってスマホを縦から横に持ち替えた。録画はしていたが撮影もしてしまったので映像がブレブレになった。この「ブレブレ動画」が自分にとって今も一番大切な競馬映像だ。

**人に言えない習慣や癖**
推し馬が負けたとき、その騎手のせいにして心の中で口汚く罵ることがある。次のレースではけろっと応援している。このサイクルに自分でうっすら気づいているが直せない。

**kazemachi に来る理由（居場所としての意味）**
ソラの予想スタイルを「自分に近い」と感じた（※実際は少し違うが、外から見るとそう見える）。競馬歴が浅いことで馬鹿にされない場所を探していた。ゲンさんに「そういう見方もある」と言われた一言が、今でも残っている。

**他キャラとの面白い組み合わせ**
ソラとの関係が面白い。ハナはソラを「先輩」と思っているが、ソラ自身は「先輩と思われているのが少し重い」と感じている。でもハナが「なんでこの馬選んだんですか？」と聞いた時のソラの答えが「うまく言えないんだけど……」で止まる瞬間、ハナが「あ、同じだ」と思う。この共鳴の瞬間。

### 一番おいしいエピソード候補

ある日、ハナが「○○（推し馬名）って今週出ないんですよ。だから今日は馬券買うか迷ってて」と言った。するとゲンさんが「推してる馬がいないと、どの馬も見えてくるぞ」と言った。ハナはその日、初めて「推してない馬」の馬券を買い、3着に来た。帰り道に「なんか……知らない人と話したみたいな気持ち」と言った。

## ◼ キャラ03｜マサ（遠藤 正一）

**「笑いながら終わっていく男」**

### 表の設計

| 項目 | 内容 |
|------|------|
| **名前** | マサ（遠藤 正一） |
| **年齢・性別** | 43歳・男 |
| **競馬との関わり方** | ガチのギャンブラー。週20〜30万円の馬券を買う週もある |
| **一言で言うとどういう人か** | 底まで行ったことがある男。でも笑っている |
| **口調・話し方** | 明るい。テンポが速い。「いやもう終わったわ今月」を笑いで言える |
| **記号** | 手持ちの競馬新聞（書き込みが異様に細かい）、ボールペン3本 |

### 裏の設計

**職業（過去・現在）**
中堅の運送会社でドライバー。深夜シフト。月収28万円前後。副業禁止だが競馬収支は「副業」と思っていない。実際の年間収支は毎年マイナス40〜80万円。口座には残高があるように見えるが、消費者金融から借りたことが2回ある。現在は借りていない（と言っている）。

**生活の時間割（平日何してる？）**
午後から深夜にかけてトラックを走らせる。帰宅は深夜1〜2時。風呂入って競馬サイトを1時間見る。午前中に眠る。昼過ぎに起きてコンビニのカレーを食べ、午後は予想をする。この生活が10年以上続いている。家族はいない。

**競馬を始めたきっかけと続けている理由**
20代前半に友人に連れて行かれた。「向いてる」と思ったのは最初の3年間だけだった。それ以降は「やめ方がわからない」という理由で続けている。だが本人はそれを認識していない（または認識したくない）。「競馬があるから月曜日に起きられる」と笑いながら言う。

**一番記憶に残っているレース**
10年前のある秋のGⅠ。その日は全財産の8割を一頭に突っ込んだ。最後の直線で先頭に立った馬がゴール前でとらえられた。着差はクビ。帰り道、競馬場の外の自動販売機の前で10分間立っていた。自分でも気づかないうちに涙が出ていた。その後コンビニに入って肉まんを買い、「まあな」と言った。誰もいなかった。

**人に言えない習慣や癖**
大きく負けた日の夜、競馬場の食堂で一人で食べる。必ずカツカレーを頼む。「今日はカツを入れておく」という呪いのようなものだ。次の週には忘れている。

**kazemachi に来る理由（居場所としての意味）**
フジさんが全員の成績を記録していることを知っている。自分の成績が記録されることを「怖い」と思っている。でも来ている。たぶん、記録されることで「自分の負けが本物になる」と思っているからだ。本物になったほうがいい、という感覚が、うっすらある。

**他キャラとの面白い組み合わせ**
フジさんとの組み合わせ。フジさんは淡々とマサの収支をつけている。マサは「フジさん、俺の数字どうですか」と冗談っぽく聞く。フジさんが一言だけ答えた時の間（ま）。フジさんは評価をしない。ただ事実だけを言う。マサが笑いで返した後の、一瞬の沈黙。

### 一番おいしいエピソード候補

ある日、マサが「今月はさすがにやばい」と言った。ナルミが「具体的にいくらですか」と聞いた（数字への興味から）。マサが数字を言った。ナルミが一瞬黙った。マサが「怒ってる？」と聞いた。ナルミが「いや……ただ確認しただけです」と言った。この後のフジさんの無言のタブレット操作。何かを入力した。何を入力したかは誰も聞かなかった。

## ◼ キャラ04｜ツルコ（鶴田 美奈子）

**「競馬が嫌いな女が競馬場にいる理由」**

### 表の設計

| 項目 | 内容 |
|------|------|
| **名前** | ツルコ（鶴田 美奈子） |
| **年齢・性別** | 36歳・女 |
| **競馬との関わり方** | 自分では馬券を買わない。競馬も好きではない |
| **一言で言うとどういう人か** | 競馬場に来る理由が、競馬ではない女 |
| **口調・話し方** | 敬語が基本だが、慣れてくると少し毒が出る。「まあ……そうですね（でも違うと思う）」系 |
| **記号** | 文庫本（競馬場で読んでいる）、飲みかけのペットボトルのお茶 |

### 裏の設計

**職業（過去・現在）**
医療事務。区内の個人クリニック勤務。週5、9時〜17時半。月収21万円。安定しているが昇給がない。同僚は全員女性で、ランチタイムの話題がずっと同じ（子供の話、夫の話、誰かの話）。この「ランチタイムの話題」から逃げたくて、休日は一人で過ごすことが多い。

**生活の時間割（平日何してる？）**
起きてクリニックへ行き、データ入力をして帰ってくる。帰宅後は料理動画を見ながら自炊する。食べながら小説を読む。11時就寝。変化がない。変化がないことに気づいているが、変化の作り方を忘れた。

**競馬を始めたきっかけと続けている理由**
「始めた」わけではない。3年前、父（ゲンさんに近い世代の常連）が具合を悪くした時に、「一緒に来てくれ」と言われた。父の付き添いで競馬場に来たのが最初。父は今は元気だが、ツルコは一人で来るようになった。理由は「静かなのに、騒がしい場所」が自分に合っていると気づいたからだ。競馬場の食堂で、知らない人が隣で声を上げるのを見ながら文庫本を読む——この組み合わせが、今の彼女の一番の休日だ。

**一番記憶に残っているレース**
記憶に残っているレースは「見ていないレース」だ。父に連れられて来た最初の日、第7レースの途中でトイレに行った。帰ってきたら父が「来た来た！」と言って興奮していた。ツルコは結果だけ聞いた。父の顔が子供みたいだったことは覚えている。

**人に言えない習慣や癖**
競馬場に来ると必ずスタンドの端の席に座る。人の流れが見える場所。レースより「レースを見ている人」を観察している。特定の人を覚えて、次週も来ているかどうか確認している。名前は知らない。「赤いダウンのおじさん」「いつも同じ位置の女の人」等で識別している。

**kazemachi に来る理由（居場所としての意味）**
最初は偶然（友人に紹介された）。でも続けているのは「馬券を買わなくても変な顔をされない場所」だったからだ。ゲンさんが「馬券より競馬を楽しめる人間は少ない」と言った一言が、何かを許可してくれた感じがした。

**他キャラとの面白い組み合わせ**
マキとの組み合わせ。マキは「場を作る人」だが、ツルコは「場に来るが中心に入らない人」だ。この二人の温度差——マキが何かを作ろうとするたびに、ツルコが一歩引いて本を読んでいる。でもツルコのほうが場の空気を正確に読んでいる、という逆転がある。マキが気づくかどうか。

### 一番おいしいエピソード候補

ある日、ソラが「どの馬が好きですか」とツルコに聞いた。ツルコが「買わないので」と答えた。ソラが「あ、買わなくていいんですけど。好きな馬ってないですか」と聞き直した。ツルコは少し間を置いて「あの……栗毛のやつ」と言った。「なんか走り方が綺麗だな、と思って」。ソラが双眼鏡を差し出した。ツルコが「え」と言いながら受け取って、初めてパドックを双眼鏡で見た。この後ツルコが何を思ったかは、誰も知らない。

## ◼ キャラ05｜タカノ リョウ（高野 亮）

**「内側にいた男」**

### 表の設計

| 項目 | 内容 |
|------|------|
| **名前** | タカノ リョウ（高野 亮） |
| **年齢・性別** | 38歳・男 |
| **競馬との関わり方** | 予想はする。でも馬券を買う手が少し遅い |
| **一言で言うとどういう人か** | 競馬を内側から見ていたせいで、外から見られなくなった男 |
| **口調・話し方** | 落ち着いている。業界用語が自然に出る。指摘されると「あ、ごめん」と直す。でもまた出る |
| **記号** | 擦り切れた厩舎のキャップ（今は被っていないが、カバンに入っている） |

### 裏の設計

**職業（過去・現在）**
20〜28歳まで、競馬関係の厩務員補助員として働いていた（※正確には「厩務員見習い」あるいは「育成牧場スタッフ」レベル、推測）。馬の世話をする日々。29歳のとき、所属していた厩舎が解散し、別の仕事を探すことになった。現在は物流会社の管理部門。馬とは関係ない仕事。月収26万円。

**生活の時間割（平日何してる？）**
9〜18時で事務仕事。帰宅後は競馬の動画（主にパドック映像）を見る。競馬関係の書籍も読む。元同僚の厩務員と月1〜2回飲む。その席でしか「昔の話」をしない。

**競馬を始めたきっかけと続けている理由**
厩舎にいた頃から馬券は買っていた（本来は業界関係者の馬券購入は制限がある場合があるが、現場の感覚は別だった——※推測）。でも「馬を知りすぎると予想が歪む」ということは体感的にわかっている。馬のコンディションを見る目はある。でも「コンディションがよくても負けることがある」ことも知っている。そのせいで予想の最後の一押しができない。

**一番記憶に残っているレース**
自分が世話をしていた馬が重賞を勝った日。スタンドで見た。ゴール後、自分はガッツポーズをしていた。馬券は持っていなかった。その後、同僚の厩務員が「なんで買ってないんだよ」と笑った。「忘れてた」と答えたが、嘘だった。怖かった、という言葉を飲み込んでいた。

**人に言えない習慣や癖**
競馬場に来ると、ゴールよりもパドックを長く見る。馬のいない時間でも、馬房があった方向をなんとなく見ている。本人は気づいていない。

**kazemachi に来る理由（居場所としての意味）**
「競馬の内側を知っている」ことが、外のコミュニティでは「知識がある人」として扱われる。それが少し居心地いい一方で、うまく言葉にできない引け目もある。業界を離れたことへの後悔なのか、馬に対する後ろめたさなのか、自分でもわかっていない。

**他キャラとの面白い組み合わせ**
テツとの組み合わせ。テツは「馬やレースに自分の感情を重ねる人」だが、タカノは「馬を生き物として知っている人」だ。テツが詩的に馬を語る時、タカノが「あの馬は右前脚に癖がある」という話をする。この2人の馬への接し方のズレが、どちらにとっても何かを動かす。

### 一番おいしいエピソード候補

ソラが「この馬、なんか今日違う気がする」とパドックで言った。ゲンさんが「気のせいだ」と言った。タカノが黙ってパドックを見ていた。レース後、ソラの言っていた馬が大差の勝利をした。ソラが「やっぱり！」と言った。タカノが「歩様が違ったんですよ。今日」と小声で言った。「なんで言ってくれなかったんですか」とソラが言った。タカノが少し間を置いて「言う言葉が見つからなかった」と答えた。
