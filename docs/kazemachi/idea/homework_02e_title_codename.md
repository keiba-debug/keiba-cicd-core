# 宿題②e — タイトル / コードネーム募集

> 各作家のシステムプロンプトの後ろに貼って投げる。
> PROJECT_DESIGN.md を添付する。

---

```
━━━━━━━━━━━━━━━━━━━━━
■ 設計書ができた
━━━━━━━━━━━━━━━━━━━━━

ここまでの全議論をPROJECT_DESIGN.mdにまとめた（添付）。
これがプロジェクトの聖典になる。

まず読んでほしい。
そのうえで、2つ頼みたい。

━━━━━━━━━━━━━━━━━━━━━
■ 1. 内部コードネーム
━━━━━━━━━━━━━━━━━━━━━

今「yosou-tv」というフォルダ名で作業しているが、
もう実態と合っていない。
「予想TV番組」ではなく「競馬×キャラクター×物語の群像劇」になった。

内部で使うコードネームが欲しい。
フォルダ名、ファイル名、会話の中で使う短い呼び名。

条件：
- 英語 or 日本語ローマ字
- 短い（10文字以内が理想）
- プロジェクトの本質を少しでも匂わせるもの
- かっこよくなくていい。わかりやすさ優先

━━━━━━━━━━━━━━━━━━━━━
■ 2. 作品タイトル候補
━━━━━━━━━━━━━━━━━━━━━

これは外向き。読者・視聴者が最初に目にする名前。

今出ている候補：
- 「競馬で生きてる」（カイ案）
- 「競馬の話をしていたら、この6人の秘密が少しずつ見えてきた」（レイ案・長い）

他にもあれば出してほしい。
日本語でも英語でもいい。複数案歓迎。

意識してほしいこと：
- この作品は「予想バトル」ではない。
  予想はするが、それは「予想エピソード」「予想ストーリー」に近い。
  その中に掛け合いや議論は出てくるが、戦いではなく物語。
- タイトルから「当たる予想が見られる」を期待されると困る
- タイトルから「この人たちの話が見たい」と思わせたい
- 長くてもいいが、略称で呼べるもの（例：「推しの子」→「おしのこ」）

━━━━━━━━━━━━━━━━━━━━━
■ 3. 設計書へのフィードバック
━━━━━━━━━━━━━━━━━━━━━

PROJECT_DESIGN.mdを読んで、
「これは違う」「これが足りない」「この表現の方がいい」
があれば遠慮なく言ってほしい。

特に：
- 自分の発言が正確に反映されているか
- 他の2人の意見とのバランスは取れているか
- 「予想バトル」→「予想エピソード/ストーリー」という言い換えについてどう思うか

自由に書いてくれ。
```
