> **使い方**: 既存プロンプト（`homework_03a_image_prompts.md`）の各キャラプロンプト末尾に、本ファイルの「顔面差別化スペック」を追記して生成する

---

## ■ 共通スタイル指定 V2（旧版を差し替え）

```
【必須スタイル指定 V2 — 顔面差別化対応】

＜絵柄統一＞
- 絵柄は添付画像をベースにすること（トーン・線の太さ・色彩の温度感・デフォルメ度合いを統一）
- リアル寄りだがイラスト的なタッチ（実写ではない）
- 色調は落ち着いた暖色ベース、競馬場の午後の光を意識
- 文字・テキスト・ロゴは入れない

＜構図＞
- 上半身〜バストアップ構図
- 背景は控えめ（キャラの空気感が伝わる最低限の環境描写）
- 記号アイテムは手元〜胸元に目立つ位置に配置

＜最重要：顔面差別化ルール＞
- キャラごとに「目の形」「視線」「瞳の大きさ」「眉の形」「輪郭」が異なること
- 同じ性別・近い年齢のキャラが並んでも一目で別人と分かること
- 以下の6要素を各キャラで個別指定する：
  1. 輪郭（丸顔 / 面長 / 卵型 / 四角い / シャープ）
  2. 目の形（丸い / 切れ長 / 垂れ目 / つり目 / 奥二重 / 一重 / 大きい二重）
  3. 視線の癖（正面 / 伏し目 / 横目 / 遠くを見ている / 上目遣い / 斜め下）
  4. 瞳の大きさ（大 / 中 / 小）
  5. 眉の形（太くまっすぐ / 細いアーチ / 困り眉 / キリッと上がった / ナチュラル太め / 薄い）
  6. 口元（口角上がり / 引き結び / 薄く開いている / への字 / 柔らかい微笑 / 無表情）
- ソラを基準（伏し目・困り眉・瞳やや大・柔らかい微笑）とし、他全員をソラと違う顔にすること

＜追加差別化＞
- 肌のトーン：全員同じにしない（日焼け・色白・標準で分ける）
- 体格の示唆：バストアップでも肩幅・首の太さで体格差を表現
- 年齢感：シワ・ほうれい線・肌のハリで年齢を正確に反映
- 髪の描き分け：質感（ツヤ / マット / パサつき）、毛量、生え際の癖
```

## ■ 使い方

### 生成手順

1. `homework_03a_image_prompts.md` から対象キャラのプロンプトを取得
2. 本ファイル（V2）の **共通スタイル指定 V2** を冒頭に差し替え
3. 対象キャラの **【顔面スペック】** ブロックをプロンプト末尾に追記
4. 既存キャラの参考画像をスタイル参照として添付
5. 生成
