# 宿題⑪ — ソラ「静かな魅力」プロンプト集

既存のS評価画像を見たうえで、今回は「驚き」ではなく「まだ言葉になっていない時間」を切り取ることに集中した。

共通して意識したのはこの3つ。

- ソラは「説明する人」ではなく「先に何かを感じてしまう人」
- 可愛さは副産物であって、主題ではない
- 1枚ごとに視線の先を少し変えて、単調さを避ける

---

### 1. もう答えに触れているのにまだ言葉にしていない

**感情メモ**: 予想はほぼ固まっている。でもソラ自身がまだそれを言葉として信じきれていない。

**プロンプト**:
```text
添付のソラS評価画像4枚をスタイルリファレンスとして使用。リアル寄りだがイラスト的タッチ、暖色ベース、競馬場の午後の柔らかい斜光、上半身〜バストアップ、背景は静かで余白多め、文字なし。
ソラ、26歳日本人女性。輪郭は卵型で柔らかいライン、目は大きい二重で少し垂れ気味、視線は伏し目で焦点が少し遠い、瞳は大、困り眉で細め、口元は閉じ気味で薄く柔らかい、色白で透明感、華奢、黒に近い柔らかいボブ。
競馬場のテーブル席、出馬表の端を左手の指先で押さえ、右手のペンが止まっている。書き込む寸前なのに止まっている。顔は少し下向き、でも目だけがもう答えに触れている。かわいくしすぎない、静かな確信と不安が同居した瞬間。
```

**用途タグ**: プロフィール,予想回,感情,Note

### 2. 人の会話の外側で、別の何かを感じている

**感情メモ**: その場の話題には参加しているようで、ソラだけが別の気配を拾っている。

**プロンプト**:
```text
添付のソラS評価画像4枚をスタイルリファレンスとして使用。リアル寄りイラスト、低彩度の暖色、午後の拡散光、上半身、背景は少しぼかした競馬場スタンド、文字なし。
ソラ、26歳女性。卵型の輪郭、大きい二重で少し垂れ気味、視線は会話の輪から少し外れた横方向、瞳は大、困り眉、唇は閉じたまま、華奢。
手前か背景にぼんやり人影があり、誰かが話している気配だけある。ソラは椅子に座って、双眼鏡を膝の横に置いたまま、首を少しだけ横に向けている。大きく反応しない。聞いているというより、会話の向こうにある別の気配を拾っている。静かで気になる絵。
```

**用途タグ**: 日常,感情,Note

### 3. 当たる前の不安と確信が同居している

**感情メモ**: 直感はある。でもその直感に自分が追いついていない。

**プロンプト**:
```text
添付のソラS評価画像4枚をスタイルリファレンスとして使用。リアル寄りイラスト、暖色、レース直前の午後光、上半身〜膝上、背景はぼけたパドック柵、文字なし。
ソラ、26歳女性。卵型、少し垂れた大きい二重、伏し目寄りだが時々前を見る、瞳は大、困り眉、口は薄く閉じている、色白で華奢。
両手で双眼鏡を下げたまま胸元に触れている。出馬表は膝の上。肩に少しだけ力が入っているが、表情は静か。怖いのにもう見えてしまっている、という空気。可愛い驚き顔ではなく、レース前の静かな緊張と確信の同居。
```

**用途タグ**: 予想回,感情,X投稿

### 4. 伏し目で出馬表を見ている

**プロンプト**:
```text

白いテーブル、出馬表と色ペンが数本。ソラは右手にペンを持っているが、書く手が止まっている。左手は出馬表の端を軽く触っているだけ。顔は下向き、でも「何かに引っかかった」気配がある。背景は無音に近い競馬場の休憩席。静かな集中。
```

**用途タグ**: プロフィール,予想回,Note

### 5. 人の話を聞いている横顔

**感情メモ**: 相手の言葉を聞きながら、その言葉の外側にあるものを拾ってしまっている。

**プロンプト**:
```text
添付のソラS評価画像4枚をスタイルリファレンスとして使用。リアル寄りイラスト、暖色、午後の横光、横顔が映える構図、背景は競馬場スタンドのぼけた座席、文字なし。
ソラ、26歳女性。卵型、少し垂れた大きい二重、視線は相手の少し手前に止まっている、瞳は大、困り眉、口元は閉じている。黒に近い柔らかいボブ、華奢。
ソラは椅子に座り、首を少し傾げて誰かの話を聞いている横顔。膝の上に出馬表。双眼鏡は横に置かれている。聞き役の顔ではなく、相手の言葉で自分の直感の輪郭を確かめている顔。静かで、返事の前の数秒を切り取る。
```

**用途タグ**: 関係性,日常,感情,Note

### 6. レース後の沈黙

**感情メモ**: 当たり外れより、自分がなぜあの馬を選んだのかを反芻している。

**プロンプト**:
```text

ソラ、26歳女性。卵型、大きい二重、視線はやや下、焦点が近くにない、瞳は大、困り眉、口元は薄く閉じている、色白、華奢。
膝の上に閉じかけた出馬表。双眼鏡は脇に置いてある。両手は小さく重ねている。喜んでも落ち込んでもいない表情。レース結果より、自分の判断の手触りを確かめている沈黙。余白の多い静かな絵。
```

**用途タグ**: 結果回,感情,Note

### 10. 誰かの言葉で揺れた瞬間

**感情メモ**: 直感を疑いたくないのに、相手の言葉が自分の中に入ってきてしまった。

**プロンプト**:
```text
添付のソラS評価画像4枚をスタイルリファレンスとして使用。リアル寄りイラスト、暖色、テーブル席の柔らかい室内外混合光、上半身、背景には人影をぼかして入れる、文字なし。
ソラ、26歳女性。卵型、少し垂れた大きい二重、視線は斜め下で少しだけ揺れている、瞳は大、困り眉、唇は閉じ気味、華奢。
テーブルの上に出馬表と赤ペン。ソラはペンを持ったまま止まっている。首がほんの少しだけ引いていて、誰かに何か言われた直後の空気。反論も納得もしていない。ただ自分の直感を一瞬だけ疑いかけている。かわいくせず、静かな動揺を描く。
```

**用途タグ**: 関係性,感情,予想回,Note

## 補足メモ

- 今回の10案は、既存の「驚き」「かわいい反応」カットと競合しないようにしている
- `1 / 4 / 8` はプロフィールやNote向き
- `3 / 9 / 10` は予想回の核
- `6 / 7` はレース後や日常投稿で効く
- `2 / 5` は他キャラとの関係性を見せるときに使いやすい

次にやるなら、この10案の中から **まず3案だけ生成して方向性検証** がいい。
おすすめは `1`, `6`, `9`。ソラの「静かな魅力」の軸が最も見えやすい。
