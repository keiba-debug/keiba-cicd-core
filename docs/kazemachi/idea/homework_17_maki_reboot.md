## 設計意図メモ

### マキの核心 = 「迎える人」

- B01が最重要。「振り返って手を振る」= マキの本質そのもの
- B02で「教える側」の顔。初心者ガイド的な役割の体現
- B04で「オフのマキ」。髪をおろすことで別の面を見せる
- B05で「実は自分もちゃんと競馬が好き」という面を出す

### 服装の方向性

| アイテム | 色味 | 備考 |
|---------|------|------|
| アウター | ベージュ〜キャメル（リネン or コットン） | ソラの青灰と対極 |
| インナー | 白T or ボーダー | 清潔感・親しみ |
| パンツ | デニム or 黒スキニー | カジュアル |
| 靴 | 白スニーカー | 動きやすさ重視 |
| バッグ | 大きめトート | 荷物が多い人（みんなの分の馬券・飲み物入ってる） |
| 小道具 | 紙コップコーヒー | ソラの双眼鏡、リツのPCに対して「日常感」 |

### 33歳の表現

- 「若作り」ではなく「年齢なりの落ち着き」
- 化粧が薄い = おしゃれに興味がないのではなく、「素のままで場に馴染める人」
- 少しだけ頬に丸みがある = 親しみやすさの記号
- 後れ毛 = きちんとしてるけど完璧ではない（ソラの「ぼさっと」とは違う方向の隙）

## Phase 2 設計意図メモ

### シチュ設計の構造（カイ視点）

- M01〜M03が「迎える人」の三段階（歩いてくる→入ってくる→聞いてる）
- M04で「実は自分も」のギャップ
- M05で「オフ」（唯一の髪おろし。カイの警告通り1枚だけ）
- M06で「帰り道の背中」（レイの伏線装置）

### 表情の設計（ユキ視点）

- M01-M03: 半笑い〜柔らかい目（公の顔）
- M04: 真剣（私の顔。「ここにいていいよ」と言ってもらえなくても来る理由）
- M05: 疲れてるが穏やか（誰も迎えなくていい時間）
- M06: 読めない（レイの「何を知っているか不明」）

### 伏線の仕込み（レイ視点）

- M06の「空の紙コップ」= 1日の終わり。コーヒー＝マキのエネルギー。空＝消耗
- M04の「柵の上にコーヒーを置く」= 手が空いている＝心が空いていない（レイの仕掛け案A）
- M05は髪をおろすことで「別人感」。考察勢向け：「この状態のマキは何かを考えている」
