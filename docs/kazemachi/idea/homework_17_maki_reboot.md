| 川本あかり | マキ v3 |

| 料理で人を迎える | コーヒーと場で人を迎える |

### 2モード設計

| | 日常マキ（眼鏡ON） | 本気マキ（眼鏡OFF） |
|--|-------------------|-------------------|
| 眼鏡 | **丸眼鏡かけてる** | **外してる（頭の上 or バッグの中）** |
| 髪 | ゆるいお団子 or ひとつ結び | **おろしてる。少し巻いてる** |
| メイク | すっぴん〜薄化粧 | **赤リップ。眉もちゃんと** |
| 服 | だぼカーデ＋ボーダー＋デニム | **少しタイトめ。色はテラコッタ〜ワインレッド** |
| 表情 | 半笑い・ゆるい | **凛。口角は上がってるけど目が笑ってない** |
| 出現 | 普段の土曜。通常開催 | **G1の日。ナイター。「勝負の日」** |

### 眼鏡の設計

- **丸フレーム**（あかりさん的な柔らかさ。リツの四角眼鏡と完全差別化）
- 度入り（PC作業用。平日の在宅ワークで使う）
- **物語装置**: 外した日＝何かがある日。「マキさん今日眼鏡ないね」が伏線台詞
- 考察勢向け：眼鏡ON/OFFの回を並べると、裏の話と相関する（レイの仕掛け）

## 設計意図メモ v3

### キャラとしての引力

v1-v2の問題は「正しいけど見たくならない」。v3で加えたもの：

1. **眼鏡ON/OFFのギャップ** = 画が語る。Before/After 1セットでキャラ紹介が成立
2. **あかりさん構造** = 家庭的な安心感と、はっとする瞬間の共存
3. **赤リップ** = 他キャラにない色。5人並んだ時にマキだけ「赤」を持ってる
4. **M03（キッチン・朝）** = 競馬場以外のマキ。あかりさんのエプロン姿的な引力

### 「両方本物」の設計（ユキ視点）

- 眼鏡ONが「仮面」ではない。あれが素のマキ
- 眼鏡OFFも「演じてる」わけじゃない。G1の日は気合が入る。それだけ
- **読者が「どっちが本当のマキ？」と思う → 両方だと気づく → 好きになる**

### 伏線の仕込み（レイ視点）

- 眼鏡ON/OFFの回を追うと、裏の話題との相関が見える
- M08の「眼鏡を手に持ってる」= ON/OFFの境界。最も意味深な1枚
- M07のバーは「マキが一人でいる」唯一のシーン。普段は常に誰かと居る人の孤独

### 他キャラとの記号すみ分け（最終版）

| キャラ | 眼鏡 | 色 | 記号 |
|--------|------|-----|------|
| ソラ | なし | 寒色 | 双眼鏡、色ペン出馬表 |
| ゲン | なし | ベージュ/白 | 赤鉛筆、競馬新聞 |
| アキヒコ | おでこにサングラス | 紺カーデ | 缶コーヒー、丸めた新聞 |
| リツ | 四角眼鏡（常時） | 黒 | ノートPC、黒ペン |
| **マキ** | **丸眼鏡（ON/OFF）** | **ベージュ〜赤リップ** | **トートバッグ、紙コップコーヒー** |
