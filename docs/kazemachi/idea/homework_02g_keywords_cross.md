# 宿題②g — キーワード・クロスポリネーション

> 各作家のシステムプロンプトの後ろに貼って投げる。
> 3人の②f回答を添付する。

---

```
━━━━━━━━━━━━━━━━━━━━━
■ 3人合わせて160語以上。すごい量が出た
━━━━━━━━━━━━━━━━━━━━━

全員の回答を添付する。まず読んでほしい。

━━━━━━━━━━━━━━━━━━━━━
■ 3人が共通して挙げたキーワード
━━━━━━━━━━━━━━━━━━━━━

面白いことに、3人が独立して同じ言葉を選んでいる。

【全員一致（3人とも挙げた）】
- 隣
- 余韻
- 予感
- 沈黙
- 残像
- 重なる
- 距離
- 合図
- 待つ

【2人が挙げた】
- 帰り道 / 日曜の帰り道（ユキ・カイ・レイ全員が時間系で言及）
- 来週 / また来週（ユキ・レイ）
- 気配（ユキ・レイ）
- 手触り（ユキ・レイ）
- 余白（カイ・レイ）
- 場（カイ・レイ）
- 根拠（ユキ・カイ）

━━━━━━━━━━━━━━━━━━━━━
■ 各人のトップ5から見える個性
━━━━━━━━━━━━━━━━━━━━━

【ユキ】隣、余韻、来週、まだ、帰り道
→ 感情と時間。「まだ」「来週」は継続の意志。ユキらしい。

【カイ】隣、余韻、積み重ね、根拠、場
→ 構造と核心。「場」「積み重ね」は設計者の目。

【レイ】常連、余韻、また来週、A面/B面、同じ場所にいた
→ 仕掛けと記憶。「常連」「A面/B面」はレイにしか出ない言葉。

━━━━━━━━━━━━━━━━━━━━━
■ プロデューサーが特に引っかかった言葉
━━━━━━━━━━━━━━━━━━━━━

全員の回答を読んで、こちらが特に気になったものを挙げる。

- 話そう、週末、話をしてたら
- 生きてる
- いつかの〇〇の話
- 隣で
- だれか
- 常連（レイ）— この1語で全部語れる。すごい
- 来週もここで（ユキ）— 続く気持ちが美しい
- まだ（ユキ）— 副詞1つでこの作品を表せる
- 帰り道（全員）— レース後がこの作品の本体
- 踊り場（レイ）— 階段の途中。止まってもいい場所
- コーダ（レイ）— 音楽の余韻部分。洒落てる
- 風待ち（ユキ）— 場所の名前みたいで美しい
- こぼれ話（ユキ）— 本筋じゃないのに本質が出る

━━━━━━━━━━━━━━━━━━━━━
■ 宿題：他の2人のキーワードを読んで反応する
━━━━━━━━━━━━━━━━━━━━━

1. 他の2人が出した言葉で「これは自分にはなかった、いい」と思ったもの3つ
2. 全員のキーワードを眺めて、2-3語を組み合わせたタイトル/コードネーム案を5つ以上出す
   例：「隣の余韻」「来週の常連」「風待ちの話」「コーダ」など
   短いもの、長いもの、日本語、英語、混合、なんでもいい
3. この中から内部コードネーム（フォルダ名）に使えそうなものを1つ推薦
4. 最後に、今の時点でタイトルの本命を1つだけ選ぶとしたら何か

自由に書いてくれ。
```
