## 1. キャラクターシート — リツ（28歳♂）

### 基本情報

| 項目 | 内容 |
|------|------|
| **名前** | リツ（律） |
| **年齢** | 28歳 |
| **世代** | Z世代〜ミレニアル境界（1998年前後生まれ） |
| **職業** | IT系（データアナリスト/エンジニア寄り）。リモートワーク中心 |
| **住居** | 一人暮らし。自室が世界の中心 |

### 性格

- **堅実でしっかりしてる**。若い世代だけど浮ついてない
- **予算を決めて、回収率を管理して、感情で買わない**。ギャンブル依存感はゼロ
- **人と居る必要を感じない**。嫌いじゃない。ただ必要性がわからない
- **でも素直**。ゲンさんの言葉を馬鹿にしない。ちゃんとメモする
- **「完成している」ことが問題**。エラーが出ない代わりに、新しい入力を受け付けない

### 競馬との関係

- **家でPC開いてデータ分析するのが至福**。netkeiba、JRA-VAN、自作スプレッドシート
- **競馬場には基本行かない**。「現場に行く意味がわからない。データは家でも見れる」
- **買い方は論理的**。期待値プラスの馬券だけ買う。当たっても外れても感情が動かない
- **感覚で買ったことがない**。「好きな馬」という概念がピンとこない

### 桐山レイとの差別化（3月のライオン）

| | 桐山レイ | リツ |
|--|---------|------|
| 孤独の質 | 壊れている。人と居る方法を知らない | **整っている。人と居る必要を感じない** |
| 盤面 | 将棋＝対人。相手がいる | 競馬データ＝対数字。相手はいない |
| 変化 | 人に触れて壊れた場所が埋まる | **人に触れて「型エラー」が増える** |
| 物語 | 欠損→修復 | **完成→揺らぎ** |

### 変化の物語（シリーズ通し）

**初期（家最高期）**:
- 自室でPC。データで完結。競馬場に行く理由がない
- SNSかネット経由でこの場の存在を知る（マキに誘われた？ソラの投稿を見た？）
- 「まあデータの答え合わせに一回くらい」で顔を出す

**中期（型エラー期）**:
- カズサに懐かれる。子供の相手なんてしたことない
- ソラの直感が当たるのを目の当たりにする。データにない根拠で当てる人間がいる
- **ゲンさんの「この馬、今日は走る」が、データ上の期待値と合わない。でも当たる**
- 「なんで？」が頭から離れなくなる

**後期（融合期）**:
- ゲンさんの「パドックで見る」「馬の気配」を、自分のデータに取り込めないか考え始める
- **デジタル×アナログの融合**。数値化できないものを数値化する試み
- 「数字は嘘をつかない」から**「数字だけでは足りない」**へ
- いつの間にか、家ではなく競馬場で分析するようになっている

### 他キャラとの差別化・関係性

| vs | リツ | 相手 | 関係性 |
|----|------|------|--------|
| ソラ | データで安心したい | 直感で感じたい | **最も遠い。でも一番気になる** |
| ゲン | デジタル | アナログ | **師弟ではないが、吸収したい** |
| マキ | 場に来る理由がない | 場を作る人 | **引き込んだ張本人** |
| アキヒコ | データを見る | データなんか見ない | **同じ男同士、意外と楽** |

### 物語における機能

1. **読者の窓**: 競馬場に来ない＝読者の初期状態。リツが場に馴染む過程＝読者が作品に馴染む過程
2. **成長キャラ**: 5人の中で唯一、物語を通じて明確に変わる人間
3. **KeibaCICDの体現者**: データとアナログの融合＝プロジェクトのテーマそのもの
4. **語り部回**: 「なぜ自分はここに来るようになったのか」を分析しようとして、分析できないことに気づく回

### 記号（小道具）

- **ノートPC**（これが一番の相棒。競馬場にも持ち込む）
- **眼鏡**（データを見る道具。おでこに上げると「見る目」が変わる）
- **EV+マグカップ**（自室用。競馬場では紙コップのコーヒー）
- **スマホ**（netkeibaアプリ、自作のスプレッドシート）
- **くしゃくしゃの馬券**（ゴミではなく検証サンプル。当たりも外れも同じ扱い）
- **ペンは持つがメモ用**。ソラのカラーペンやゲンの赤丸とは違い、黒ペン1本で数字だけ書く
