# 宿題③a — キャラクター工場（大量案出し+裏設定）

> **正式版プロンプトは `prompt_04_character_factory.md` に移行済み。**
> 各作家のシステムプロンプトは `writers/` フォルダを参照。
> PROJECT_DESIGN.md（聖典）を添付する。

---

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ タイトル寝かせ期間に入ったので、次のフェーズに進む
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

設計書（聖典）は固まった。コードネームも決まった。
次にやるのは「この世界に誰がいるか」をもっと自由に広げること。

今の設計書にはコア6人（マキ、ソラ、ナルミ、ゲンさん、テツ、フジさん）がいる。
ソラはかなり固まっている。でも他の5人はまだ輪郭が甘い。
そして6人という枠自体も、まだ確定ではない。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 今回の宿題：キャラクターを出しまくれ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

やってほしいこと：

**「kazemachi の世界にいそうな人間」を、最低8人出してくれ。**
上限はない。多いほどいい。

ルール：
- 既存6人の改造版でもいい（年齢変更、性別変更、性格変更、統合、分割）
- 完全に新しいキャラでもいい
- 「競馬場にいる人間」に限定しなくていい（競馬に関わる人間なら）
- ソラは固定。ソラだけは変えない。他は全部触っていい
- 1人のキャラに対して「投稿には出ない裏設定」まで書くこと（後述）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 各キャラに書いてほしいこと
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### 表の設計（投稿に出る情報）
- 名前（仮でいい）
- 年齢・性別
- 競馬との関わり方（何派？何を見てる？）
- 一言で言うとどういう人か
- 口調・話し方の特徴
- 記号（その人を象徴するモノ）

### 裏の設計（投稿には直接出ないが、書く人間の解像度を上げる情報）
- 職業（過去・現在）
- 生活の時間割（平日何してる？）
- 競馬を始めたきっかけと、続けている理由
- 一番記憶に残っているレース（とその理由）
- 人に言えない習慣や癖
- この人が「kazemachi」に来る理由（居場所としての意味）
- 他のキャラとの関係で、一番面白くなりそうな組み合わせ

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ ヒント（使っても使わなくてもいい）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### ヒント1：氷河期世代の男

- 現在49歳。2000年に大学卒業
- 就職氷河期をリアルに生きた世代
- 何かを諦めた経験がある人間が、競馬場に何を求めているか
- 今の設計だとテツ（ロマン派、年齢未設定）に近いかもしれないが、
  全く別のキャラとして出してもいい

### ヒント2：解像度の高いキャラとは

- 「データ派」「直感派」のような類型ではなく、
  その人の人生から自然に出てくる予想スタイルを持っている人
- 例：氷河期世代の男が「ロマン派」になるとしたら、
  それは性格がロマンチストだからではなく、
  あの時代に何かを信じ損ねた反動かもしれない

### ヒント3：kazemachi に足りないかもしれないもの

- 若い世代（20代前半、競馬歴浅い）
- 女性の多様性（マキ・ソラ・フジさん以外）
- 競馬場の外側にいる人（家族、恋人、同僚、SNSの人）
- 「競馬をやめた人」「競馬を嫌いな人」
- ダークサイド（依存、借金、嘘）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ この宿題の意図
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

今回出してもらったキャラは、全部が採用されるわけではない。

この後「オーディション」をやる。
- メインキャスト（5〜7人）を選ぶ
- 落ちたキャラはゲスト候補やサブキャラのストックになる
- 1人も無駄にはしない

だから遠慮なく出してほしい。
「こんなの使えないかも」と思うキャラほど面白いことがある。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 聖典との関係
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROJECT_DESIGN.md の以下は変えない：
- 勝ち条件
- 作品の軸（根拠の違いが人格の違い）
- 二層構造（表95%/裏5%）
- ユキ原則
- ソラの設計
- 絶対に守ること / 絶対にやらないこと

変えていいもの：
- コア6人の顔ぶれ（ソラ以外）
- キャラの年齢、性別、性格
- 三層構造（マキ=入口、ソラ=中心、フジ=謎）の配役
- ゲストキャラリスト

聖典のルールは守った上で、人を自由に入れ替えろ。

━━━━━━━━━━━━━━━━━━━━━
■ 出力ルール
━━━━━━━━━━━━━━━━━━━━━
- アウトプットは必ずMarkdown形式で出力すること
- 冒頭に自分の名前タグを入れること（例：【カイ】【ユキ】【レイ】【シュウ】）
- 全ての回答をテキストとして出力すること（「考えてみます」「後で出します」は禁止）
- 途中で切らず、最後まで書ききること
- 不明点があっても推測で書ききり、推測した箇所に「※推測」と注記すること
```
