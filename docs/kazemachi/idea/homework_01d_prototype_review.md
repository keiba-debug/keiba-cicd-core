# 宿題①d — プロトタイプ画像レビュー依頼

> 各作家のシステムプロンプトの後ろに貼って投げる。
> 各キャラの画像を添付する（MC 2枚、ナルミ 2枚、ソラ 2枚、ゲンさん 2枚、テツ 2枚、記録係 4枚）

---

```

■ プロトタイプ画像ができた

3人のプロンプト案を統合して、Seedream 4.0で全キャラのプロトタイプ画像を生成した。
添付画像を全部見てほしい。

ツール: Leonardo AI (Seedream 4.0)
テイスト指示: 全キャラ共通で「Skip and Loafer風、漫画寄りスケッチ線、低彩度」

━━━━━━━━━━━━━━━━━━━━━
■ 各キャラの現状と所感
━━━━━━━━━━━━━━━━━━━━━

【MC（進行役）】2枚
- テイストは半笑い1に近い再現度。ベースとしてはOK
- 書類まみれ感、半目の表情、水色シャツ+名札 → 方向性は合ってる

【ナルミ（データ派）】2枚
- 紺カーディガン、EV+マグカップ、丸めた馬券、スプレッドシート → 要素は入った
- 腕組みもたれポーズ版もある
- 課題：最初の版が若すぎた（20代前半に見えた）。調整して30歳感は出たが、まだ詰める余地あり
- PC画面が裏側に出てしまう問題あり

【ソラ（直感派）】2枚
- スマホの馬写真、レーシングジャケット、ヘアクリップ → キャラは立ってる
- 課題：元気すぎる。テイストが少年漫画寄りで、MC/ナルミとトーンが合ってない
- 目がデカすぎ、彩度が高い

【ゲンさん（経験派）】2枚
- チェックシャツ、ペン蓋くわえ、手書きノート、スマホ裏返し → 要素OK
- 課題：やさしすぎる。「近所の好々爺」になってて、クセ者感がない
- テイストが一番AI感が強い。線がツルッとしてて他キャラと合わない
- ノートが英語になっちゃった

【テツ（ロマン派）】2枚
- ロングコート、白スカーフ、手ポケット、曇り空の柵、古い馬券 → 完璧に近い
- 2枚目が特に良い。年齢感、達観した表情、モノクロに近いトーン
- 他キャラと明らかに空気が違う → G1回のインパクト用としてそのまま使えそう
- 黄色がかり問題もここだけ解消されてる

【記録係（新キャラ）】4枚
- タブレット+スタイラス、鉛筆、ブカブカカーディガン、メモ帳 → 要素OK
- 3枚目・4枚目の「端にいる観測者」感が良い
- テイストはMCに近くて統一感がある

■ プロデューサーからの問題提起
━━━━━━━━━━━━━━━━━━━━━

【1. かわいい女の子に偏りすぎ問題】
MC、ソラ、記録係の3人が全員「若いかわいい女の子」になっている。
ロスター全体の幅を考えると、誰かの年齢層やタイプを変えるべきではないか？

案：
- ソラを「お姉さま系（30代）」に引き上げる
- 記録係を「ベテラン事務のおばちゃん」にする（データに詳しいおばちゃんが端で全部見てる怖さ）
- MCはこのままで、他の女性キャラで幅を出す
- あるいは全く別の発想？

【2. テイスト統一問題】
キャラごとに生成タイミングが違ったため、テイストがバラバラ。
- MCと記録係は近いトーン
- テツだけモノクロ寄り（意図的にこれでいい？）
- ソラが少年漫画寄り
- ゲンさんがAI感強い
→ 最終的には全員統一する必要があるが、まず方向性を固めてから

【3. ゲンさんの再生成】
一番テイストが合ってない。もう少し「穏やかだけど目だけ鋭い」「クセ者ベテラン」感を出したい。
プロンプト調整案があれば欲しい。

■ 聞きたいこと
━━━━━━━━━━━━━━━━━━━━━

全キャラの画像を見た上で、以下について自由に意見をくれ。

【全員共通】
1. 各キャラの「これはいい / これは違う」を率直に
2. 並べた時の全体バランス。誰と誰が近すぎる？足りないタイプは？
3. かわいい女の子偏り問題への提案

【あなたの専門視点で】
- カイ：並べた時の対立構造、視覚テンポ、X投稿で映えるか
- ユキ：表情から感情が読めるか、キャラの内面が伝わるか、関係性が想像できるか
- レイ：仕掛けを埋める余地があるか、画像横断の伏線が機能しそうか

出力形式は自由。率直に書いてくれ。
```
