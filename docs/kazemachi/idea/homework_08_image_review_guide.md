## レビュー方針

### 「残す」基準
- kazemachiの画風（日常アニメ系、低彩度、スケッチタッチ）に合っている
- キャラの記号アイテムが正しく描かれている
- 表情にキャラの人格が出ている
- ソラの既存画像と並べて違和感がない

### 「落選」基準
- 画風が明らかに違う（リアルすぎる、ポップすぎる等）
- 別キャラに見える（年齢感・雰囲気が設定と合わない）
- 記号アイテムが間違っている or 欠けている
- 目が他キャラと同じに見える（差別化不足）

### 落選の移動先
```
C:\KEIBA-CICD\キャラクター\オーディション\_落選\{元フォルダ名}\
```

---

## 進め方

1. **上位キャラから順に**（ゲン → フジ → カズヒロ → マキ）
2. 各フォルダの画像をカカシが1枚ずつ表示
3. ふくだが「残す / 落選 / 保留」を判断
4. 落選画像を `_落選/` に移動
5. 残った画像がブラッシュアップのベースになる
6. 宿題⑦（ユキのプロンプト修正版）で再生成
