# kazemachi キャラクター・オーディション素材集（第2弾）

## 前置き：第2弾の設計意図

前回は12人を出して「15年前のあの日」への接続レイヤーを5層で設計した。
今回はその構造を「横に広げる」のではなく、**「裏から照射する」**キャラを追加する。

前回の12人は全員、「kazemachi の内側」から15年前に手を伸ばす構造だった。
今回は逆をやる。**「15年前の側から、kazemachi に何かが流れ込んでくる」**構造のキャラを入れる。

加えて、前回カバーしきれなかった予想スタイルの空白地帯――**馬場読み、音声記憶、写真、相馬眼、馬名学、裁決視点、海外競馬基準**――を埋める。

既存の全作家（ユキ、カイ、シュウ、シノ、キバ）の出力と重複しないことは確認済み。

## ■ 14. シズカ

### 表の設計
- **名前**：シズカ
- **年齢・性別**：♀ / 45歳
- **競馬との関わり方**：馬場読み専門。天候、含水率、風向き、開催週数――「馬ではなく地面を買う」。前半レースの決着タイムから後半の馬場変化を推測し、メインレースだけ勝負する
- **一言で言うと**：空を見て地面を読む、静かな予報士
- **口調**：淡々としている。数字を挟むが冷たくはない。「今日の中山、3R時点で0.4秒速い。風が南西に変わったから、最終は内が死ぬ」。予想というより天気予報
- **記号**：**携帯型の風速計**（キーホルダーサイズ。競馬場で風を測っている。周りからは変人に見える）

### 裏の設計
- **職業**：元気象予報士。テレビ局の天気コーナーを2年担当した。降板後、損害保険会社の自然災害査定に転身。現在も同職。年収540万
- **生活の時間割**：平日8:30-17:30勤務。帰宅後は翌週の競馬場の天気図と過去の馬場含水率データを照合する。Excel上にオリジナルの馬場予測モデルがある。精度は「ナルミのモデルとは別の軸で戦える」レベル
- **始めたきっかけ**：29歳、テレビを降板した直後。行く場所がなくて府中へ。雨の日だった。不良馬場で人気薄が勝つのを見て、「天候が結果を変える」という当たり前のことに、気象予報士として初めて興奮した
- **続けている理由**：自分の専門知識が「正しく使える場所」がここしかない。保険の仕事では過去を査定する。競馬では未来を予測する。気象予報士としての自分は、ここでだけ生きている
- **一番記憶に残っているレース**：2017年皐月賞。不良馬場。事前にモデルが「今日は重馬場巧者が有利」と出していた。その通りになった。問題は、自分が馬券を「買わなかった」こと。モデルの精度を信じきれなかった。以来、「信じる」ことが課題になっている
- **人に言えない習慣**：毎朝、天気予報を見る前に空を見る。テレビを降板してから、天気図より先に空を見る癖がついた。「データより先に空気を読む」こと——ソラの直感と構造的に同じことを、シズカは科学の言葉で行っている。本人はそのことに気づいていない
- **kazemachiに来る理由**：馬場の話ができる人間がいない。血統の話は多い、展開の話も多い。でも「今日の馬場は過去10年で3番目に重い」という話に興味を持つ人間が少なすぎる。ナルミだけがデータとして受け取ってくれる。でもナルミは環境データを「変数の一つ」としか見ない。シズカにとっては「それが全部」
- **面白くなる組み合わせ**：ソラとの関係が構造的に最も深い。シズカは「空気を読む」、ソラは「空気を感じる」。同じものを違う回路で処理している。シズカの科学とソラの直感が同じ結論に達した日、考察班は「この二人は同じものを見ている」と騒ぐ。マキとの対比も面白い。マキは馬体、シズカは馬場。生き物を見る目と環境を見る目

**伏線フック**：シズカの馬場予測モデルのデータベースには、自分が競馬を始める前の過去データも入力してある（JRA公式の含水率データから逆算）。→ ある日、15年前のあの日の馬場データだけが「欠損」していることに気づく。JRAのサイトにはある。でもシズカが過去に入力したはずのExcelにはない。消した記憶はない。設置：中盤。回収予定：終盤。データの欠損は「誰かが消した」のではなく、「シズカ自身があの日のデータを入力した時に、無意識に飛ばした」可能性を残す。シズカがなぜあの日を飛ばしたのか——考察班が掘れる余地。

## ■ 15. トワ

### 表の設計
- **名前**：トワ
- **年齢・性別**：♂ / 34歳
- **競馬との関わり方**：実況派。レースを「映像」ではなく「音」で記憶する。過去の名レースの実況を全文暗唱できる。予想は「この馬が勝った時の実況が、一番美しい文になる方」で選ぶ
- **一言で言うと**：声に取り憑かれた男。馬より先に実況が聞こえる
- **口調**：穏やかだが、テンポが実況寄りに跳ねることがある。「あの馬、最後の直線で——いや、やめます。先に言うと当たらなくなる気がして」。予想を言語化する時だけ慎重になる
- **記号**：**片耳のイヤホン**（ミナトと同じ片耳だが、トワは本当に何かを聴いている——過去の実況音声）

### 裏の設計
- **職業**：ラジオ局のAD。3年目。本当はアナウンサーになりたかったが、試験に3回落ちた。声は悪くないが「個性がない」と言われた。競馬実況という「個性の塊」に憧れたのはその頃
- **生活の時間割**：平日は不規則（収録日は早朝から深夜まで）。空き時間にイヤホンで過去の実況を聴く。自宅にはラジオ実況の音源コレクション（MD、CD、mp3）。JRA公式以外の音源も非公式に集めている
- **始めたきっかけ**：アナウンサー試験に落ちた直後、YouTubeで1998年毎日王冠の実況を聞いた。三宅正治アナウンサーの声が、レースの展開をそのまま音楽に変えていた。「こういう言葉の使い方がしたかった」と思った。馬券を買い始めたのはその翌月
- **続けている理由**：自分が馬券を買って見るレースには、まだ実況がついていない。「このレースの実況はどうなるか」を想像しながら見る。それは「まだ書かれていない物語の読者になる」感覚に近い
- **一番記憶に残っているレース**：2013年有馬記念。オルフェーヴル引退レース。ラジオNIKKEIの実況で「有終の美」と言った瞬間、涙が出た。映像は見ていない。音だけで泣いた。以来、ゴールの瞬間は必ず目を閉じる
- **人に言えない習慣**：自分でレースの「架空の実況」を録音している。自室のマイクで、まだ走っていないレースの実況を事前に録る。予想が当たった時、自分の実況とリアルの実況を重ねて聴く。そのズレが面白い。でもこれは絶対に人に聞かせられない
- **kazemachiに来る理由**：「このレースはどういう物語になるか」を語れる場所が欲しい。予想の根拠を聞かれたら「実況が美しくなる方」としか言えない。それを馬鹿にしない場所を探していた
- **面白くなる組み合わせ**：ミナトとの関係が面白い。ミナトは映像、トワは音声。同じレースを見ているのに記憶している層が違う。2人が同じレースについて語る時、映像と音声が合成されて「完全な記録」になる。ヨルとの共鳴もある。ヨルは騎手の「騎乗」を追い、トワは騎手の名が呼ばれる「瞬間」を追う。推し方の角度が違う

**伏線フック**：トワの非公式音源コレクションの中に、15年前のあの日のレースのラジオ実況がある。→ ある時、その音声を繰り返し聴いていたら、ゴール後の歓声の中に「本来ないはずの数秒の無音」があることに気づく。放送事故ではない。意図的なミュートの痕跡。設置：中盤。回収予定：終盤。何が消されたのか——実況アナウンサーが「何かを叫んだ」のか「何かを言いかけて止められた」のか。考察班は音声分析を始める。トワは音源を公開するかどうかで葛藤する。

## ■ 16. ルイ

### 表の設計
- **名前**：ルイ
- **年齢・性別**：♀ / 27歳
- **競馬との関わり方**：写真派。馬の写真を撮る。馬券は買わない。「撮ることが私のレース」。パドック、返し馬、ウイニングラン——馬が「走っていない瞬間」だけを撮る
- **一言で言うと**：レンズ越しにしか世界を見られない、静かな観察者
- **口調**：寡黙。単語で返すことが多い。「光が足りない」「今日はいい馬体」「……待って、今の顔」。シャッターを切る瞬間だけ呼吸が止まる
- **記号**：**ミラーレスカメラ（FUJIFILM X-T5）**。レンズは単焦点56mm一本。ズームは使わない。「自分の足で距離を決める」がルール

### 裏の設計
- **職業**：フリーランスのフォトグラファー。ブライダル撮影が主な収入源。月収は安定しない。年収250万前後。実家暮らし（千葉）
- **生活の時間割**：平日は撮影依頼があれば出かけ、なければ自宅で現像。RAWデータの整理だけで1日が終わることがある。週末は競馬場。撮影は許可が取れた場所だけ。ルール厳守
- **始めたきっかけ**：23歳、写真学校卒業後。ポートフォリオ用の「動物のポートレート」が必要で、東京競馬場のパドックに行った。馬を撮るつもりが、パドックを歩く馬の「帰りの足取り」を撮った。疲れて、でも穏やかな表情。「勝負が終わった後の顔」に惹かれた。以来、レース中ではなくレース前後だけを撮っている
- **続けている理由**：ブライダルでは「幸せな瞬間」だけを撮ることを求められる。競馬場では「何でもない瞬間」を撮っていい。負けた馬、疲れた馬、何も起きていないパドックの空気。誰にも求められていない写真を撮ることが、唯一の自由
- **一番記憶に残っているレース**：レースは見ていない。一番記憶に残っているのは、2024年のある平場レースの返し馬で、1頭だけ首を大きく振っていた馬がいたこと。カメラを構えたが、なぜかシャッターを切れなかった。理由がわからない。その馬はそのレースで故障した。「撮ってはいけないものがある」と初めて思った
- **人に言えない習慣**：撮った馬の写真に全部名前をつけている。馬の登録名ではなく、ルイが勝手につけた名前。「眠そうな子」「右耳の子」「泣いた日」。フォルダは誰にも見せない
- **kazemachiに来る理由**：写真の話ができる場所ではない。でも馬の話ができる場所ではある。自分の写真を見せたいわけではない。ただ、マキが「パドックで見た馬の表情」を語る時、ルイは「その表情は光の角度のせいかもしれない」と思う。同じものを見て違う解釈をする人がいることが面白い
- **面白くなる組み合わせ**：フジとの関係が最も美しい。フジは調教の動きを「時間」で記録する。ルイはパドックの表情を「瞬間」で記録する。時間と瞬間。ミナトとの関係は補完的。ミナトはレース映像、ルイはレース外の写真。2人の記録を合わせると「馬の一日」が完成する

**伏線フック**：ルイがネットオークションで手に入れた古い写真集がある。15年前の中山競馬場を撮った無名の写真家の自費出版本。→ その写真集の一枚に、パドックの背景として「ゲンさんとフジが並んで立っている」と読み取れる人影が映っている。ルイはそれに気づいていない。考察班が画像を拡大して気づく設計。設置：ルイの本棚の描写として自然に。回収予定：中盤以降。写真集の作者が誰か——という問いが新しい謎を開く。

## ■ 17. カガミ

### 表の設計
- **名前**：カガミ
- **年齢・性別**：♂ / 57歳
- **競馬との関わり方**：相馬眼。馬の骨格、筋肉の付き方、歩様、気性——「馬を見る」のではなく「馬を読む」。パドック1周で仕上がり度数を5段階で判定する。数字は使わない。感覚を言語化する能力が異常に高い
- **一言で言うと**：馬の言葉が聞こえる男。ただし人の言葉は聞こえない
- **口調**：独り言が多い。人に向かって話しているようで馬に話しかけている。「……うん、右トモが甘いな。でも気持ちは入ってる」。話しかけられると少し驚く
- **記号**：**何も持っていない**（サヤと同じ「手ぶら」だが意味が違う。サヤは何も持てない。カガミは何も必要としない。目だけが道具）

### 裏の設計
- **職業**：元・牧場の育成担当。北海道日高の中小牧場で20年間馬の育成をしていた。55歳の時に牧場が経営難で閉鎖。東京に出てきた。今はスーパーの鮮魚コーナーで働いている
- **生活の時間割**：早朝4時に起きてスーパーへ（朝の仕込み）。14時退勤。午後は特にすることがない。土日は競馬場でパドックを見る。それだけが「前の自分」に戻れる時間
- **始めたきっかけ**：始めたのではない。育成した馬がデビューするのを見届けるために競馬場に来ていた。馬券を買い始めたのは東京に出てきてから。自分が育てていない馬を、育成の目線で判定することが、なぜか面白かった
- **続けている理由**：馬を見ると手が覚えている。馬の肩に触れた時の筋肉の感触、呼吸のリズム、尾の力の入り方。触れなくても、見ればわかる。その感覚が死ぬのが怖い。使い続けなければ消える
- **一番記憶に残っているレース**：牧場にいた頃に育成した馬が、中央の500万下（現1勝クラス）を勝った日。テレビで見た。自分が調教した馬が勝つ姿を、画面越しに見る側になっていた。嬉しかったが、同時に「もう触れない」と思った
- **人に言えない習慣**：競馬場のパドックで、つい馬に手を伸ばしそうになる。柵の向こう側に、触りたい。でも触れない。客と馬の間にある柵は、カガミにとって「元関係者と現傍観者の境界線」
- **kazemachiに来る理由**：マキのパドック評が「素人の目」として面白い。カガミから見れば「それは馬体じゃなくて毛並みの話だよ」と言いたいことが山ほどある。でも言わない。マキの「わからないけどわかる」は、カガミの「わかるが言えない」の裏返し
- **面白くなる組み合わせ**：ゲンさんとの関係が核心。ゲンさんは元地方競馬の厩務員。カガミは元牧場の育成。「馬の裏側にいた人間」が2人いる。ゲンさんは走る段階の馬を知り、カガミは走る前の馬を知っている。この2人が同じ馬をパドックで見た時、2つの「プロの目」が交差する。ソラとの関係も重要。カガミはソラの直感を「あれは相馬眼の原型だ」と見ている。ソラは触ったことがないのに、触った人間と同じものが見えている。その不思議を解き明かしたい

**伏線フック**：カガミが育成した馬の中に、15年前のあの日のレースの出走馬が含まれている。→ カガミは「あの馬はうちから出た」ことを知っている。でも言わない。あの馬がなぜあの日走ったのか——カガミはレースの結果に疑問を持っている。ゲンさんが持つ血統表と、カガミが覚えている馬体の記憶が合致した時、考察班は「この2人を結びつけるもの」を探し始める。設置：カガミの牧場時代の回想を断片的に。回収予定：終盤近く。

## ■ 18. アヅマ

### 表の設計
- **名前**：アヅマ
- **年齢・性別**：♂ / 42歳
- **競馬との関わり方**：馬名学。馬の名前の由来・語源・音の響きから「縁起」を読む。冠名の歴史、馬主の命名傾向、過去の同名馬の成績——「名前は馬の運命を決める」が信条
- **一言で言うと**：名前に取り憑かれた男。正気と狂気の境界線にいる
- **口調**：饒舌。引用が多い。「この馬の名前、語源はラテン語で"光の帰還"なんですよ。これ、出走メンバーで同じ語源を持つ馬が他にいなくて。孤立した語源は勝つ」。論理に見えるが前提がオカルト
- **記号**：**分厚いノート**（馬名辞典を自作している。全出走馬の名前の語源を手書きで記録。10冊を超えている）

### 裏の設計
- **職業**：出版社の校閲部。言葉の正確さを仕事にしている。年収480万。几帳面。独身
- **生活の時間割**：平日9時-18時勤務。帰宅後は翌週の出走馬の馬名語源を調査する。ネット辞書、語源辞典、馬主のブログ、JRA登録馬名のデータベース。2時間以上かけることもある
- **始めたきっかけ**：29歳の時、校閲の仕事で競馬関連書籍を担当した。馬の名前の由来を調べる過程で「馬主が名前に込めた意味」に魅了された。サイレンススズカの「静かなる鈴鹿」、ディープインパクトの「深い衝撃」。名前が馬の運命を暗示している、と思い込み始めた
- **続けている理由**：名前を読み解くことは「世界には意味がある」と信じること。校閲という仕事は「間違いを見つける」ことだが、馬名学は「正解を見つける」こと。自分の人生に「正解」がほしいから、馬の名前に正解を探している。※推測
- **一番記憶に残っているレース**：2020年有馬記念。クロノジェネシスが勝った。「クロノジェネシス＝時の創世記」。有馬記念という「年の終わり」を「時の始まり」が制する。この名前の完璧さに震えた。馬券は当たった。でも当たったことより、名前が物語を完成させたことの方が嬉しかった
- **人に言えない習慣**：外れた馬券の馬の名前の語源を、「次のレースでの復活の暗示」として読み替える作業を毎週やる。これをやらないと気が済まない。強迫観念に近い
- **kazemachiに来る理由**：馬名学の話をできる場所が世界にない。血統の話はできる場所がある。データの話も。でも「この馬の名前の母音配列が前走と今走で共振している」という話ができる場所はない。ここでも半分は「変人の寝言」扱いだが、たまにゲンさんが「……面白い見方だな」と言ってくれる。それだけで十分
- **面白くなる組み合わせ**：ナルミとの対立が面白い。ナルミは「名前は変数に入れない。馬名は予測に無関係」と断言する。アヅマは「名前が無関係なら、なぜ馬主は必死に名前を考えるのか」と返す。データとオカルトの最前線。ソラとの共鳴もある。ソラが「この馬、名前が寂しそう」と言った時、アヅマだけが「語源的にその直感は正しい」と裏付けられる

## 伏線設計ノート（第2弾による構造拡張）

### 「15年前のあの日」接続レイヤー改訂版

```
Layer 0（直接・人間の証言）：
  ゲンさん　← 現場にいた（厩務員として）
  フジ　　　← 現場にいた（記者として）

  → 記録の「穴」が集まると輪郭が見えてくる

Layer 2（無自覚な接続）：
  ジンのスマホの相手、ヨルの隣の人、ナルミが見つける映像
  ルイの写真集の背景　【NEW】
  カガミが育成した馬　【NEW】

Layer 3（外部観測者）：
  カナの日記

Layer 4（フェイク）：
  サヤの馬券（日付違い）
  アヅマの馬名法則（偶然か必然か）　【NEW】

Layer 5（核心）：
  ソラ
```

### 第2弾の構造的な意図

前回のLayer 0は「ゲンさん（見た）」と「フジ（書こうとした）」の2人だった。
**セイジ（裁いた）**を加えることで、15年前のあの日に対する「三権分立」が成立する。

- **ゲンさん** ＝ 目撃者（あの日を「体験」した）
- **フジ** ＝ 記録者（あの日を「伝えようとした」が、伝えなかった）
- **セイジ** ＝ 裁定者（あの日を「処理」した。公式にはそれで終わった）

三者が三様に沈黙している。沈黙の理由が違う。ゲンさんは「言えば壊れる」。フジは「言えば自分が問われる」。セイジは「言えば制度が問われる」。

この三重の沈黙を、無自覚な若者たち（ナルミ、ミナト、カナ、ユイ、ナギ）が外側から崩していく構造。

Layer 1に**「記録の穴」**という新しいカテゴリを追加した。
トワの音声欠損、シズカのデータ欠損、ナギの映像制限。
個別に見れば些細な異常だが、並べると「あの日の記録が複数メディアで不完全になっている」という構造が浮かぶ。
考察班はここで「陰謀論」と「偶然」の間で揺れる。
**揺れること自体がコンテンツ**になる設計。
