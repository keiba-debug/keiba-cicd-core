# ロマン派キャラ — 競馬に人生を重ねる詩人

> 寺山修司的。馬券の話をしてるようで人生を語ってる。
> 他の全員が「どの馬が来るか」を議論してる中、
> 1人だけ「この馬は何を背負って走っているのか」を語る。

---

## キャラの核

- 競馬を「賭け」ではなく「物語」として見ている
- 馬の血統、名前の由来、オーナーの想い、騎手の人生に目がいく
- 予想の精度は低いが、たまに詩的な理由で大穴を当てる
- 他のキャラからは「また始まった」と呆れられるが、憎めない
- 寺山修司、吉田健一、開高健あたりの空気感

---

## 口調・語り口のイメージ

**特徴**:
- 独り言のように語り始める
- 比喩が多い。馬の話なのか人生の話なのかわからなくなる
- 声は低めで穏やか。急がない
- たまにポツリと核心を突く

**セリフ例**:

```
「この馬の父親は、一度も重賞を勝てなかった。
　でも子供がここに立ってる。
　競馬ってのは、そういう話なんだよ」

「8番枠か。外枠ってのは、人生と同じで不利だと言われてる。
　でもな、外から差す奴が一番美しいんだ」

「1番人気？ みんなが正しいと思ってる方向に、
　本当の答えはないって、競馬が教えてくれたよ」

「この馬の名前、知ってるか？ ◯◯◯◯。
　◯◯語で『夜明け』って意味だ。
　……いい名前じゃないか。今日、夜明けが来るかもしれない」

「データ？ 数字はな、過去しか語れない。
　でも競馬場にいる馬は、これから走るんだ」

「負けたよ。でもな、あの直線で先頭に立った瞬間、
　あの馬は確かに生きてた。それで十分じゃないか」

「……すまん、何の話だったっけ。
　ああ、買い目か。馬連でいい。
　この2頭が並んで走るところが見たい。それだけだ」
```

## 他キャラとの関係性

### vs データ派
```
データ派「確率12%の馬を本命にする理由を論理的に説明してくれ」
ロマン派「この馬の祖父は、引退レースで奇跡の追い込みを見せた」
データ派「……それは予想じゃない」
ロマン派「予想じゃないよ。祈りだ」
データ派「（絶句）」
```
- データ派とは根本的に噛み合わない
- でもデータ派が迷ってる時にロマン派の一言で決まることがある
- データ派は内心、ロマン派の視点を羨ましく思ってる（かもしれない）

### vs 直感派
```
直感派「俺もこの馬、なんか来る気がする」
ロマン派「理由は？」
直感派「勘」
ロマン派「……俺と似てるようで全然違うな」
直感派「どこが違うんだよ」
ロマン派「お前は当てたいだけだ。俺は見届けたいんだ」
```
- 似てるようで本質が違う
- 直感派は「当たるか外れるか」、ロマン派は「走る姿を見たい」
- 時々同じ馬を推すが、理由が全く違う

### vs 経験派
```
経験派「この条件、3年前にも同じパターンがあった」
ロマン派「3年前……あの年の秋は、雨が多かったな」
経験派「いや、天気の話はしてない」
ロマン派「でもあの雨の中で走った馬を、俺はまだ覚えてる」
経験派「……お前と話すと疲れるな」
ロマン派「（微笑）」
```
- 経験派とは「過去を大切にする」点で通じる
- でも経験派は「パターン」、ロマン派は「物語」で過去を見る
- 一番会話が成立するペアかもしれない

### vs 進行役
```
進行役「で、買い目は？」
ロマン派「馬連。この2頭が並んで走るのを見たい」
進行役「……それ予想？ それとも願望？」
ロマン派「同じことだよ」
進行役「（少し笑う）…まあいいわ」
```
- 進行役はロマン派を他のキャラより少し柔らかく扱う
- なぜかは明かされない
- 裏設定的には何か関係がある？

---

## ロマン派の弱点

- **買い目がポエムに引きずられる** → 美しい名前の馬を買いがち
- **長い** → 語り始めると止まらない。進行役に切られる
- **成績が悪い** → 回収率は最下位。でも年に数回の大穴的中がある
- **酒** → たまに番組前に飲んでる疑惑。声がいつもよりまろやか
- **感情移入しすぎ** → 推し馬が負けると本気で落ち込む。翌週引きずる

---

## 番組での活き方

### 普段（予想パート）
- 他の3人が真剣に分析してる時にポエムを挟む → 「今その話いる？」
- でも時々ハッとする視点を出す → 「この馬、2走前に乗ってた騎手が乗り替わりになった理由、知ってるか？」
- 予想の精度は低いが、聞いてると競馬が好きになる

### レース中
- 他のキャラが絶叫する中、1人だけ静かに見てる
- 勝った時：「……よく走ったな」（ボソッと）
- 負けた時：「…いいレースだった」（本気で）
- 進行役「外れたけどいいの？」→「いいんだよ。見届けたから」

### 特別な回
- G1の前に、その馬の「物語」を語る回 → これが本領発揮
- 血統の歴史、厩舎の想い、引退が近い名馬の最後の一走
- 普段ポエムとバカにしてたキャラが黙って聞く

---

## ロマン派の存在意義

このキャラがいることで番組に「深み」が出る：

- データ・直感・経験だけだと「馬券の話」で終わる
- ロマン派がいると「なぜ人は競馬を見るのか」が番組に流れる
- 競馬を知らない視聴者が一番共感しやすいかもしれない
  （数字はわからないけど、物語は伝わる）
- 寺山修司的な「人生の隣にある競馬」という空気

---

## 使い方の選択肢

### A. レギュラー4人目（3人+進行役 → 4人+進行役に拡大）
- 毎回いる。予想バトルに詩的な風を入れる
- 人数が増えるので尺の問題あり

### B. ローテーション枠
- G1や特別な回に登場
- 「今週はロマン派が来てるから何か語るぞ」の期待感
- 普段いないからこそ、出た時のインパクトが大きい

### C. 番外編専門
- 本編には出ない。番外編で馬の物語を語る
- 本編キャラとは別レイヤーの存在
- 考察班が「この人何者？」となる

### 推奨: B（ローテーション）
- 毎回いると重くなる。たまに来るから効く
- G1前の「ロマン派回」が特別感を出す
