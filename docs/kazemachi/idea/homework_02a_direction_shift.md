# 宿題②a — コンテンツの方向性を見直す

> 各作家のシステムプロンプトの後ろに貼って投げる。
> キャラクター画像（MC、ソラ、ゲンさん等）を参考として添付。

---

```
━━━━━━━━━━━━━━━━━━━━━
■ 方向転換の提案
━━━━━━━━━━━━━━━━━━━━━

キャラクターのビジュアルを作ってみて、気づいたことがある。

今の設計は「AI競馬予想バトル番組」に寄りすぎている。
予想は確かにコンテンツの柱だが、それだけだと：
- 発信が土日に集中する（平日ネタがない）
- 「的中率」「回収率」で評価されてしまう（外れたら価値がないと思われる）
- キャラの魅力が「予想の上手さ」に縛られる

━━━━━━━━━━━━━━━━━━━━━
■ 新しい方向性
━━━━━━━━━━━━━━━━━━━━━

「予想バトル番組」ではなく、
**「競馬好きのコミュニティ」**。

キャラクターたちは競馬が好きな人たちの集まり。
予想はその活動の一つに過ぎない。

【コンテンツの幅が広がる】
- 予想（土日）：これは変わらない。各キャラの根拠が面白いのも変わらない
- 競馬場グルメ：みんなで競馬場のおいしいものを食べに行く
- パドック・馬の話：馬の美しさ、血統のロマン、厩舎訪問
- ギャンブルとの向き合い方：依存症について真剣に話し合う回もある
- 歴史・名馬の物語：ゲンさんが語る昔のレース、テツの「あの馬」の話
- 収支管理：フジさんの冷静な数字の話。健全に楽しむための知恵
- 競馬場の旅：地方競馬場巡りなど
- 初心者ガイド：そもそも馬券ってどう買うの？

→ 毎日発信できる。月〜金は「競馬のある生活」、土日は「予想」

【キャラの魅力が予想力に依存しなくなる】
- ゲンさんの話は外れても面白い（昔話自体がコンテンツ）
- ソラの直感は当たらなくても「あの瞬間に何を感じたか」が面白い
- ギャンブル依存の話ができるのは「この人たちが信頼できる」証拠

【進行役の再定義】
- 「番組のMC」ではなく「コミュニティのつなぎ役・主催者」
- みんなをつなぐ、企画を立てる、場を作る役割
- 女の子一人がこの役目にいるのは良いと思った
- 「平凡だけど精度がおかしい」よりも「平凡だけどみんなが集まってくる」

━━━━━━━━━━━━━━━━━━━━━
■ 変わること・変わらないこと
━━━━━━━━━━━━━━━━━━━━━

【変わらない】
- キャラクターたち自体（ソラ、ゲンさん、ナルミ、テツ等）
- 「根拠が面白い」が予想コンテンツの核
- 外れ予想は消さない
- Phase 1はX投稿
- 画像テイストは日常アニメ系
- KeibaCICDのチラ見せ（ツールとして自然に出る）

【変わる】
- コンテンツの範囲：予想だけ → 競馬のある生活全般
- 発信頻度：週末中心 → 毎日何かしら
- 進行役の役割：MC → コミュニティ主催者
- 評価軸：的中率 → 「この人たちと一緒に競馬を楽しみたいか」
- トーン：バトル → 仲間

━━━━━━━━━━━━━━━━━━━━━
■ 聞きたいこと
━━━━━━━━━━━━━━━━━━━━━

1. この方向転換についてどう思うか？ 賛成・反対・条件付き賛成、何でもいい
2. 「競馬好きのコミュニティ」にした場合、あなたの専門領域で何が変わるか？
   - カイ：コンテンツの構成・テンポ設計はどう変わる？
   - ユキ：キャラの感情設計はどう変わる？ 進行役の再設計は？
   - レイ：伏線・仕掛けの設計はどう変わる？ 予想以外のコンテンツに仕込めるものは？
3. 「予想バトル」要素は完全になくすべきか、コミュニティの中の一コーナーとして残すべきか？
4. 平日のコンテンツ案を3つ以上出してほしい
5. ギャンブル依存の話を真剣に扱う場合、キャラの誰がその役割を担うべきか？
6. この方向転換で「これだけは失ってはいけない」と思うものは何か？

自由に書いてくれ。
この判断は大きいので、全員の本音が聞きたい。
```
