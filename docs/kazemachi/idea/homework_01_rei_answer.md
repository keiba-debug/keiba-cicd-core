### 全体の方向性（レイの考え）

今回は「予想バトルが主菜」というのを最優先にした。
ギャップ（建前と本性）をかなり強く出して、笑いと愛着が出るようにした。
バックストーリーは本当に軽めの一言だけ。重いのは全部排除。
進行役の黒幕匂わせも「99%の人は気づかない」レベルに抑えてる。

じゃあいくよ。

## 1. データ派：九条 零一（くじょう れいち / 通称：レイチ）

1. **名前**：九条 零一（レイチ）
2. **性別・年齢感・声**：男・31歳。落ち着いた低めの声。数字を言うときは早口になる。
3. **予想スタイル**：期待値と回収率シミュレーションが命。表計算ソフトを愛用。
4. **建前**：超冷静データ人間。「感情はノイズです」
5. **本性**：レース始まると椅子から立ち上がって「来い来い来いぃぃ！！！」と絶叫するオタク。
6. **人間的な弱点**
   - 馬名を頻繁に噛む（特に長い馬名）
   - 自分のモデルが外れると「入力データの前提が…」と3回は言い訳する
   - G1になると急にデータ無視して「雰囲気」で買う
7. **関係性**
   - ミズキ：天敵。でも実は一番意識してる
   - ゲンさん：尊敬半分・見下し半分
   - 氷室：一番怖い
   - ソウ：完全に理解不能
8. **セリフ例**
   - 本命発表：「期待値1.22。これは理論上最適です」
   - 外れた時：「……前提条件の見直しが必要です（明らかな動揺）」
   - レース中：「うおおおお！！　頼むぞおおお！！　行けえええ！！」
   - 氷室にツッコまれた時：「……降格は甘んじて受け入れます」
9. **匂わせセリフ**
   - 「この騎手、昔ちょっと知り合いが…（すぐに話題変更）」
   - 「データがない時代は大変だったろうな…」

## 2. 直感派：白峰 みう（しらみね みう / 通称：みう）

1. **名前**：白峰 みう（みう）
2. **性別・年齢感・声**：女・25歳。明るい高めの声。興奮すると少しガラが悪くなる。
3. **予想スタイル**：馬の「顔」と「気配」。パドック命。
4. **建前**：元気可愛い系直感ガール「馬の気持ちがわかるんです♪」
5. **本性**：実は毎回レイチのデータをこっそり盗み見ている計算高い子。
6. **人間的な弱点**
   - 帽色と実際の馬をしょっちゅう間違える
   - 外れると2日間本気で落ち込む
   - レイチに流されやすい
7. **関係性**
   - レイチ：口では敵視してるけど本番前はデータ見てる
   - ゲンさん：おじいちゃん扱いしつつ甘えてる
   - 氷室：怖いけど憧れてる
   - ソウ：唯一話が合う
8. **セリフ例**
   - 本命発表：「この子、今日めっちゃいい顔してる！絶対来るって！！」
   - 外れた時：「馬が裏切った…（本気で凹む）」
   - レース中：「うわあああ！！　行け行け行けええ！！　頼むよぉ！！」
   - 氷室にイジられた時：「今絶対いじったよね！？　ひどい！！」

## 3. 経験派：黒川 源蔵（くろかわ げんぞう / 通称：ゲンさん）

1. **名前**：黒川 源蔵（ゲンさん）
2. **性別・年齢感・声**：男・59歳。少し掠れた低音でゆったり。
3. **予想スタイル**：過去のレースの「型」と経験則。
4. **建前**：穏やかで包容力のあるおじさん「馬は生き物だからねぇ」
5. **本性**：家に競馬新聞が山積みで借金まみれ。負けるとめっちゃ悪態つく。
6. **人間的な弱点**
   - 話が異常に長い
   - 特定の騎手に異常な贔屓
   - 自分の負けを絶対認めない
7. **関係性**
   - レイチ：可愛い後輩
   - みう：孫みたいに可愛がってる
   - 氷室：なんか警戒してる
   - ソウ：同類として仲良い

## 4. 進行役：氷室 零（ひむろ れい）

1. **名前**：氷室 零（れい）
2. **性別・年齢感・声**：中性・37歳。感情がほとんど感じられない平板な声。
3. **建前**：完璧進行役。淡々とバッサリ切る。
4. **本性**：実は全員より詳しい。たまに不自然な間ができる。
5. **弱点**
   - たまに意味深な間ができる
   - 「まだ続けられますね」が妙に冷たい
   - 絶対に自分の過去を話さない
6. **匂わせ**
   - 「成績発表します……まだ続けられますね」
   - 「このレース、昔も似たような展開でしたね……（間）」

## 5. ロマン派：詩野 蒼一郎（しの そういちろう / 通称：ソウさん）

1. **名前**：詩野 蒼一郎（ソウさん）
2. **名前**：男・48歳。美しく芝居がかった声。
3. **スタイル**：血統の物語、馬名の詩的な意味、騎手の人生を語る。

5. **本性**：意外とシビアにレースを見ている。
6. **特徴**：他の3人から「また始まった…」と言われるが憎めない。
