# 作家陣への宿題② — プロトタイプ脚本

> キャラが固まったら、実際の1回分の脚本を書いてもらう。
> {キャラ設定} の部分を宿題①の結果で埋めてから投げる。

---

## プロンプト

```
あなたは競馬予想バトル番組の脚本家です。
以下のキャラクター設定と番組構成に基づいて、第1回の脚本を書いてください。

━━━━━━━━━━━━━━━━━━━━━
■ キャラクター設定
━━━━━━━━━━━━━━━━━━━━━

{宿題①で確定したキャラ設定をここに貼る}

━━━━━━━━━━━━━━━━━━━━━
■ 対象レース（例）
━━━━━━━━━━━━━━━━━━━━━

レース名: 高松宮記念（G1）
コース: 中京 芝1200m
頭数: 18頭
馬場: 良
コース特性: 直線が長め、差し馬にもチャンス

注目馬（公開情報のみ、3-5頭）:
- A馬: 前走スプリント重賞1着、1番人気想定
- B馬: 前走大敗、でも2走前は重賞勝ち。調子の波が激しい
- C馬: 初G1挑戦。条件戦を圧勝して勢いあり。人気薄
- D馬: ベテラン8歳馬。全盛期は過ぎたがコース実績抜群
- E馬: 外国馬。データが少なく読みにくい

━━━━━━━━━━━━━━━━━━━━━
■ 番組構成（約12分想定）
━━━━━━━━━━━━━━━━━━━━━

[0:00-0:30] OP
  - 進行役がレース概要を紹介
  - 軽い挨拶、前回の振り返りを一言

[0:30-1:30] 前回の成績発表
  - 進行役が淡々と数字を出す
  - 外した人がイジられる
  - ここでキャラの性格と関係性が見える

[1:30-3:00] レース概要・コース解説
  - コース特性、馬場状態、展開予想
  - 各キャラが自分の視点でコメント

[3:00-5:00] 本命発表
  - 3人が順番に本命馬と理由を発表
  - 進行役が「ほう」「また？」等のリアクション
  - この時点で意見が割れる（or 一致して気まずくなる）

[5:00-8:00] 予想バトル
  - 互いの予想にツッコミ・質問・反論
  - 一番熱くなるパート
  - 「その馬のどこがいいの？」「お前の根拠は？」
  - ギャップが出る場面を1つ入れる

[8:00-10:00] 穴馬・危険な人気馬
  - 人気馬を消す（緊張感）
  - 穴馬を推す（ロマン）
  - ここでも対立が生まれる

[10:00-11:00] 最終予想・買い目発表
  - 進行役が整理「つまりこういうこと？」
  - 各キャラの最終的な買い目を確定
  - 馬券種と金額も（少額・リアル購入する想定）

[11:00-12:00] ED
  - 進行役の締めの一言
  - 次回予告的なチラ見せ

━━━━━━━━━━━━━━━━━━━━━
■ 脚本のルール
━━━━━━━━━━━━━━━━━━━━━

1. 台本形式（キャラ名: セリフ）で書く
2. ト書き（動作・表情指示）も入れる
3. 各キャラの口調・性格が明確に区別できること
4. 予想の中身は具体的に（「この馬は〜だから」レベルで）
   ただしIDM等の具体的指数名は使わない
   「能力的にメンバー中トップ」「データ上は過小評価されてる」等の表現を使う
5. バトルパートで最低2回は意見が衝突すること
6. ギャップ（建前→本性）が最低1回出ること
7. 進行役のツッコミが最低3回効くこと
8. バックストーリーの匂わせは入れても入れなくてもいい（第1回なので）
9. 笑えるシーンを最低2つ入れること
10. 専門用語を使ったら、会話の流れで自然に説明すること
    （初心者が聞いてもわかるように）
```
