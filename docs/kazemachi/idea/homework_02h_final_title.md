# 宿題②h — 最終タイトル・プロジェクト名決定

> 各作家のシステムプロンプトの後ろに貼って投げる。
> ②f, ②gの全回答 + 以下のヒントを添付する。

---

```
━━━━━━━━━━━━━━━━━━━━━
■ キーワードブレスト、素晴らしかった
━━━━━━━━━━━━━━━━━━━━━

3人合わせて160語以上、そこからクロスして組み合わせも出揃った。
その後、さらにヒントをもらって新しい方向が見えてきた。

━━━━━━━━━━━━━━━━━━━━━
■ 構造のヒント：「昨夜のカレー、明日のパン」
━━━━━━━━━━━━━━━━━━━━━

木皿泉の小説/ドラマ「昨夜のカレー、明日のパン」のタイトル構造が
この企画にかなり合うことに気づいた。

この構造の気持ちよさを分解すると：

1. 時間の対比がある（昨夜 ↔ 明日）
2. 具体的な名詞がある（カレー、パン）
3. 「今日」が省略されている
4. 食べ物なのに人生が見える

この企画で言えば：
- 「今日」は毎週のレースが埋める（自分ではコントロールできない）
- 先週の余韻と来週の期待の間に、予測不能な「今」がある
- この構造自体がこの作品のリズムそのもの

━━━━━━━━━━━━━━━━━━━━━
■ この構造で生まれた候補
━━━━━━━━━━━━━━━━━━━━━

【対比構造あり】
- 先週の根拠、来週の予感
- 先週の隣、来週の席
- 昨日の外れ、明日の予感
- 先週の沈黙、来週の一言
- 外れた馬券、来週の風
- 先週の常連、来週の余韻
- 昨日の余韻、また来週
- 帰り道のカレー、来週の席

【短く攻めたもの】
- 外れたまま、また来週
- まだ、風待ち

━━━━━━━━━━━━━━━━━━━━━
■ これまでの有力候補まとめ（全セッション通じて）
━━━━━━━━━━━━━━━━━━━━━

【タイトル候補】
- 競馬で生きてる（カイ原案、全員支持）
- 来週も隣で（ユキ本命）
- 帰り道の話（レイ本命）
- 常連の余韻（カイ本命）
- いつかの馬の話（レイ案）
- 競馬のあとで話そう（ユキ案）
- 踊り場の常連（ユキ案）
- 風待ちの常連（レイ案）
- まだ、隣で（レイ案）
- 外れたまま、また来週（新案）
- 先週の根拠、来週の予感（新案）

【コードネーム候補】
- kazemachi（ユキ推薦）
- yoin（レイ推薦）
- tonari-no-yoin（カイ推薦）

━━━━━━━━━━━━━━━━━━━━━
■ 最終宿題：2つ決めてほしい
━━━━━━━━━━━━━━━━━━━━━

1. 【作品タイトル】
   外向きの正式名称。読者が最初に目にする名前。
   上の候補から選んでもいいし、新しく作ってもいい。
   理由を添えて1つ選んでくれ。

2. 【内部プロジェクト名（コードネーム）】
   フォルダ名、ファイル名、会話で使う短い呼び名。
   英語 or ローマ字。短いほどいい。
   タイトルと同じでも、違ってもいい。
   1つ選んでくれ。

【判断基準として意識してほしいこと】
- 「当たる予想が見られる」を期待させない
- 「この人たちの話が見たい」と思わせる
- 略称で呼べる（会話の中で「〇〇見た？」が成立する）
- 数字を含まない（キャラ数は今後変わりうる）
- この作品の空気感に触れている

最後なので、迷いがあればそれも書いてくれ。
「AとBで迷っている」でもいい。
決め切れなくても、その理由が聞きたい。
```
