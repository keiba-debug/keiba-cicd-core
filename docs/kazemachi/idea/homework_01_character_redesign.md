# 作家陣への宿題① — キャラクター再設計

> 前回のブレスト結果を踏まえ、方向性を大幅修正。
> 以下のプロンプトを複数AIに投げてキャラ案を再収集する。

---

## プロンプト

```
あなたはキャラクターデザイナー兼脚本家です。
以下の番組企画に登場するキャラクターを設計してください。

━━━━━━━━━━━━━━━━━━━━━
■ 番組概要
━━━━━━━━━━━━━━━━━━━━━

AIアニメの競馬予想バトル番組。
毎週のレースについてキャラクターたちが本気で議論・対立・予想する。

【最重要】これは予想番組です。
- 本質は「レース前の予想バトル」。結果はまだわからない状態で面白くする
- 「なぜその馬なのか」の説得バトル、人気馬を消す緊張感、意見が割れる瞬間が見どころ
- バックストーリーやドラマはスパイス程度。主菜は予想の対立

【ターゲット】
- 競馬に詳しくない人がラジオ感覚で聞ける
- でも予想に関しては熱く、本気の対立がある

【トーン】
- 基本は笑えるエンタメ。時に熱い
- キャラは「完璧じゃないダメな大人たち」
- 真面目一辺倒ではなく、人間臭さと笑いがある

━━━━━━━━━━━━━━━━━━━━━
■ キャラクター構成
━━━━━━━━━━━━━━━━━━━━━

【レギュラー】
- 予想家3人：「データ派」「直感派」「経験派」
- 進行役1人：冷静ツッコミ型（後述）

【ローテーション枠（たまに登場）】
- ロマン派1人：競馬に人生を重ねる詩人（後述）

━━━━━━━━━━━━━━━━━━━━━
■ キャラ設計の絶対条件
━━━━━━━━━━━━━━━━━━━━━

【1. ギャップが命】
全キャラに「建前と本性」の二面性を持たせること。
例：
- 冷静にデータ分析する → レース中は「来い来い来いぃぃ！！」と絶叫
- 「期待値マイナスの馬券は買わない」→ 自分の買い目だけ理論と違う
- 穏やかに「馬は生き物だからね」→ 馬券で家のローンを2回組み直してる
- 「俺の勘を信じろ」→ 隣の人の買い目を見て急に乗り換える
- 「今日は見送り」→ 結局買う。毎回買う

このギャップが笑いと愛着を生む。真面目な分析パートとダメ人間パートの落差を大きく。

【2. 弱点は人間的に】
予想の精度だけでなく、人間としての欠陥を持たせる：
- 他人に流されやすい
- 自信がないのに虚勢を張る
- 馬名を微妙に間違える
- すぐ脱線する、話が長い
- 特定の馬や騎手に贔屓が入る
- G1になると急にビビる
- 先週の外れを絶対認めない

【3. 対立構造は三角形】
3人の予想家は、場面によって味方が変わる：
- AとBが組んでCを攻める回もあれば
- 次の週はBとCが意気投合してAが孤立する
- 全ペアに固有の対立理由がある（予想哲学＋性格の相性）

【4. バックストーリーは軽く】
重い過去やトラウマは不要。匂わせは会話の端々に自然に出る一言程度：
- ○「あー、この騎手か。昔ちょっとな…」（それ以上言わない）
- ○「このコース、俺にとっては因縁があんだよ」（笑って流す）
- ✕ 涙を流す、15年前のトラウマ、重い回想シーン

━━━━━━━━━━━━━━━━━━━━━
■ 進行役の設計
━━━━━━━━━━━━━━━━━━━━━

参考：ハンドイットオーバーのアレクシア
- 冷静に、容赦なくツッコむ。怒鳴るんじゃなく淡々とバッサリ
- ボケ（予想家）を最大限引き立てる聞き手
- 呆れてるけど見捨てない
- 自分のペースを崩さない

【特殊設定】
- 予想はしない。しかし視聴者が「この人、実は一番詳しい？」と感じる瞬間がたまにある
- なぜ予想しないかは謎。永遠に明かさなくてもいい
- 成績発表を淡々と行う。外した人への容赦がない
- 「で、結局どの馬？」「先週もそれ言って外したわよね」が基本

【裏設定（直接は描写しない）】
進行役には黒幕的な裏設定がある。表向きは楽しい番組だが、
注意深い視聴者だけが「この人、何かおかしくない？」と気づく。
- 成績発表の「まだ続けられますね」が妙に意味深
- たまに不自然な間がある
- これは匂わせレベル。99%の視聴者は気づかない程度に

━━━━━━━━━━━━━━━━━━━━━
■ ロマン派（ローテーション枠）
━━━━━━━━━━━━━━━━━━━━━

寺山修司や清水成駿のような存在。
- 競馬を「賭け」ではなく「物語」として見ている
- 馬券の話をしてるようで人生を語ってる
- 「お前は当てたいだけだ。俺は見届けたいんだ」
- 血統の歴史、馬名の由来、騎手の人生に目がいく
- 予想精度は低いが、たまに詩的な理由で大穴を当てる
- 他のキャラから「また始まった」と呆れられるが憎めない
- G1前など特別な回に登場。毎回いると重いのでたまにが効く
- 「外れても読みたい（聞きたい）」と言われる存在を目指す

━━━━━━━━━━━━━━━━━━━━━
■ 出力してほしいもの
━━━━━━━━━━━━━━━━━━━━━

【各キャラ（5人分）について】
1. 名前（呼びやすいもの。ニックネーム可）
2. 性別・年齢感・声のトーン（ラジオで全員区別できること）
3. 予想スタイルの詳細（何を見て、どう判断するか）
4. 建前の人格（番組中の振る舞い）
5. 本性（馬券・レース中・負けた時のリアルな姿）
6. 人間的な弱点・欠陥（3つ以上）
7. 他の全キャラとの関係性（好き嫌い、リスペクト、イジり等）
8. セリフサンプル（以下の場面で各3つ以上）
   - 自分の本命を発表する時
   - 他人の予想にツッコむ時
   - 自分の予想が外れた時
   - レース中（本性が出る場面）
   - 進行役にイジられた時の返し
9. バックストーリーの匂わせセリフ（軽いもの3つ）

【全体】
10. 5人の関係性マップ（図で）
11. 第1回の冒頭5分の会話サンプル
    - あるレースの予想バトルが始まるところ
    - 各キャラの口調・性格・対立が自然に伝わること
    - ギャップ（建前→本性）が1回は出ること
    - 進行役のツッコミが効いていること
12. ロマン派が登場するG1回の会話サンプル（3分程度）
    - 通常の3人+進行役の空気がどう変わるか
```
