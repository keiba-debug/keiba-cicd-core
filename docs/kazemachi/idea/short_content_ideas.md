# ショートコンテンツ案

> 数秒〜1分で完結する発信。流入導線＋単体でも楽しめる。

---

## フォーマット別

### Instagram / TikTok / YouTube Shorts（15-60秒）

**① 一言予想カード（静止画+テキスト、5-15秒）**
```
[キャラ立ち絵]
「明日の◯◯賞、この馬だけは買っとけ」
  → 3番 ◯◯◯◯
  → 「理由？ 勘。」
  → 進行役「また勘…」
```
- 1枚絵+セリフ2-3個で完結
- 量産しやすい。毎レース出せる

**② 対立ワンシーン（15-30秒）**
```
データ派「この馬、数字的には完璧」
経験派「俺はこの馬、信用してない」
データ派「根拠は？」
経験派「…勘だよ勘」
データ派「お前それ直感派の仕事だろ」
進行役「仲良くしなさい」
```
- 予想バトルの一番面白い部分だけ切り出す
- 本編への導線「続きはプロフのリンクから」

**③ 結果速報（10-15秒）**
```
[レース名]
データ派: ◯ 的中！+1,200円
直感派:  ✕
経験派:  ◯ 的中！+800円

進行役「直感派さん、今月マイナス3000円です」
直感派「………」
```
- レース直後に出せるとリアルタイム感

**④ 危険な人気馬アラート（15秒）**
```
⚠️ 明日の◯◯賞
1番人気 ◯◯◯◯

データ派「人気しすぎ。実力以上にオッズが低い」
直感派 「俺も今回はパス」
経験派 「3人とも消しは珍しいぞ」

→ 買うなら自己責任で
```
- 「消し」情報は注目されやすい

**⑤ 週間成績発表（15-30秒）**
```
📊 今週の成績

🥇 経験派 +2,100円「言うたやろ」
🥈 データ派 +400円「想定通り」
🥉 直感派 -1,500円「来週見とけ」

累計: データ派+8,200 / 経験派+5,100 / 直感派-3,400
```

**⑥ キャラの一言コラム（15秒）**
```
[データ派の顔]
「今週覚えておくべき数字は1つだけ。
　◯◯の複勝率。3回に1回来る馬が8倍ついてる。
　以上。」
```
- キャラの個性が出る一言だけ
- 毎日1個出しても負担少ない

### X / Twitter（テキスト+画像）

**① レース前ティーザー**
```
明日の◯◯賞、3人の意見が真っ二つ 🔥
データ派 vs 直感派、本命が正反対。
経験派は沈黙。何か知ってる？

詳しくは → [Noteリンク]
```

**② 結果速報**
```
◯◯賞 結果

データ派 ◯（+1,200円）
直感派  ✕（-1,000円）
経験派  △（±0円）

直感派「次は当てる」←先週も言ってた

#予想TV #競馬
```

**③ キャラ語録**
```
今日の経験派:
「お前ら若いのはすぐ数字に頼る。
　馬は数字じゃねえ。
　……まあ今月の収支は数字だけどな」

#予想TV
```

## 量産のコツ

### 本編から自動切り出し
```
本編スクリプト（Claude API生成）
    │
    ├─ 対立が一番熱い2-3往復 → ショート動画用
    ├─ 各キャラの本命+一言 → カード画像用
    ├─ 進行役のツッコミベスト → 語録用
    └─ 成績データ → 速報用
```

本編を作れば、ショートコンテンツは**自動で派生できる**。
別途作るんじゃなく、本編の副産物として出す。

### 制作コスト感
| コンテンツ | 制作時間 | 頻度 |
|-----------|---------|------|
| 一言予想カード | 5分（テンプレ+テキスト差し替え） | 毎レース |
| 対立ワンシーン | 10分（本編から切り出し） | 週2-3本 |
| 結果速報 | 5分（テンプレ+数字差し替え） | 毎レース日 |
| 成績発表 | 5分（自動集計+テンプレ） | 週1 |

## プラットフォーム戦略

```
X（テキスト速報）──→ 認知・拡散
  ↓
Instagram/TikTok（ショート）──→ キャラの魅力で引く
  ↓
Note / Web（本編）──→ じっくり読む・データ見る
  ↓
YouTube（将来・動画）──→ フル体験
```

全部同時に始める必要はない。
Note + X から始めて、キャラ立ち絵ができたらInstagramに展開、が現実的。
