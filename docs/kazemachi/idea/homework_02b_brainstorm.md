# 宿題②b — ブレスト：「競馬×キャラクター×物語」の軸を探す

> 各作家のシステムプロンプトの後ろに貼って投げる。

---

```
━━━━━━━━━━━━━━━━━━━━━
■ 方向性が決まった
━━━━━━━━━━━━━━━━━━━━━

宿題②aの方向転換、3人とも賛成ありがとう。
全員の意見を踏まえて、プロジェクトの勝ち条件を定義した。

【勝ち条件】
「競馬×キャラクター×物語」で、作品として成立する世界を作り、
その世界に人が集まった結果として収益が生まれる状態。

つまり：
- まず「作品」として面白いこと（物語・キャラ・世界観）
- その結果として人が集まること（フォロワー・ファン・コミュニティ）
- その結果として収益が生まれること（Note課金・CSV配布・グッズ等）

順序が大事。収益は結果であって目的ではない。
「売るために作る」ではなく「良いものを作ったら人が来た」を目指す。

━━━━━━━━━━━━━━━━━━━━━
■ 今回の宿題：自由ブレスト
━━━━━━━━━━━━━━━━━━━━━

上の勝ち条件を前提に、自由にアイデアを出してほしい。

テーマは「競馬×キャラクター×物語」の軸を探すこと。

【考えてほしいこと】

1. 「物語」とは何か？
   - このプロジェクトに「ストーリー」は必要か？
   - 必要なら、それはどんな種類のストーリーか？
     - 一本の大きな物語（始まりと終わりがある）？
     - 日常の積み重ね（終わらない日常系）？
     - キャラ同士の関係性の変化？
     - 読者が参加できるインタラクティブな物語？
     - 競馬そのもの（実際のレース結果）が物語を動かす？
   - 「ストーリー性を感じる」とはどういうことか？
     毎回の予想や投稿が単発で終わらず、
     何かが積み重なっている感覚。それは何で実現できる？

2. 「軸」とは何か？
   - この世界の中心にあるものは何？
   - キャラクターたちを同じ場所に集めている力は何？
   - 読者がこの世界に戻ってくる理由は何？
   - 類似プロジェクトとの差別化ポイントは？

3. キャラクターについて
   - 今のキャラクター設計（ソラ、ゲンさん、ナルミ、テツ、フジさん、マキ）を
     この新しい方向性でどう活かすか？ 変えるべきところは？
   - キャラクターの数は適切か？ 多すぎ？ 足りない？
   - 「物語」がある場合、主人公は誰か？ それとも群像劇か？

4. アウトプットの形
   - X投稿、Note、動画、音声、漫画…
     「作品として成立する」ために最適な形式は何か？
   - インスタ/TikTok/YouTubeショートなどの短尺動画も視野に入れている。
     キャラ×物語×競馬を30秒〜1分で切り取るとしたら、どんな形が効くか？
     （例：ゲンさんの昔話15秒、ソラのパドック感想30秒、フジさんの一言収支報告）
   - 予想コンテンツと物語コンテンツの比率は？
   - 毎日出すもの、毎週出すもの、月1で出すもの、
     それぞれ何が最適か？

5. 収益との接続
   - 「作品として面白い」がどの段階で「お金を払いたい」に変わるか？
   - 何にお金を払うのか？（予想？物語？キャラグッズ？コミュニティ参加？）
   - 無料で出すものと有料にするものの線引きは？

【ルール】
- 自由に書いてくれ。箇条書きでも長文でもいい
- 「こうすべき」だけでなく「こういうのはどうだろう？」も歓迎
- 他の2人の意見と矛盾してもいい。ブレストだから
- 実現可能性は今は気にしなくていい
- あなたの専門領域以外のことも遠慮なく踏み込んでいい
- 「これは絶対やめたほうがいい」も歓迎
- 具体的な投稿例やシーンのイメージがあれば書いてほしい

【あなたの視点で特に聞きたいこと】
- カイ：「作品として成立する構成」とは？ 週刊連載？ シーズン制？ どういう構造なら長く続く？
- ユキ：「物語を感じる」とはどういう感情の積み重ね？ キャラの関係性はどう変化すべき？
- レイ：「伏線と回収」を実際のレース結果と絡められるか？ 現実が物語を動かす仕組みは？

何の制約もなく、思いつくままに書いてくれ。
ここから軸を見つけたい。
```
