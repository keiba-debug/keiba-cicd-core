## 1. キャラクターシート — アキヒコ（49歳♂）

### 基本情報

| 項目 | 内容 |
|------|------|
| **呼び名** | アッキーさん（本人が望んだわけではない。自然とそうなった） |
| **年齢** | 49歳 |
| **世代** | 就職氷河期（1977年前後生まれ） |
| **職業** | IT系受託会社のシステムエンジニア（中堅）。変えてもよい |
| **家族** | 娘カズサ（10歳）。妻とは離婚 or 別居 or 妻が多忙——週末は父娘二人が多い |

### 性格

- **普段は適当**。省エネ。必要最低限しか動かない
- **愛想はよくない**。でも嫌な奴ではない。ただ面倒くさいだけ
- **人付き合いは得意じゃない**。自分からは話しかけない
- **でもいざという時には頼れる大人**。子供が泣いた時、誰かが困ってる時、黙って動く
- 「適当」の裏に「もう期待しない」がある。氷河期を生きた男の処世術

### 競馬との関係

- **原体験は1997-1998年**。20歳前後
- **サニーブライアン**: 誰も信じなかったのに皐月賞もダービーも勝った馬。「認められない側」にいた自分を重ねた
- **サイレンススズカ**: 圧倒的に強くて、突然いなくなった。98年天皇賞秋。語らないけど忘れてない
- **普段の買い方は適当**。新聞ざっと見るか、スマホでオッズだけ見て勘で買う。ペンすら持たないことも
- **スイッチが入る瞬間がある**。パドックで「何か」を感じた時、目つきが別人になる。本人も制御してない

### 他キャラとの差別化

| vs | アキヒコ | 相手 |
|----|---------|------|
| ゲン | 「別にどうでもいい」から黙る | 「見てきた」から黙る |
| ナルミ | データなんか見ない | データで安心したい |
| ソラ | 適当に買う | 本気で感じて買う |
| マキ | 場に馴染む気がない | 場を回したい |

### 物語における機能

1. **語り部回**: 普段適当な男が「サニーブライアンの皐月賞」を語る回。読者がこの人を初めて知る瞬間。**シリーズで最も刺さる回になりうる**
2. **日常回**: 本人は何もしてない。カズサが場の空気を動かしてる。アキヒコは横で缶コーヒー飲んでるだけ
3. **ギャップ回**: いざという時に黙って動く。終わったらまた適当に戻る。誰も触れない

### 記号（小道具）

- **缶コーヒー**（ゲンと同じBLACKだが、握り方が雑。飲みかけを放置する）
- **スマホ**（オッズだけ見てる。たまにスマホすら見てない）
- **丸めたスポーツ新聞**（几帳面な書き込みはしない。買っただけで読まないこともある）
- **ペンは持たない**。これがゲンやソラとの最大の視覚的差別化

## 2. キャラクターシート — カズサ（10歳♀）

### 基本情報

| 項目 | 内容 |
|------|------|
| **名前** | カズサ（和紗 / 一紗） |
| **年齢** | 10歳（小学4-5年） |
| **立場** | アキヒコの娘。週末は父と二人で過ごすことが多い |

### 性格

- **しっかり者**。父より大人。家でもたぶん父をリードしてる
- **社交的**。誰にでも寄っていける。人見知りしない
- **ソラやマキに自分から懐く**。「ソラちゃん」「マキさん」と自然に呼ぶ
- ゲンさんにも臆さない。ゲンさんの横に座って双眼鏡を覗かせてもらう
- **父のことは好き**。でも「パパ、ちゃんとして」くらいの距離感

### 競馬との関係

- 競馬のことはよくわからない。でも競馬場は好き。広いし、馬が綺麗だし、知り合いのお姉さんたちがいるから
- 父が急に真剣になる瞬間を見ている。理由はわからないけど、その顔が好き
- いずれ「パパ、サニーブライアンってどんな馬？」と聞く日が来る。**その時が物語の転換点になりうる**

### 物語における機能

- **場の接着剤**: カズサがいるからアキヒコが場にいる。カズサが懐くからソラやマキとアキヒコの接点が生まれる
- **読者の窓**: 競馬を知らない読者＝カズサの視点。「なんでみんなあんなに真剣なの？」
- **父の鏡**: カズサがいるからこそ、アキヒコの「いざという時」が映える

### ビジュアルの方向性

- 父の横にいる。基本は父の腰〜肩くらいの背丈
- 服装は年相応にカジュアル。パーカーかTシャツ+スカートorショートパンツ
- **表情は豊か**。父との温度差が画の核
- 髪は肩くらい。ポニーテールか、おろしたまま
