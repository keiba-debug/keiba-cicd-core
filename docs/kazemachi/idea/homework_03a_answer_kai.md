【カイ】（モデル名: Claude Opus 4.6）

# kazemachi キャラクター・オーディション素材集

> 構成意図：「根拠の違い＝人格の違い」を最大化するために、予想スタイルだけでなく「人生のどこで競馬と交差したか」から逆算して設計した。対立軸を複数仕込んであるので、誰と誰を組み合わせても摩擦が生まれるようにしてある。

---

## 1. マキ（改造版）

### 表の設計
- **名前**：マキ
- **年齢・性別**：36歳・女性
- **競馬との関わり方**：「場」を回す人間。自分の予想は出すが、それより他人の予想を引き出すことに快感がある。パドック派。馬体を見るが、本当に見ているのは「周りの人間の目の動き」
- **一言で言うと**：場の空気を読んで回す元キャバ嬢の仕切り屋
- **口調**：テンポが速い。語尾が断定的。「〜でしょ」「〜じゃん」が多い。でもたまに急にゆっくりになる（それは本気のとき）
- **記号**：ペットボトルのカフェオレ（常に持ってる。冬でも冷たいやつ）

### 裏の設計
- **職業**：元キャバクラ勤務→現在は都内の小さいイベントスペースの運営スタッフ。雇われ店長に近い立場
- **生活の時間割**：平日は昼過ぎから出勤、夜22時まで。朝はゆっくり起きてスマホで馬柱を眺める。木曜に枠順が出ると目が覚める
- **競馬を始めたきっかけ**：キャバ時代の太客に連れられて中山競馬場に行った。馬券は全部外れた。でも「自分で選んで自分で外れる」という行為に、接客業にはない自由を感じた。続けている理由は「自分のカネで自分の判断を試す場所」が他にないから
- **一番記憶に残っているレース**：2018年エリザベス女王杯（リスグラシュー）。「勝つ馬って、パドックで一番地味なことがある」と知った日。それ以来パドックを見るようになった
- **人に言えない習慣**：当たった馬券の写真を撮って、LINEの自分だけのグループに送っている。誰にも見せない。外れた馬券は撮らない
- **kazemachiに来る理由**：「ここは私が回さなくていい場所」。仕事では常に場を仕切る側。ここでは仕切り役をやりつつも、自分も予想を出す側に回れる。その両方ができる場所が他にない
- **面白くなる組み合わせ**：ソラとの関係。マキはソラの直感を「面白い」と思って引き出そうとするが、ソラはそれを「利用されてる」と感じる瞬間がある。この微妙なズレが一番ドラマになる

## 2. ナルミ（改造版）

### 表の設計
- **名前**：ナルミ
- **年齢・性別**：31歳・男性
- **競馬との関わり方**：データ。だが「データで勝てる」とは思っていない。「データで負けの理由を説明できる」ことに価値を置いている。スピード指数と血統の掛け合わせが主武器
- **一言で言うと**：正しさに誠実すぎて面白くなってしまう男
- **口調**：丁寧語ベース。「〜ですね」「〜かと」。感情が乗ると急に早口になり、語尾が崩れる（「いや待ってそれはおかしい」）
- **記号**：ノートPC（常にスプレッドシートが開いている。たまに関数の説明を始めて周りが引く）

### 裏の設計
- **職業**：SIerの社内SE。業務系システムの保守運用。楽ではないが暇な時間もある
- **生活の時間割**：9:00-18:00勤務、残業少なめ。帰宅後20:00-23:00が競馬データの時間。金曜夜は「予想の仕上げ日」として酒を飲まない
- **競馬を始めたきっかけ**：大学の研究室（統計学）で「公営ギャンブルの期待値」をテーマにした先輩がいた。データを見せてもらったとき「これはゲームとして解ける構造がある」と思った。続けている理由は「まだ解けていない」から
- **一番記憶に残っているレース**：2023年天皇賞・秋（イクイノックス）。「全てのデータが1頭を指している」レースを初めて経験した。買ったが、あまりに確信がありすぎて逆に楽しくなかった。「データが完全に正しいと楽しくない」と気づいた重要な日
- **人に言えない習慣**：毎週の回収率をExcelに記録しているが、年間トータルがマイナスの年のシートには色をつけている。その色が好きな色（青）なのが自分でも気持ち悪い
- **kazemachiに来る理由**：データの外側にある判断を「浴びたい」。自分は数字から出られない人間だと知っている。でもソラやゲンさんの判断には数字にない何かがあって、それが悔しいし、面白い
- **面白くなる組み合わせ**：後述のクドウ（氷河期の男）との関係。ナルミの「正しさ」とクドウの「正しさなんて意味がなかった人生」の衝突が一番エグい対立になる

## 3. ゲンさん（改造版）

### 表の設計
- **名前**：ゲンさん
- **年齢・性別**：63歳・男性
- **競馬との関わり方**：パドック＋厩舎情報。40年以上見てきた目と、友人ルートの調教情報が武器。でも一番信じているのは「その馬が今日走りたそうかどうか」
- **一言で言うと**：黙って見てる男が一言だけ言うと、全員が黙る
- **口調**：基本無口。「うん」「まあ」「どうかね」。でも喋り出すと止まらないときがある（酒が入ったとき、または自分の予想が大きく外れたとき）
- **記号**：折りたたみの新聞（競馬新聞ではなく一般紙。競馬面だけ折ってある）

### 裏の設計
- **職業**：元印刷工場の工場長。60歳で定年退職、今は週3でマンションの管理人をやっている
- **生活の時間割**：朝5時起床。管理人の仕事がない日は午前中にBSの再放送か録画した地方競馬を見る。昼飯のあとに散歩。競馬新聞は買わなくなった（「全部頭に入ってる」）
- **競馬を始めたきっかけ**：父親に連れられて中山競馬場に行ったのが5歳。それ以来、途切れたことがない。続けている理由は「やめる理由がない」。呼吸みたいなもの
- **一番記憶に残っているレース**：1998年毎日王冠（サイレンススズカ vs エルコンドルパサー vs グラスワンダー）。「3頭とも本物で、3頭とも違う走りをした」。あの日にレースの見方が完成したと思っている
- **人に言えない習慣**：妻に「今日は競馬見てない」と嘘をつく日がある。見ている。スマホで見ている。妻も気づいている。お互い気づいていることに気づいている
- **kazemachiに来る理由**：若い連中の話を聞くのが面白い。自分と全然違う見方をしている。それが「間違っている」とは思わない。ただ、自分が知っていることを伝えたい場面がたまにある。でもお節介はしたくない。その距離感が心地いい
- **面白くなる組み合わせ**：ソラとの関係が一番豊か。ゲンさんはソラの直感に「昔の自分を見る」が、それを絶対に口にしない。ソラもゲンさんの沈黙の意味を正しく受け取っているかは分からない

## 4. クドウ（新キャラ／ヒント1ベース）

### 表の設計
- **名前**：クドウ
- **年齢・性別**：49歳・男性
- **競馬との関わり方**：単勝一本。買うのは1レース1頭だけ。複勝も買わない。「勝つか負けるか」以外の結果に興味がない
- **一言で言うと**：人生で何度も2着だった男が、競馬で1着だけを買い続けている
- **口調**：ぼそぼそ喋る。主語がない。「……まあ、そういうことだよね」「別にいいけど」。同意を求めない。でも人の話はちゃんと聞いている
- **記号**：缶コーヒー（BLACKのみ。飲み終わった缶を握りつぶす癖がある）

### 裏の設計
- **職業**：2000年卒。大学（文学部）→就職浪人→契約社員（物流）→正社員登用→リーマンショックで部署ごと消滅→派遣→現在は倉庫管理の正社員（3社目）。年収は大学の同期の半分以下だと思うが、同期と会わないから知らない
- **生活の時間割**：朝6時出勤、15時退勤。帰宅後はアパートで競馬と酒。金曜の夜に翌日の出走表を見て、土曜の朝に1レースだけ決める。日曜も同じ。それ以外の日は特に何もしていない
- **競馬を始めたきっかけ**：就職浪人中に金がなくて、「100円で2時間楽しめるのは競馬だけだ」と気づいた。それ以来。続けている理由は「他に自分が真剣になれるものがない」。これを悲しいと思ったことはない。事実だから
- **一番記憶に残っているレース**：2006年天皇賞・春（ディープインパクト）。府中のスタンドで見た。ゴールの瞬間、隣にいた知らないおっさんと握手した。あのおっさんの顔を今でも覚えている。競馬場で人と繋がった唯一の記憶
- **人に言えない習慣**：毎年1月に「今年の目標」をスマホのメモに書く。でも年末に見返したことは一度もない。書くところまでが儀式
- **kazemachiに来る理由**：「予想に正しい理由がある」世界が見たい。自分は理由なんかなくても買う。でも、ナルミみたいに理由を積み上げる人間がいて、ソラみたいに理由がないのに当たる人間がいて、その両方が同じ場所にいるのが不思議で、居心地がいい
- **面白くなる組み合わせ**：ナルミとの対立が最大の構造。「正しさを武器にしてきた若い男」と「正しくやっても報われなかった男」のぶつかり合い。ナルミが正論を言うたびにクドウの沈黙が重くなる。でも二人とも嫌い合っているわけではない。そこがいい

## 5. フジさん（改造版）

### 表の設計
- **名前**：フジさん
- **年齢・性別**：50歳・女性
- **競馬との関わり方**：記録係。全レースの結果を自分のノートに手書きしている。馬券は買うが金額は一定（全レース200円）。「儲ける」ではなく「記録する」が動機
- **一言で言うと**：競馬場の図書館司書
- **口調**：穏やか。「〜なのよ」「〜だったかしら」。でも記録に関する間違いを指摘されると急にトーンが変わる（「それは違います」）
- **記号**：B5のノート（10年以上続いているシリーズ。背表紙に年度が書いてある）

### 裏の設計
- **職業**：区立図書館の非常勤司書。週4勤務
- **生活の時間割**：朝9時から17時まで図書館。帰宅後にその週のレース結果をノートに転記する。土日は可能な限り競馬場かWINSに行く。「現地で書く」ことにこだわりがある
- **競馬を始めたきっかけ**：25歳のとき、当時付き合っていた人に連れられて東京競馬場に行った。その人とは別れたが、競馬は残った。記録を始めたのは「別れた人との思い出を上書きしたかった」から。今はもうそんな動機は忘れていて、「記録すること自体」が生きがいになっている
- **一番記憶に残っているレース**：2003年中山グランドジャンプ（カラジ）。障害レースなんて興味がなかったのに、あの馬の飛越を見て鳥肌が立った。「自分の好き嫌いの外側にも競馬がある」と知った日。※推測
- **人に言えない習慣**：ノートの余白に、馬の名前で短い詩を書いている。誰にも見せたことがない
- **kazemachiに来る理由**：「誰かの記憶を残す場所」が欲しかった。自分のノートは自分しか見ない。でもkazemachiに書けば、誰かの予想の根拠が「記録」として残る。図書館司書の本能がそれを求めている
- **面白くなる組み合わせ**：マキとの関係。マキは「今この瞬間」の人、フジさんは「過去の蓄積」の人。この二人が同じレースを見ると、見ている時間軸が違う。それを互いに「面白い」と思えるかどうかが、この世界の温度を決める

- **面白くなる組み合わせ**：ゲンさんとの関係が構造的に一番おいしい。40年の蓄積を持つ沈黙の男と、3年の経験をXで叫ぶ男。ゲンさんがミナトを「うるさい」と思っているかと思いきや、たまにミナトの投稿を読んでいる。それをミナトは知らない

## 7. サエ（新キャラ）

### 表の設計
- **名前**：サエ
- **年齢・性別**：28歳・女性
- **競馬との関わり方**：血統派。でも「この配合が走る」ではなく「この血が走りたがっている」という言い方をする。科学的ではない血統論
- **一言で言うと**：競馬の話をしているのに、いつの間にか人間の話になっている人
- **口調**：落ち着いている。「〜だと思う」「〜な気がする」。断定しない。でも予想は意外と変えない（「気がする」のままブレない）
- **記号**：文庫本（競馬場にいつも1冊持ってくる。読んでいるところを見たことがない。お守り）

### 裏の設計
- **職業**：介護施設の生活相談員。大学は心理学科
- **生活の時間割**：8:30-17:30勤務。帰宅後は疲れてソファで寝落ちすることが多い。金曜の夜だけ血統表をPCで見る。「金曜の夜が一週間で一番好きな時間」
- **競馬を始めたきっかけ**：父親が馬主（零細の一口馬主）だった。幼少期から「あの馬の子がデビューする」「あの牝系は気性が荒い」という話を食卓で聞いていた。父親が亡くなった後、血統表を見ることが父との対話の延長になった。続けている理由は「血が繋がっていく」ことに、何かの希望を見ているから
- **一番記憶に残っているレース**：2019年桜花賞（グランアレグリア）。父が生前「ディープ×ダンスの牝馬はいつか大物が出る」と言っていた。それがグランアレグリアだと思った。根拠はない。でも泣いた
- **人に言えない習慣**：出走馬の母馬の名前を全レース分調べている。「母の名前が好きかどうか」で迷ったときの最終判断を下すことがある。これは絶対に人に言えない
- **kazemachiに来る理由**：ナルミのデータとソラの直感の間に「もう一つの根拠」を置きたい。血統は数字でも直感でもない。物語だと思っている。その居場所がkazemachiにはある気がする
- **面白くなる組み合わせ**：ソラとの関係が化学反応を起こす。二人とも「理由がないのに確信がある」タイプだが、根っこが違う。ソラは身体的（見た感覚）、サエは物語的（血が語る）。この二人が同じ馬を推したとき、説得力が倍になる。逆に割れたとき、どちらの「理由のなさ」が正しいかが見どころになる

- **記号**：ホワイトボードマーカー（どこにでもレースの展開図を書き始める。紙ナプキンの裏、スマホの画面、最悪テーブル）

- **職業**：広告代理店のプランナー。中堅。企画は通るが出世しないタイプ。※推測

- **競馬を始めたきっかけ**：新卒で入った会社の先輩に「お前は展開力があるから競馬向いてる」と言われた。最初は意味が分からなかったが、ラップタイムを初めて見たときに「これは物語の拍子だ」と直感した。続けている理由は「レースの脚本を読む」のが仕事のストーリーボードを作るのと同じ脳の使い方だから
- **一番記憶に残っているレース**：2020年ジャパンカップ（アーモンドアイ引退レース）。「完璧な脚本」だと思った。でもそれは「完璧すぎて予想の面白さがない」ということでもあった。競馬が面白いのは脚本が破れるときだと知った日
- **人に言えない習慣**：外れた予想のメモを捨てられない。段ボール箱に入れてクローゼットの奥にある。「いつか外れた理由を全部分析する」と言い続けて5年経った
- **kazemachiに来る理由**：「自分の展開予想を人に聞いてほしい」。それだけ。仕事では企画を通すために人の顔色を見るが、ここでは「俺はこう読んだ」を言い切っていい。それが気持ちいい
- **面白くなる組み合わせ**：マキとの関係。二人とも「場を仕切りたい」タイプで、テツは予想の主導権を取りたいし、マキは会話の主導権を取りたい。この二人が同じ空間にいると権力闘争が起きる。でもそれは仲が悪いのではなく、「どっちが今日の主役か」を毎回やっている

## 対立構造マップ（構成メモ）

最後に、カイとしてこの12人の対立軸を整理しておく。

```
【根拠の対立】
  ナルミ（データ）─── ソラ（直感）─── サエ（血統/物語）
      │                                        │
  「正しさ」の軸                          「信じる」の軸

【態度の対立】
  ダイチ（金のために買う）──── リコ（馬のために見る）
  ミナト（コンテンツとして消費）── ゲンさん（呼吸として生きる）
  テツ（予想を語りたい）──── クドウ（黙って買う）

【時間の対立】
  フジさん（過去の記録）──── マキ（今この瞬間）──── ミナト（バズる未来）

【距離の対立】
  ユカリ（やめた人）──── ダイチ（やめられない人）
  リコ（買わない人）──── クドウ（買うしかない人）

【世代の対立】
  ゲンさん(63) ─ コマツ(55) ─ クドウ(49) ─ ダイチ(45) ─ テツ(42) ─ ユカリ(38)
                    ─ マキ(36) ─ ナルミ(31) ─ サエ(28) ─ ソラ(26) ─ ミナト(23) ─ リコ(21)
```

### 構成上の推薦：メインキャスト候補

カイの意見として、オーディションに先立って「この7人なら回る」という仮組みを置いておく。

| 役割 | キャラ | 理由 |
|------|--------|------|
| 進行・引き出し役 | **マキ** | 場を回せる唯一の人間。抜けると構造が崩壊する |
| 核・引力 | **ソラ** | 固定枠。異論なし |
| 対立軸A（論理） | **ナルミ** | ソラとの対比が世界の基本構造 |
| 対立軸B（沈黙） | **クドウ** | ナルミの論理を「人生で」否定できる唯一のキャラ。氷河期世代の重力 |
| 深度・記録 | **フジさん** | 時間軸を管理できる。kazemachiに歴史を与える人間 |
| 刺激・速度 | **テツ** | テンポメーカー。テツがいないと静かになりすぎる |
| 窓・外の視線 | **リコ or ユカリ** | 「買わない人間」が1人いることで、全体の意味が変わる。リコなら明るくなる、ユカリなら深くなる |

ダイチ、コマツ、ミナト、サエはゲストやサブキャラのストックとして最高に機能する。特にダイチは「年に2〜3回登場して空気を壊す」使い方が一番効く。
