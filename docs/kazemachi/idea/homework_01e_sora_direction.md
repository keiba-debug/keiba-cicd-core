# 宿題①e — ソラのビジュアル方向性相談

> 各作家のシステムプロンプトの後ろに貼って投げる。
> ソラの生成画像を全パターン添付（初期版〜最新版）+ MC画像も参考として添付

---

```
━━━━━━━━━━━━━━━━━━━━━
■ ソラのビジュアルで詰まっている
━━━━━━━━━━━━━━━━━━━━━

ソラ（直感派）の画像を何パターンも生成したが、うまくハマらない。
添付画像を全部見てほしい。MC（進行役）の画像も参考として添付する。

【試した変遷】
1. 最初：茶髪ミディアム＋レーシングジャケット＋スマホ → 元気すぎ、少年漫画寄り、テイスト不一致
2. 目の修正：成瀬あかり的な強い意志の目を指定 → 良くなったがまだ「かわいい女の子」の枠内
3. 大胆変更：黒髪ショート＋腕組み＋スマホなし → かっこよくなったが画風まで変わってしまった

【現状の問題】
- MCとテイストが合わない（MCは漫画寄りスケッチ線、ソラだけ別タッチになりがち）
- 「かわいい」を抜こうとすると「かっこいい別作品のキャラ」になる
- もうひと押しのアイデアが欲しい

【守りたいもの】
- MCと同じ世界の住人に見えること（テイスト統一が最優先）
- 直感派の「自分の目を信じてる」感
- ナルミ（データ派・静・寒色）と並べた時に対照的であること
- 26歳の若さと脆さ（ユキの指摘：30代にしない）

【まだ試していないこと】
- 髪型のバリエーション（ポニーテール？ 無造作お団子？ ベリーショート？）
- 服装の方向性（ジャケット以外の選択肢？）
- ポーズ（腕組み以外で「かっこいい」を出す方法？）
- 表情の方向（笑顔でも泣き顔でもない、第三の選択肢？）
- そもそもソラの「記号」は何であるべきか？
  （ナルミ=PC/眼鏡、ゲンさん=手書きノート、MC=クリップボード、テツ=古い馬券）

━━━━━━━━━━━━━━━━━━━━━
■ 聞きたいこと
━━━━━━━━━━━━━━━━━━━━━

添付画像を全部見た上で、ソラのビジュアルについて「もうひと押し」のアイデアをくれ。

具体的には：
1. 今までの試行の中で「これが一番近い」と思うものはどれ？ その理由
2. 「ここをこう変えればハマる」という具体的な修正案
3. 画像生成AIに投げるプロンプト案（あれば）

あなたの専門視点で：
- カイ：ナルミと並べた時に対立が映えるビジュアルは？ Xのタイムラインで止まる絵は？
- ユキ：ソラの感情（自信と脆さの同居）が伝わる表情・ポーズは？ 「この子好きだな」と思わせる一要素は？
- レイ：ソラの画像に仕掛けを仕込む余地を残すには？ ジャケットの背中のロゴなど、小道具でどう仕込む？

自由に書いてくれ。短くてもいい。「この一言」があれば変わる、みたいなのが欲しい。
```
