# ギャップコメディ路線 — キャラの本性と建前

> 真面目な予想番組なのに、馬券になると人格が壊れる大人たち

---

## コアコンセプト

予想パートでは知的な議論をしてるのに、
レースが始まると（or 馬券の話になると）**全員壊れる**。

「わかってるのにやめられない」が競馬の本質。
それをキャラに入れると、競馬知らない人にも「バカだなぁw」って愛される。

---

## ギャップパターン集

### データ派
| 予想中（建前） | 本性 |
|---------------|------|
| 「確率的にはこの馬一択です」 | 借金まみれ。レース中は「来い来い来いぃぃ！！」 |
| 「期待値がマイナスの馬券は買いません」 | 自分の買い目だけ理論と全然違う |
| 冷静にスプレッドシートを披露 | 「……今回は例外です」が口癖 |
| 他人の予想を論理的に否定 | 自分の馬が沈むと無言で天を仰ぐ |

### 直感派
| 予想中（建前） | 本性 |
|---------------|------|
| 「この馬、空気が違う」 | 当日の朝ごはんの内容で本命を変えたことがある（認めない） |
| 「パドックの目が違った」 | パドック見てない回がある |
| 「俺の感覚を信じろ」 | 隣の人が買ってるの見て急に乗り換える |
| 自信満々に穴馬推奨 | 直前で100円だけ人気馬の複勝を買う保険 |

### 経験派
| 予想中（建前） | 本性 |
|---------------|------|
| 穏やかに「馬は生き物だからね」 | 馬券で家のローン2回組み直してる |
| 「昔同じパターンで来た」 | その「昔」が毎回違うレースを指してる |
| 若い2人に「まあ落ち着け」 | 自分が一番落ち着いてない（手が震えてる） |
| 「今日は見送りでいい」 | 結局買う。毎回買う |

### 進行役
| 番組中（建前） | 本性 |
|---------------|------|
| 冷静に「で、結局どの馬？」 | CM中にスマホで馬券買ってるのがたまに映る |
| 「先週も外したわよね」と他人をイジる | 自分の収支は絶対言わない |
| 予想しないスタンス | ボソッと言った一言がなぜか当たる |
| 淡々と成績発表 | 特定キャラが負けた時だけ微かに笑ってる |

## 場面アイデア

### 「理論が崩壊する瞬間」
- データ派が完璧な分析を披露 → 出走取消 → 「………」 → 進行役「大丈夫？」
- 経験派「この馬は絶対来ない」→ 1着 → 直感派「おい、絶対って言ったよな」

### 「予想と買い目が違う問題」
- 本命発表では3番を推した → 買い目見たら7番の単勝が入ってる
- 「保険だよ保険」「それ本命より厚いじゃん」

### 「全員同じ馬を推した回」
- 珍しく全員一致 → 進行役「全員一致って初めてね」
- 全員「…逆に怖い」 → 結局1人が裏切る

### 「借金エピソード」（チラ見せ）
- データ派が計算中に電卓叩く → 「…あ、これ別の計算だった」（何の？）
- 経験派がたまに「今月はもう…」と言いかけてやめる
- 進行役が「あなたたち、そこまでして買うの？」→ 全員目を逸らす

### 「レース実況中の豹変」
- 普段冷静なデータ派「うおおおお差せ差せ差せ！！！」
- 経験派の声が裏返る「きたきたきたきたきた！！」
- 直感派だけなぜか静か（大穴を1人で握ってるから）

---

## なぜこの路線が効くか

- **共感**: 馬券買う人は全員こう。「わかるわかるw」
- **親しみ**: 完璧な予想家より「ダメな大人」の方が好かれる
- **ラジオ向き**: 絶叫やリアクションは音声映えする
- **リピート性**: 「今週はデータ派がまた壊れるかな」で毎週見たくなる
- **競馬知らない人にも刺さる**: 予想の中身がわからなくても人間ドラマとして面白い
