# 宿題①c — キャラクタービジュアル プロンプト募集

> 各作家のシステムプロンプトの後ろに貼って投げる。
> 画像2枚（進行役ベース確定の「半笑い1」＋ 別キャラ候補の「鉛筆困り顔」）を添付する。

---

```
━━━━━━━━━━━━━━━━━━━━━
■ ブラッシュアップ回答、3人ともありがとう
━━━━━━━━━━━━━━━━━━━━━

全員の回答を読んで突き合わせた。めちゃくちゃ良かった。
まず現状の合意事項を共有する。

━━━━━━━━━━━━━━━━━━━━━
■ 3人の意見が一致したこと
━━━━━━━━━━━━━━━━━━━━━

【進行役のベース画像が決まった】
添付画像1枚目の「書類まみれで半笑いっぽい子」を進行役のベースにする。
- カイ：「壊れた業務感が進行役にかなり合ってる」
- ユキ：「何考えてるんだろう、が先に来る唯一の絵」
- レイ：画像仕込みの観点から「線の密度が程よく情報を埋められる」

3人とも「可愛すぎるのはダメ」「有能すぎるのもダメ」で一致。
「平凡・頼りないのに、たまに精度がおかしい」を絵で表現する方向で確定。

【Phase 1は小さく始める】
- カイ：「初期は2予想家+進行役の3アカ。世界を広げすぎるな」
- ユキ：「キャラを先に好きにさせる。予想は人格の延長線上」
- レイ：「伏線は進行役中心に絞る」

全員「最初から全キャラ出すな」で一致。

【「看板ラリー」を先に作る】
- カイ：「番組といえばこれ、という往復を3本作れ」
- ユキ：「第1投は予想じゃなく自己紹介にしろ」
- レイ：「X列の匂わせを最初に仕込め」

コンテンツの核は「会話の往復」で一致。

━━━━━━━━━━━━━━━━━━━━━
■ 3人の意見が割れたこと
━━━━━━━━━━━━━━━━━━━━━

【スタートペア問題】
- カイ：「データ派 × 直感派。対立理由が一瞬で分かる。初速向き」
- ユキ：「データ派 × 進行役。ナルミを先に好きにさせてからソラ合流のインパクトを最大化」
→ これはまだ決めない。キャラのビジュアルが見えてから判断したほうがいい

【ARG・伏線の深さ】
- レイ：「画像のメタデータ、HTMLソースのコメント、投稿時間の規則性」
- ユキ：「ARGはキャラの行動から自然に生まれるものに限定しろ。伏線のためにキャラを動かすな」
→ ユキの原則を採用。伏線はキャラの行動の結果として生まれるもの

【鉛筆困り顔の子の扱い】
添付画像2枚目の「鉛筆を髪に差した困り顔の子」。
- カイ：「進行役の候補としてバランスが良い」
- ユキ：「新人スタッフ感が一番強い。可愛い寄りでマキの怖さが出にくい」
- プロデューサー案：「進行役ではなく別キャラに回す。
  例えば誰かの娘で実は競馬にめちゃくちゃ詳しい子。
  一生懸命だけどどこか抜けてる見た目で、質問が鋭すぎるギャップ」
→ この子の役割も、ビジュアルを並べてから考えたい

━━━━━━━━━━━━━━━━━━━━━
■ ここからが本題 — キャラ画像プロンプト募集
━━━━━━━━━━━━━━━━━━━━━

ビジュアルがあると話が一気に進む。
これから画像生成AIで各キャラのイメージを作りたい。

そこで、あなたの視点から
**「このキャラをこういうイメージで描いてほしい」**
というプロンプト案を出してほしい。

【ルール】
- 画像生成AIに投げるプロンプト形式で書く（英語でも日本語でもOK）
- 1キャラにつき1プロンプトでいい（差分は後で作る）
- 「このキャラはまずこの1枚で人格が伝わるべき」という決定的な1枚を狙う
- テイストは添付画像の「半笑い1」に合わせる
  （日常アニメ調、漫画寄りのラフな線、低彩度、日常感）
- 全キャラ出す必要はない。「まずこのキャラが見たい」という優先順位をつけてくれ

【対象キャラ候補】
- データ派（ナルミ / レン / ケイ）
- 直感派（ソラ / ミオ / ミク）
- 経験派（ゲンさん）
- 進行役の表情差分（半笑い1ベースで）
- ロマン派
- 鉛筆困り顔の子（新キャラとして活かす場合）
- その他、あなたが「このキャラが要る」と思う存在

【プロンプトに含めてほしい要素】
1. キャラの役割と性格が伝わる「状況」（何をしている場面か）
2. 服装・小道具・姿勢で性格を表す「記号」
3. 表情の方向性（建前の顔？本性が漏れた顔？）
4. 他キャラとの差別化ポイント（隣に並べた時に一目で区別できるか）
5. テイストの指定（半笑い1に合わせるための具体的な指示）

【あなたの視点ならではの観点も入れて】
- カイなら：「この見た目だと対立構造が映える」「ビジュアルでテンポが出る」
- ユキなら：「この表情だと感情が伝わる」「このポーズで関係性が見える」
- レイなら：「この小道具に仕掛けを埋められる」「この構図で裏が匂う」

出力形式は自由。何キャラ分出してもいい。
優先順位だけはつけてくれ。「まずこの1人」があると助かる。
```
