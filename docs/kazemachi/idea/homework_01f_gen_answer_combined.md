# 宿題①f — ゲンさんビジュアルレビュー（3人統合回答）

---

## 全体評価：v2は「方向」は正解。ただし振れ幅が大きすぎる

v1の不評ポイント（好々爺、AI感、色温度、英語ノート）は**ほぼ解消された**。
しかし新しい問題が発生している。

## 【ユキ】口と目の温度差は実現できたか？

### 答え：アクション版では出た。ベース版ではまだ。

**ノート書き込み版（NOTEBOOK-1）が一番近い。** ノートを見下ろしながらの表情に、笑みと集中が同居している。口は笑っているが、目がノートの何かに引っかかっている。これがユキの言う「口と目の温度差」に最も近い。

**ペン書き版（WARMTH-1）も悪くない。** 前のめりでペンを持って書いている。笑顔だが、目が「書いているもの」に集中していて笑っていない。日常のゲンさんとして自然。

**問題：アクション版（B/C variant）は振れすぎた。**

「この展開…前にも見た」のテキスト付きの3枚（身を乗り出し版、立ち上がり版2枚）は**少年漫画のボスキャラ**になっている。目を見開いて口を全開にして叫んでいる——これはゲンさんの「本性」かもしれないが、**怖すぎる**。

ユキが求めていたのは「穏やかさの裏に鋭さがチラッと見える」であって「突然ヤクザの親分になる」ではない。ギャップが大きすぎるとキャラが分裂する。

### ユキの修正案

**ベース画像の表情指示を修正する：**

- 口：閉じている。唇の端だけが少し上がっている。笑顔ではなく「笑みの痕跡」
- 目：開いているが見開いていない。まっすぐ前を見ている。瞳が少し奥にある感じ（焦点が遠い）
- 全体：**「何か思い出しかけているけど、まだ言わない」の顔**。話す0.5秒前ではなく、思い出す3秒前。もっと静か

**「この人の話を聞きたい」と思わせる一要素：**

ゲンさんが**黙っている**こと。今の画像は全部、笑っているか叫んでいる。ゲンさんの一番魅力的な瞬間は**黙って何かを見ている時間**。その沈黙が長いほど、口を開いた時のインパクトが大きい。

ベース画像のゲンさんは**黙っているべき**。

## 【カイ】5人並べた時のバランスは？

### 色温度：解消された

グレーグリーンのワークシャツに変わったことで、MC（水色）、ソラ（くすんだ青灰）と同じ色世界に入った。v1のフランネルと比べると格段にいい。**この色は確定で。**

### テンポ：アクション版は使えるが、ベース版が足りない

```
現状の5人のテンポ（v2ベース版を入れた場合）：

[ナルミ♂30] [ソラ♀26] [ゲンさん♂60] [MC♀34] [フジさん♀48]
  冷          脆          ???          無        圧
```

ゲンさんの場所に入るべきテンポは「渋」。静かだけど重い。今のv2ベース版は笑いすぎていて「陽」に見える。黙っている顔のゲンさんなら「渋」が成立する。

### 「0.5秒前」は出ているか？

**B variant（身を乗り出し版）は0.5秒前ではなく「0秒」。** もう喋り始めている。0.5秒前はもっと静か。「あ、なんか引っかかった」の顔。まだ言葉になっていない。

カイの理想：**ゲンさんがペンを止めた瞬間。** 書いていた手が止まる。顔は上げない。でも目だけが動く——ノートから、正面の何かに視線が移る。「あれ？」と心の中で言っている。まだ口は開かない。

この「ペンが止まった瞬間」がベース画像であるべき。

### アクション版の使い道

B/C variant（身を乗り出し・立ち上がり）は**レース中の実況投稿専用**として使える。ベースとのギャップが大きいほうがタイムライン上のインパクトになる。ただし**イヤリングは削除**（画像3, 4に出ている。キャラ設計にない）。

## 【レイ】ノートの赤丸は仕掛けとして機能するか？

### NOTEBOOK-1版：ほぼ完璧

ノートの中身が見える画像（NOTEBOOK-1）は仕掛けの観点から非常にいい：
- 日本語の殴り書き ✅
- コース図（楕円） ✅
- 数字が散在 ✅
- 赤丸の固有名詞 ✅
- スマホ裏向き ✅

**赤丸が画面上で唯一の赤になっている。** これは設計通り。読者の目が自然にそこに行く。

### 課題：ベース版でもノートがチラッと見えるべき

アクション版のノートは詳細に見えるが、ベース版（WARMTH-1）ではノートが白紙に見える。**ベース画像でもノートの端に殴り書きと赤丸が少しだけ見切れる**ように設計すべき。

毎回の投稿で赤丸が「何気なく」映っていることが、考察班が気づく導線になる。特別版でだけ見えるのでは「仕掛けに気づくチャンス」が少なすぎる。

### アクション版の仕掛け

「この展開…前にも見た」のテキストは仕掛けではなく演出。これは使っていい。ただし**テキストの内容を毎回少しだけ変える**ことで、考察班が「前にも見たのは何のことか？」を追い始める。

## 3人の結論：共通の修正方向

### v2で確定するもの

| 要素 | 確定 |
|---|---|
| 服装 | グレーグリーンのワークシャツ |
| 体格 | ゴツい、肩幅広い、腕が太い |
| 小道具 | 手書きノート＋赤丸、スマホ裏向き、眼鏡おでこ |
| アクション版 | B/C variantの方向性（使用頻度は低い） |
| テイスト | MC・ソラに近づいた |

### v3で変えるべきもの

| 問題 | 解決策 |
|---|---|
| ベース版が笑いすぎ | **口を閉じる。笑みの痕跡だけ残す** |
| アクション版が怖すぎ | ベースとの差で十分。これ以上強くしない |
| 「0.5秒前」が出ない | **ペンが止まった瞬間。視線だけ動く** |
| 目が見開きすぎ or 閉じすぎ | **普通に開いている目の中に鋭さを入れる** |
| イヤリングが出る | ネガティブプロンプトで排除 |
| ノートがベース版で見えない | ベースでもノートの端に内容を見せる |
