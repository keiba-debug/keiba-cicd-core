━━━━━━━━━━━━━━━━━━━━━
■ ゲストキャラクターという構造
━━━━━━━━━━━━━━━━━━━━━

今の設計はコア6人が固定されている。
それ自体は正しい。中心が安定しているから読者が安心できる。

でも、6人だけで回し続けると：
- 反応パターンが固定化する（「ナルミはこう言うだろう」が読めてしまう）
- 新しい面が引き出されにくい
- ネタが枯れるリスク
- 読者が自分を重ねる入口が少ない

そこで提案したいのが、
**コア6人は「場」として固定し、そこにゲストキャラが来る構造**。

━━━━━━━━━━━━━━━━━━━━━
■ リファレンス：「路傍のフジイ」
━━━━━━━━━━━━━━━━━━━━━

鹿子木灯の漫画「路傍のフジイ」を知っているかもしれない。

フジイは中年の独身男性で、毎日を淡々と生きている。
物語の中心にいるのはフジイだが、フジイ自身は大きく動かない。
周りの人たち——同僚、隣人、すれ違う人——がそれぞれの人生を持っていて、
フジイとの接点で、その人の本質が一瞬だけ見える。

この構造の強みは：
- 主人公（フジイ）は安定した固定点
- 物語の広がりはゲストが担う
- 毎回新鮮だが、世界観はブレない
- ゲストは1回限りでも、濃い印象を残せる
- 主人公の新しい面が、ゲストとの関わりで引き出される

━━━━━━━━━━━━━━━━━━━━━
■ この企画に当てはめると
━━━━━━━━━━━━━━━━━━━━━

【コア6人 = 「場」】
ナルミ、ソラ、ゲンさん、テツ、フジさん、マキ
→ この人たちは毎週いる。読者のホームになる。

【ゲストキャラ = 「通り過ぎていく人たち」】
例：
- 初めて競馬場に来た大学生（マキが案内する）
- ゲンさんの昔の飲み仲間（30年ぶりに再会）
- ナルミのデータを批判しに来るTwitterの論客
- ソラと同じ馬を推していた見知らぬおばさん
- フジさんの前の職場の後輩（「フジさんって昔は…」）
- テツが競馬場で毎週見かけるけど話したことがない男

ゲストが来ることで：
- コア6人の「普段出ない面」が引き出される
- 読者が自分を重ねやすい（ゲスト＝読者の分身）
- コンテンツが無限に広がる
- 「この場に人が集まってくる」というマキの主催者としての価値が可視化される
- レイの仕掛け：ゲストの何気ない一言が「15年前のあの日」に関係している可能性

━━━━━━━━━━━━━━━━━━━━━
■ 聞きたいこと
━━━━━━━━━━━━━━━━━━━━━

1. このゲストキャラ構造についてどう思うか？
   コア6人だけで回すのと比べて、メリット・デメリットは？

2. ゲストキャラの登場頻度はどのくらいが適切か？
   毎週？ 月1？ シーズンに1人？

3. あなたの専門領域でゲストキャラはどう活きるか？
   - カイ：構成やテンポにどう影響する？
   - ユキ：関係性の変化にどう作用する？ ゲストが来ることでコア6人の何が見える？
   - レイ：仕掛けや伏線にどう使える？ ゲストの一言が謎を動かせるか？

4. 「路傍のフジイ」的な構造で、
   この企画の「フジイ」に当たるのは誰か？
   マキ？ フジさん？ それとも「場」そのもの？

5. ゲストキャラの具体例を3人以上出してほしい。
   その人が来ることで、コア6人の何が変わるか？

━━━━━━━━━━━━━━━━━━━━━
■ もう一つ聞きたいこと：ソラの引力
━━━━━━━━━━━━━━━━━━━━━

正直に言うと、番組という要素が薄くなった今、
プロデューサーとして一番惹かれているキャラクターはソラ。

理由：
- 脆さがある。強いキャラより揺れてるキャラの方が物語を引っ張る
- 毎週「根拠のない直感」に身を晒している。それ自体がドラマ
- 外れた時が一番美しい
- ソラの直感に周りがどう反応するかで、全員の人格が見える（鏡になれる）
- 動画にした時の表情の力が圧倒的（「この子の隣に座りたい」と思わせる）

「路傍のフジイ」で言うと、ソラはフジイではない。
でもソラは「見ていたい人」であり、「この子がいるから周りの人も見えてくる」存在かもしれない。

聞きたいこと：
- ソラを感情の中心に据えた場合、構造はどう変わるか？
- マキが入口、ソラが感情の中心、フジさんが謎の中心、という三層は成立するか？
- ソラが「見ていたい存在」であるために、何を守り、何を変えるべきか？

自由に書いてくれ。
