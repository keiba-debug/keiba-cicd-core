# リアル馬券 × レース観戦回

> 予想バトルの結論で実際に馬券を買い、後日レースを一緒に見る

---

## コンセプト

- 予想回で出した各キャラの結論を**少額でも実際に購入**
- 別の回（or 同回の後半）でレース映像を一緒に見る
- キャラの反応がリアル（自分の金がかかってる）

---

## なぜ強いか

### 視聴者側
- 「このキャラの予想で買ったらいくらになった？」が追える
- 推しキャラの成績を応援する構造 → エンゲージメント
- 「予想回→結果回」の2本セットで毎週見たくなる

### コンテンツ側
- リアルマネーが乗ってるからリアクションが本物になる
- 「先週の借金がまた増えた」がギャグじゃなくなるw
- 長期の収支推移がコンテンツ自体になる

### KeibaCICDとの接続
- confirmed_bets / bets.json で購入記録は既に管理できる
- キャラ別の成績トラッキングは仕組みとして作れる
- Webの収支画面をキャラ別に拡張する余地あり

---

## 番組構成（2本セット案）

### 予想回（レース前日公開）
```
通常の予想バトル
  → 最後に各キャラの買い目を確定
  → 進行役「では買いました。◯◯は単勝500円、◯◯はワイド200円…」
  → ED「さて来週、誰の財布が軽くなるでしょうね」
```

### 結果回（レース後公開）
```
[OP] 進行役「結果が出ました」
[レース映像] 一緒に見る（※映像の権利に注意、実況音声+キャラの被せリアクション？）
  → データ派「来い来い…あぁぁぁ」
  → 直感派「ほら見ろ！俺が言った通り！」
  → 経験派「……（無言で財布を見る）」
[成績発表] 今回の収支 + 累計成績
  → 進行役「◯◯さん、今月マイナス3200円ですけど」
  → 「うるさい」
[次回予告] 来週のレースをチラ見せ
```

## 馬券ルール案

### シンプル版
- 各キャラ1レースにつき上限1000円
- 馬券種は自由（単勝・複勝・ワイド・馬連・三連単なんでも）
- キャラの予想スタイルに合った券種を選ぶ
  - データ派: 複勝・ワイドで手堅く
  - 直感派: 単勝一点勝負が多い
  - 経験派: 馬連・三連複で「この組み合わせ」
  - 進行役: 買ってない（はず）

### 成績管理
- 月間収支をランキング化
- 年間チャンピオン制度
- 「今月一番儲かったキャラは？」→ 視聴者が推しを変える動機にも

---

## レース映像の権利について

- JRAの映像をそのまま使うのは要確認（YouTube的にグレー）
- 代替案:
  - 実況音声のみ + キャラの反応アニメ
  - 着順テロップだけ表示してキャラが語る
  - 自作のレース再現アニメ（将来）
- 結果だけテキストで出してリアクションに振る、が安全

---

## 長期的な面白さ

- 「データ派、3ヶ月連続プラス収支」→ 他キャラが焦る
- 「直感派、今年の万馬券的中ゼロだけど回収率は一番高い」→ 意外な展開
- 「経験派、G1だけ異常に強い」→ キャラの個性が数字で証明される
- 年末に「年間成績表彰式」回 → 特別コンテンツ
