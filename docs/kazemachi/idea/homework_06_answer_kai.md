### 【投票結果】メインキャスト推薦（6人）

**1. マキ（カイ版 K-01）— 役割: 入口役 / MC / 場回し**

- 推薦理由: 構成上、MC不在の番組は成立しない。76人見渡して「場を回せる人間」はこの人だけ。元キャバで培った空気の読み方、テンポの速い断定口調、そして「本気のときだけゆっくりになる」緩急。これはプロのMCの呼吸そのもの。ユキ版マキ（Y-01）は編集者で知的だが「引き出す力」が弱い。レイ版マキ（R-01）は美容師で親しみやすいがドライブ力が足りない。K-01だけが「この場の全員に喋らせて、かつ自分も予想を出せる」という二刀流ができる。
- 現状画像評価: 巻き髪の華やかさとこなれ感は合格。6枚あるのも心強い。ただし「元キャバ」の匂いがもう少し残っていていい。完全にこなれすぎると面白くない。少しだけ「場末」を匂わせるアクセサリーか、ネイルの派手さが欲しい。
- ソラとの関係性: マキはソラの直感を「面白い」と思って引き出そうとする。でもソラはそれを「利用されてる」と一瞬感じる。この微妙なズレが構成上の爆弾になる。MC＝善人ではない。MCが中心人物と「ちょっとだけ噛み合わない」のが最高のドラマ。

**2. テツ（カイ版 K-08）— 役割: テンポメーカー / 展開語り / ボケ**

- 推薦理由: 構成にはエンジンが要る。テツは「俺の展開読みを聞け」で暴走し、外れると3時間黙る男。このテンションの上下動が場のリズムを作る。バラエティで言えば"回される側の華"。ユキ版テツ（Y-04）は映像編集者で少し地味。レイ版テツ（R-05）はタクシー運転手で渋いが展開語りの熱量が足りない。K-08のプランナー設定が一番「語りたがり」の解像度が高い。紙ナプキンの裏に展開図を書くという記号も、画として強い。
- 現状画像評価: まだ画像が生成されていない。※推測。作る場合は「ホワイトボードマーカーを持って前のめりに語る42歳」。スーツの上だけ脱いでワイシャツ腕まくり。目が爛々としてる感じ。
- ソラとの関係性: テツは「レースはこう流れる」と脚本を読む。ソラは「この馬が来る」としか言えない。テツの脚本通りにソラの馬が勝ったとき、二人のどちらが正しかったのかが宙に浮く。この「正しさの曖昧さ」がkazemachiの軸と一致する。

**3. クドウ（カイ版 K-04）— 役割: 沈黙のカウンター / 氷河期の重力 / 間の支配者**

- 推薦理由: テンポの速いマキとテツの間に、「黙っている男」を1人置く。構成の鉄則。ずっと喋ってる番組は疲れる。クドウがぼそっと「……別にいいけど」と言った瞬間、場の空気が変わる。それが構成上の「転換点」になる。ユキ版カズヒロ（Y-06）も氷河期だが「感情派」で言葉が多い。シュウはカズヒロを推したが、俺はクドウを推す。単勝一本という買い方のシンプルさが、この男の人生の切実さをそのまま映している。「勝つか負けるか以外に興味がない」。セリフにすると5文字で終わるが、その5文字に49年分の重さがある。
- 現状画像評価: 49歳氷河期の画像5枚がある（作業服、赤ペン、競馬新聞）。クドウにそのまま使える素材。ただしクドウは「赤ペン」ではなく「缶コーヒーBLACK」が記号。缶を握り潰す癖を画で表現したい。
- ソラとの関係性: ソラの脆い直感とクドウの頑なな単勝主義。一見噛み合わないが、「根拠なんかなくても買う」という点で実は同じ側にいる。ナルミ的論理派とは対極。二人が同じ馬を買った回は、何も言わなくても"通じた"感がある。

**4. エミ（キバ版 KB-04）— 役割: テンポ崩し / 母性 / 空気のリセット装置**

- 推薦理由: ここが俺の選択のキモ。シュウはこの人を見逃した。67歳の元給食のおばちゃん。花柄カーディガン。おしゃべり。——バラエティ構成で一番大事なのは「論理の暴走を止められる人間」がいること。マキ・テツ・クドウ・ソラが予想論で白熱したときに、エミが「あらー、でもあの馬かわいかったわよねえ」と言う。一瞬全員の肩の力が抜ける。これがないと息が詰まる。「面白さ」には「緩む瞬間」が不可欠。笑いのメカニズムとして「緊張→弛緩」は基本中の基本。エミはその弛緩を一人で担える稀有なキャラ。
- 現状画像評価: 花柄カーディガンの67歳女性、3枚あり。おしゃべり感も出ている。悪くない。ただ「元給食のおばちゃん」らしさ——エプロンの名残とか、人にすぐ食べ物を勧める感じ——をもう少し足したい。
- ソラとの関係性: エミにとってソラは「孫みたいな子」。ソラにとってエミは「自分の予想を否定も肯定もせず、ただ受け止めてくれる存在」。マキが引き出し、テツが検証し、クドウが黙る中で、エミだけが「そうなの、すごいわねえ」と言う。この一言がソラの居場所になる。

**5. ダイチ（カイ版 K-10）— 役割: ダークサイド / 空気破壊者 / 劇薬**

- 推薦理由: 構成上、「ヤバい人」枠は絶対に要る。全員がいい人のドラマは退屈だ。ダイチは消費者金融に80万の借金があって、退職金を馬券に突っ込んで、ウーバーで食いつないでいる。「俺はプロだ」と言いながら食えていない。これが笑えるかと言うと——笑えてしまうのが怖い。ダイチがいることでkazemachiは「競馬の楽しい面」だけでなく「競馬の業」も描ける。構成として、ダイチが登場する回は必ず空気が変わる。毎回出す必要はない。年に数回、不意に現れて空気を壊す。それが一番効く。
- 現状画像評価: 画像未生成。※推測。作る場合は「電卓を叩く45歳男性、ウーバーのリュックが背景に見える」。目つきは鋭いが、どこか疲れている。「勝てる」と言っている口元と「負けている」と言っている目のギャップが欲しい。
- ソラとの関係性: ソラの直感が当たるのを見て、ダイチは焦る。「根拠がないのに当たる」のが、データで勝とうとしている自分の人生を全否定するから。でもソラを嫌いにはなれない。ソラの前でだけ「……すげえな」と本音が漏れる。

**6. カンノ（シュウ版 SH-12）— 役割: データツッコミ / 冷静な知性 / 若手の切れ味**

- 推薦理由: シュウが作ったキャラだが、構成的に必要なポジションだから推す。27歳データサイエンティスト。MacBook。コメダ珈琲。黒服。——この男は「空気を読まずにデータだけで斬る」枠。テツの展開語りに「でもそれ、数字的には矛盾してますよ」と平然と言える冷徹さ。ナルミ（K-02等）も論理派だが、ナルミは「正しさに誠実すぎて面白くなる」タイプで、つまりボケに転がる。カンノは「正しさで人を刺す」タイプで、ツッコミとして機能する。構成にはツッコミが要る。マキのツッコミは「対人」のツッコミ。カンノのツッコミは「対論理」のツッコミ。この二種類が揃うと会話の精度が上がる。画像も9枚と最多級で、ビジュアルの土台が強い。
- 現状画像評価: MacBook+コメダの構図が9枚。「今の若いやつ」感が出ていて良い。黒服が少しクールすぎるので、コメダのシロノワールを前に置くなどして「この男にも隙はある」を出したい。
- ソラとの関係性: カンノはソラの直感を「再現性のないデータ」として分析しようとする。でも分析すればするほど「説明できない」という結論に辿り着く。その困惑がカンノの人間味になる。ソラの存在がカンノの「データで全て説明できる」という信念を静かに壊していく。

### 【落選したが惜しかったキャラ】（3名）

**1. イイヅカ（シュウ版 SH-03）**

怒りと愛嬌の二面性は構成的にめちゃくちゃ美味しい。49歳内装業。一重の目。シュウが推した理由もよく分かる。ただ——テツと「テンション高い男」のポジションが被る。二人とも感情の振れ幅が大きいキャラを同じ場に置くと、視聴者の目が散る。「ひな壇に同じタイプは2人いらない」の原則。イイヅカは「ゲスト回で来て空気を一変させる」使い方が最も活きる。

**2. ユカリ（カイ版 K-09）**

「競馬をやめた女が、競馬の話だけ聞きに来ている」。この設定は構造としては最高級。クドウとの対比（やめた女 vs やめられない男）は書けるシーンが無限にある。ただ、レギュラーにするとテンポが落ちる。ユカリは基本「黙っている」キャラで、クドウも黙っている。沈黙が2人いると場が重くなりすぎる。年に2〜3回、不意に「それは違うと思う」と切り込む回をやる方がインパクトが出る。ゲスト枠の最上位。

**3. ミナト（カイ版 K-06）**

GoPro装着の23歳映像AD。SNS実況。「競馬をコンテンツとして消費する世代」の象徴。ゲンさんとの世代対立は画として面白い。ただ、カンノが若手知性枠に入った以上、若手が2人になるとバランスが崩れる。kazemachiの年齢構成は「中年多め」が自然。若手は1人で十分。カンノの「冷静さ」に対してミナトの「やかましさ」はキャラとして面白いが、今回は構成の密度を取った。ミナトは「ミナトのカメラ視点で1話まるごと撮る」特別回のために温存する。

### 【構成コメント】

**■ テンポ設計**

このメンバーの会話には「3段ギア」がある。

- **1速**（エミ・クドウ）: ゆっくり、間がある。場の温度を下げる。読者が息をつける
- **2速**（マキ・ソラ）: 通常テンポ。掛け合いの基本速度
- **3速**（テツ・カンノ）: 速い。論理の応酬。ここが盛り上がりのピーク

この3段を意識して脚本を組めば、1エピソードの中に「静→動→静」の波が自然に生まれる。テレビで言えば「CMまたぎ」のタイミングが分かりやすい構成。

**■ 対立構造で生まれる「笑い」**

- テツが5分間展開語りをして→カンノが「それ過去5年のデータと真逆ですけど」→テツ沈黙→エミ「あらあら」→マキ「はい次」。この流れで30秒のコントが成立する
- ダイチが「今日は仕上がってる」→全員の目が泳ぐ→クドウだけが「……ふーん」→ダイチ「おい、何だよその反応」→クドウ「別に」。ここに笑いと切なさが同居する
- エミがソラに「あなた今日のお昼ごはん食べた？」と聞く→予想の話が中断→テツ「今レースの話してるんですけど！」→エミ「だってお腹空いてたら直感も鈍るでしょ」→マキ「……一理あるな」。テンポの脱線と復帰が笑いになる

**■ 弱点と補強案**

1. **弱点: 記録者がいない**。シュウはフジさんを推したが、俺は外した。記録・アーカイブ機能が弱い。
   → **補強**: フジさん（K-05等）をゲスト枠で定期登場させる。月1回「フジさんのノート」回として、過去の振り返りエピソードを組む。レギュラーにしなくても機能する。

2. **弱点: ゲンさん不在**。63歳の重鎮ポジションがいない。エミは67歳だが「重鎮」ではなく「おかん」。
   → **補強**: ゲンさんは「年に数回だけ来る伝説の常連」として残す。登場するたびに「ゲンさんが来た」と場が緊張する演出。レギュラーにすると沈黙キャラが渋滞する（クドウがいるから）。

3. **弱点: 女性キャラのバリエーション**。マキ（36歳、場回し）とエミ（67歳、母性）の2人で、中間世代の女性がソラしかいない。
   → **補強**: サエ（K-07、28歳、血統派）かアカギ（KB-08、31歳、元JRAバイト）をサブレギュラーとして月2回程度投入。ソラと同世代の女性がもう1人いると、会話のバリエーションが広がる。

**■ シュウとの選択の違いについて**

シュウは「本当に競馬場にいそうな人間」を選んだ。リアリティ重視。その視点は正しい。

俺は違う。「この7人が画面にいるとき、何が起きるか」を選んだ。構成重視。

シュウのメンバーは全員が「静かに競馬を愛している人たち」で、画としては美しいが、テンポが均一になるリスクがある。全員が穏やかだと、対立が生まれにくい。

俺のメンバーには「暴走する人」（テツ）、「空気を壊す人」（ダイチ）、「場違いな人」（エミ）がいる。kazemachiが「居心地のいい場所」であるためには、逆説的に「居心地を壊す人間」が必要なんだ。壊されて、でも壊れなくて、だからこの場所は本物だと分かる——その構造。

最終的に決めるのはプロデューサーだが、構成のカイとしてはこの布陣を推す。
