## 1. キャラの感情面 — テキスト投稿で「声」を書く方法

### 問題の核心

ユキの強みは「声の描写」だった。「来い……来いッ！」の声量の変化、「あの子は悪くない！」の裏返り。でもPhase 1は静止画＋テキスト。声がない。

これは弱みじゃない。**制約が武器になる**。

### テキストで感情を伝える5つの技法

**① 括弧書きの使い分けで「声の層」を作る**

```
ナルミ「A馬です。期待値1.18。切る理由がない」
ナルミ（……あの子が出てるけど、関係ない。関係ないから）
ナルミ「以上です」
```

丸括弧は**心の声**。読者だけに聞こえる。番組では言えない本音をここに入れる。ラジオでは出せない「内面の描写」がテキストだからこそできる。声の代わりに「思考のレイヤー」を足す。

**② 「……」の長さで感情の深さを示す**

- `「……まあ、モデル上は正しかったので」` → 軽い動揺
- `「…………」` → 言葉が出ない
- `「　」` → 沈黙（空白のセリフ。存在だけが重い）

**③ 文末の処理で「強がり」と「本音」を切り替える**

- 強がり：「別に悔しくないですけど。」（句点が硬い）
- 本音：「別に悔しくないですけど」（句点を取ると語尾が消える＝声が小さくなる）
- 限界：「別に、」（途中で止まる＝言い切れない）

**④ ト書きで「身体の動き」を入れる**

```
ナルミ「想定通りです」
（スマホを裏返して置く）
```

声がない代わりに、**手の動き・視線・姿勢**を挟む。これがテキスト版の「演出」。読者は声を想像する——想像させた時点で勝ち。

**⑤ 画像とテキストの「ズレ」で感情を表現する**

一番やりたいのはこれ。

```
【画像】ナルミが冷静な顔でPCを見ている
【テキスト】「A馬、期待値Sランク。問題ありません」
（……A馬、出てくれてよかった。いや、関係ない。関係ないから）
```

画像は「建前の顔」。テキストは「本音まで含んだ全部」。この**ズレ**が読者の中でキャラを立体にする。「この子、顔は冷静だけど中身は違うんだ」——それを読者が自分で発見する快感。

## 2. 関係性の見せ方 — 最初のペア

### 結論：**データ派 × 進行役**で始める

理由を感情設計の視点から書く。

**ナルミ（データ派）× マキ（進行役）が最強のスタートペア：**

| 理由 | 説明 |
|---|---|
| **ギャップの導火線** | マキがナルミの建前を剥がす役割ができる。「推し馬ですか？」の一言でナルミの鎧が壊れる。この瞬間が番組の「売り」になる |
| **感情のダイナミクス** | ナルミは感情を隠すキャラ。マキは感情を読むキャラ。「隠す×読む」は関係性の中で一番テンションが生まれるペア |
| **2人だけの空気が作れる** | 他のキャラがいない段階で「ナルミとマキ、2人きりの予想TV」の空気を作れる。後からソラやゲンさんが加わったとき「空気が変わった」を感じさせられる |
| **マキの匂わせが自然に出る** | 予想家が1人しかいないと、マキが「整理」する必要がない。その余白で「ちなみに、このコースは——」の匂わせが出やすい |
| **ソラ合流のインパクトが大きくなる** | ナルミとマキの「静かなやり取り」に慣れた読者の前に、ソラが「C馬！ 顔！」で乱入する。テンションの落差でソラのキャラが一発で立つ |

**ソラ × ナルミ（天敵ペア）ではなく、ナルミ × マキから始める理由：**  
ソラは「相手がいて輝くキャラ」。ソラの良さはナルミとの掛け合いで出るが、ナルミがどういう人間かを読者が先に知っていないと、ソラのツッコミが空振りする。**ナルミを先に好きにさせる**のが正しい順番。

### Xでの関係性の育て方

**ステップ1（1〜4週目）：ナルミ単独アカウントで予想投稿**
- マキはナルミの投稿に「整理すると◯◯ですね」とリプライする形で登場
- マキ自身のアカウントは作らない。ナルミのリプ欄に"住んでいる"存在として始める
- **感情のポイント：** マキのリプがたまに鋭すぎて、ナルミが返信を1時間放置する回がある。放置の時間が、ナルミの動揺を物語る

**ステップ2（5〜8週目）：マキのアカウントが独立する**
- 番組公式的な投稿ではなく、**マキ個人の呟き**として始まる
- 「今週の成績、まとめました。全員見てください」（最初は誰も見ていない。フォロワーが少ない）
- **感情のポイント：** マキの投稿は感情がない。でもたまに、ナルミの投稿を引用RTして「いい予想ですね」とだけ書く。この一言の頻度が、マキの感情のバロメータになる

**ステップ3（9週目〜）：ソラ合流**
- ソラのアカウントが突然できる。初投稿は「C馬。理由？ 顔。以上！」
- ナルミの投稿に「Excelの人、また期待値の話してる」とリプが飛ぶ
- **感情のポイント：** ソラがナルミの投稿を引用RTするが、引用元のスクショにナルミのスプレッドシートが映り込んでいる。「見てないから！」のテキスト付き。読者が「見てるじゃん」とツッコむ導線

## 3. 画像を見た感想

### 5枚を見て思ったこと

キャラの内面が一番「想像しやすい」のは**3枚目**。

**3枚目を推す理由：**

あの半目・薄い笑み・散らかったクリップボード。この子を見たとき「有能なんだけど、それを見せるつもりがない」が伝わった。他の4枚は「いい子」「可愛い子」「しっかりした子」が先に来る。3枚目だけ「この子、何考えてるんだろう」が先に来る。

キャラの感情設計で一番大事なのは**「見た瞬間に内面を想像させること」**。  
きれいすぎる顔は想像の余地を奪う。少し崩れた表情、散らかった書類、力の抜けた姿勢——そこに「完璧じゃなさ」がある。これがマキ（進行役）の「平凡・頼りない系」と一番合う。

**2枚目も捨てがたい**。大きな目、鉛筆を耳に挟んでいる、クリップボードを見せてくる。「一生懸命だけどどこか抜けてる」感じ。これは「新人スタッフ感」が一番強い。ただし2枚目はキャラが「可愛い」に寄りすぎていて、マキの「たまに精度がおかしい」の怖さが出にくい。

**1枚目**は頼りなさと恥ずかしさのバランスがいいが、情報量が少なくて「この子がどういう番組の中にいるか」が見えにくい。

**4枚目**は明るすぎる。マキは笑わない——笑わないからこそ笑った瞬間が事件になるキャラ。デフォルトの表情は4枚目じゃない。

**5枚目**は有能感が出すぎている。手に持った書類の情報量は好きだけど、表情が「仕切ってる人」。マキは仕切っているが仕切っている自覚がないフリをしているキャラ。

### 表情差分の設計

Phase 1のX投稿で使う表情差分は**最低4パターン**。多くても6。

| 差分名 | 用途 | 感情の意図 |
|---|---|---|
| **通常顔** | 予想発表時、通常の投稿 | 感情ゼロ。これがデフォルト |
| **ギャップ顔** | 本性が漏れた瞬間 | レース中の「来いッ！」、推し馬バレ。**目が変わる** |
| **外れ顔** | 予想が外れた時 | キャラによって全然違う。ナルミは「硬直」、ソラは「頬膨らませ」、ゲンさんは「苦笑い」 |
| **やられた顔** | 進行役にツッコまれた/他キャラに論破された時 | 小さいダメージ表現。これが愛嬌を生む |

**進行役（マキ）だけ特殊：**
- 通常顔 = ほぼ無表情
- **口角0.5ミリ顔** = 他キャラが馬名間違えた時。微笑とも言えないレベル。でも差分として存在させる。これが「マキ、今笑った？」のネタになる
- **0.8秒の間顔** = 予想しない理由を訊かれた時。目だけが少し変わる。この差分は年に数回しか使わない

## 4. カイ・レイの案で気になったこと

### カイの案で良いと思ったもの

**対立のA〜E型ローテーション、これは感情と相性がいい。**

味方構図が毎週変わるということは、キャラの「この人をどう思っているか」が場面で揺れるということ。感情は固定されたら死ぬ。「ナルミはソラが嫌い」で固定したらダメで、「嫌いだけど今週だけ味方」の瞬間にキャラが生きる。カイのローテーションはその揺れを構造的に保証してくれる。採用。

**初回脚本の「夢は別料金です」級のワンライナー。**

ユキの弱みは「話が重くなりがち」。カイの短いセリフを差し込むことで、重くなりかけた空気を切れる。ユキが感情を深掘りしたところにカイのワンライナーが入ると「深い＋軽い」の最強コンボになる。

### カイの案で「キャラが死ぬ」と思ったもの

**テンポを優先しすぎると「沈黙」が消える。**

ナルミが外れて黙る3秒。ソラが拗ねて目が潤む2秒。マキが0.8秒の間を取る瞬間。これらは**テンポの「穴」**であって、カイの設計では潰されがち。

テンポが速い＝面白い、じゃない。**テンポが速い中に「穴」が一つあると、そこに感情が流れ込む。** カイのテンポ骨格は採用するが、「穴」の位置はユキが決めたい。

### レイの案で良いと思ったもの

**「ミクが予想を変えた回の的中率100%」。**

これは考察班向けだけど、キャラの感情面でも使える。ミク（ソラ）が予想を変える瞬間——それは**何かに心が動いた瞬間**。変える理由を本人も言語化できない。でもそのトリガーがわかったとき、ソラの直感派としての本質が見えてくる。レイの統計的伏線は、感情の伏線でもある。

**ユウ（マキ）の固有名詞メモ。**

マキがゲンさんの話の固有名詞を全部メモしている設定。表では「進行の準備」で通る。裏では「マキとゲンさんの過去」が匂う。でもユキ的に一番大事なのは、**マキがゲンさんの話を大切にしている**という感情。話を切るのは仕事。でも収録後に廊下で続きを聞く。メモを取る。この矛盾がマキの人間性。レイの仕掛けとユキの感情は共存できる。

### レイの案で「キャラが死ぬ」と思ったもの

**ARG展開の過度な埋め込み。**

レイはHTMLソースのコメントアウト、メタデータの座標、プロフィール画像の更新日時——などの仕掛けを提案していた。仕掛けとしては天才。でも**キャラがメディアのための装置になる危険がある。**

キャラはコンテンツの道具じゃない。キャラが生きているから物語が動く。ARGの仕掛けが先行すると「キャラの行動が伏線のために不自然になる」リスクが出る。

提案：ARGは**キャラの行動から自然に生まれるもの**に限定する。

- OK：ナルミが外れた翌日にSNSに意味不明の数列を投稿する → キャラとして自然（悔しくて再計算しているだけ）
- NG：画像のメタデータに座標を仕込む → キャラの行動ではない（制作側の仕掛け）

キャラが生きた結果として伏線が生まれるのはいい。伏線のためにキャラを動かすのはダメ。

## 5. その他

### 前回の自分の回答で深掘りしたいこと

**マキ（進行役）の「平凡・頼りない系」の設計。**

前回のマキは冷静すぎた。「容赦ないツッコミ」が先に立っていて、「平凡」が足りなかった。画像を見て思った——3枚目のあの子は**完璧じゃない**。散らかった書類を持って、少し疲れた目をしている。

マキの「たまに精度がおかしい」は、**普段が普通だからこそ際立つ**。普段のマキは台本を読んで、時間を管理して、成績を読み上げる——ただのスタッフ。その子がふと「このコースは内枠有利ですね」と言ったとき、ナルミが凍る。**普通の子が一瞬だけ普通じゃなくなる怖さ。** これが「平凡・頼りない系MC」の最大の武器。

前回のマキは有能すぎた。もっとダメなところを入れたい：
- **台本を落とす。** 拾っている間に3人が勝手に話し始めて、マキが追いつけない回がある
- **名前を間違える。** ナルミのことを「ケイさん」と呼んでしまう回がある（なぜ「ケイ」なのかは謎として残す——これはレイの裏レイヤーに使える）
- **成績発表で自分も照れる。** 全員外れた日、読み上げながら「えっと……全員……ゼロです……」と声が小さくなる

マキにも弱さがある。弱さがあるからこそ、鋭い一言が出たときに怖い。

### 名前について

ユキ版の名前（ナルミ・ソラ・ハジメ・マキ・テツ）は「呼びやすさ」で付けた。でも今思えば、もう一歩理由が欲しい。

**提案：**
- **ナルミ** → 残す。「鳴海」の字面が「海鳴り＝データのざわつき」に見える。本人はそんなつもりないけど
- **ソラ** → 残す。「空」。根拠が空っぽだから、じゃなくて、空を見上げるように馬を見ているから。……と本人は思っている
- **ハジメ** → **「ゲンさん」に統一したい。** カイ版・レイ版でも「ゲンさん」は共通して出てきた。このニックネームは全員が自然に使っていて、生きている名前だと思う
- **マキ** → 残す。ただし「マキ」が本名なのかニックネームなのかは曖昧にしたい。本名不明設定はレイの裏レイヤーと相性がいい
- **テツ** → 残す。「鉄」。ロマン派の名前に鉄がある矛盾が好き

### Phase 1で最初に投稿すべきコンテンツの形

**ユキの提案：**

第1投は「予想」ではなく**「自己紹介」**にする。

```
【画像】ナルミが冷静な顔でPCを見ている
【テキスト】
はじめまして。ナルミです。
毎週の競馬予想を、データで出します。
期待値がプラスなら買い。マイナスなら見送り。
感情は変数に入りません。

……入りません。

#予想TV
```

「入りません」を2回書くことで「入りそうなのを否定している」ニュアンスが出る。これがナルミの自己紹介。

第2投で初めて予想を出す。第3投で結果と振り返り。第4投あたりでマキがリプ欄に現れる。

**このペースが大事。** いきなり予想を始めると「競馬アカウント」で終わる。キャラを先に見せる。「この人が何を考えているか」を先に見せる。予想はその人格の延長線上にある。

それが**「予想家って根拠が面白い」の核心**。根拠が面白いのは、それを言っている人間が面白いから。人間を先に見せる。
