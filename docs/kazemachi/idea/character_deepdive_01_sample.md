## 2. 弱点の具体的エピソード（各キャラ3つ）

### レン

1. **モデル更新の夜に限って馬名が崩壊**  
   レース後、視聴者投稿の「あの馬の名前覚えてます？」に答えるコーナーで、**同一馬を別名で3回呼ぶ**。氷室が「同一個体です」と板につく。

2. **正しさで空気を凍らせる**  
   ミナミが負けた直後に「だから主観は」と言いかけ、**0.5秒沈黙してから謝罪**。謝るまでが長いので傷は残る。

3. **取消連鎖で心が折れる**  
   本命候補が連続取消の日、**「今日は指数を読み上げるだけにします」と予想を実質的に放棄**。シゲルがフォローで型の話を始めると、レンが「……お願いします」と頼る。

### ミナミ

1. **帽色の取り違えで大喧嘩しそうになる**  
   収録前、レンが出した「印の一覧」とミナミの頭の中の馬が**1頭ずれており**、本番で「ピンク推し！」と言った馬が実は別枠。氷室が写真で正す。

2. **外れ後の強がりでSNS炎上ギリ**  
   「私は見たものは見た」と言い切った直後に結果が確定。**カット後に泣きそうな声**になり、レンが無言でティッシュを置く（言葉はない）。

3. **シゲルの昔話に噛みついて脱線**  
   「若い頃もそうだった」と言われた瞬間に**10分前のレースの話を忘れて論破モード**に入り、氷室に「番組の時間、返してください」と言われる。

### シゲル

1. **勝者の物語に記憶が寄せられる**  
   「あの年も同じ脚質の名馬がいた」と語った馬が、後日スタッフ検証で**別年の別馬だった**。シゲルは笑って「記憶は編集者だね」と認めるが、レンが黙ってメモる。

2. **孫自慢でコーナーが潰れる寸前**  
   馬券の話の途中で「孫がね」と入り、**エピソードが完結するまで止まらない**。氷室が「今の子、騎手免許持ってます？」と一刀。

3. **SNS用語の誤爆**  
   「それ、バズるねえ」と言って**意味を取り違えている**。ミナミが優しく訂正し、シゲルが「バズは蜂かい？」と真顔で返して二度傷つく。

### 氷室

1. **鋭すぎてゲストが沈黙**  
   特集回で新人解説者が**質問を切られて言葉を失う**。オフで氷室だけが謝りに行くが、オンでは謝らない。

2. **本音が漏れてからの取り繕いが不自然**  
   下記セクション5のように**一瞬プロの目**が出る。取り繕いが早すぎて「隠してる感」が視聴者にバレる。

3. **自分の地雷を触られた時の低体温**  
   「なんで予想しないんですか？」と聞かれた瞬間、**一拍長い沈黙のあと「ルールです」だけ**。その日はミナミまで空気を読んで喋らない。

## 3. 関係性の温度感（ペア別）

### レン × ミナミ

| 場面 | 反応 |
|------|------|
| 二人だけ（進行役トイレ等） | ミナミが「レンさん、さっきの本命本気？」と聞き、レンが**スマホの画面を一瞬見せて**すぐ閉じる。ミナミ「チッ、ケチ」。互いに**本番では言わない情報**をちらっと共有している匂い。 |
| 片方3連敗（例: ミナミ） | レンは**数字では触れない**。「……次、分散下げます」と自分の話にすり替える。ミナミはわかっていて「優しいね、嫌味なしで」と突く。 |
| 一緒に馬券を買いに | レンが**買い目を言い、ミナミが別の子を足す**。結果、レンが「あなたのは感情」と言い、ミナミが「あなたのは説明書」と言う。**二人で負けた日が一番仲良い。** |

### レン × シゲル

| 場面 | 反応 |
|------|------|
| 二人だけ | シゲルが**昔のレース名を出さずに**「あの時、現場は静かだった」とだけ言う。レンが「……データには残ってます」と返し、シゲルが「残るものと、残すべきものは違う」と笑う。**師弟ではないが、教訓の授受がある。** |
| 片方3連敗（例: レン） | シゲルは**慰めない**。「君の夜は長いねえ」とだけ。レンが「承知です」と答える。**過保護にしない距離。** |
| 一緒に馬券を買いに | シゲルが現金と投票用紙に慣れすぎていて**レンより速い**。レンが焦って馬番を言い間違え、シゲルが「若いねえ」と直す。**レンが初めて年下扱いされる。** |

### ミナミ × シゲル

| 場面 | 反応 |
|------|------|
| 二人だけ | シゲルが「直感は歳を取ると濁る」と**自分を含めて**言う。ミナミが「シゲルさんは濁ってない、ただ長い」と返す。**同族嫌悪が薄れる瞬間。** |
| 片方3連敗（例: シゲル） | ミナミが**珍しく素直**に「今日は私が前出ます」と肩を代わる。シゲルが「頼むよ」と短く。**逆転の師弟みたいな1日。** |
| 一緒に馬券を買いに | ミナミがスマホ投票、シゲルが紙で並ぶ。**列が別レーン**になり、合流したとき「買ってる馬被ってた！」で握手。**世代ギャップが笑いに変わる。** |

### レン × 氷室

| 場面 | 反応 |
|------|------|
| 二人だけ | 氷室が**論点整理だけ**する。レンが「質問です、今のは誘導尋問ですか？」と真顔。氷室「はい。答えは不要」。**緊張は高いが、壊れない。** |
| 片方3連敗（レン） | 氷室はオンエアで**追撃しない**。「次のコーナーに価値がある」とだけ言い、オフで「定義、見直しな」と一言。 |
| 馬券（仮に同行したら） | 氷室は買わない。**レンが買うのを横で見て**「その買い方、性格出ますね」。レンが「性格は入力に入れてません」。 |

### ミナミ × 氷室

| 場面 | 反応 |
|------|------|
| 二人だけ | ミナミが「氷室さん、私ってちゃんと見えてます？」と聞く。氷室「見えすぎです」。ミナミが**一瞬ホッとして**から「それ嫌な褒め方」と笑う。 |
| 片方3連敗（ミナミ） | 氷室は**いつも通りツッコむ**が、最後に「次は顔以外も」と**実務的な宿題**を渡す。ミナミだけが「それ優しさ」と解釈する。 |
| 馬券 | 氷室は買わない。ミナミが「一口だけ！」と言い、氷室「私の一口は番組が持ちます」。 |

### シゲル × 氷室

| 場面 | 反応 |
|------|------|
| 二人だけ | **短い言葉で通じる**。「あの年」「分かる」「言わんとこう」。レンとミナミが戻ってきた瞬間、二人とも**丁寧語に戻る**。 |
| 片方3連敗（シゲル） | 氷室が「今日は長い話、禁止」と**ルールで守る**。シゲル「了解。老人は黙る」。 |
| 馬券 | シゲルが買う。氷室が「見せないで」と言い、シゲル「見せないよ、昔からね」。**過去の匂い。** |
