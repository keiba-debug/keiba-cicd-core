# 宿題②f — タイトル/コードネームのためのキーワード出し

> 各作家のシステムプロンプトの後ろに貼って投げる。

---

```
━━━━━━━━━━━━━━━━━━━━━
■ タイトル候補、良いのがたくさん出た。ありがとう
━━━━━━━━━━━━━━━━━━━━━

「競馬で生きてる」「いつかの馬の話」「競馬のあとで話そう」など、
どれもいい候補だった。

でもまだ決めない。
もう少し広く考えたい。

━━━━━━━━━━━━━━━━━━━━━
■ 今回の宿題：キーワードを自由に出す
━━━━━━━━━━━━━━━━━━━━━

タイトルやコードネームの「素材」になりそうな言葉を、
ジャンル問わず、自由に出してほしい。

【ルール】
- 競馬に紐づかない言葉でも全然いい
- むしろ競馬から離れた言葉の方が新鮮かもしれない
- 好きな漫画、小説、映画、音楽、詩から借りてもいい
- 1語でも、フレーズでも、文章でもいい
- 「なぜその言葉が気になったか」を一言添えてくれると助かる
- 数は多ければ多いほどいい。20個でも30個でも

【意識してほしいこと】

この作品の空気感を思い出してほしい：

- 毎週同じ場所に集まってしまう人たち
- 正解がないのに、毎回自分の答えを出す
- 外れても来週また来る
- 言葉にできない何かを感じて、それを信じる
- 誰かの隣にいること自体に意味がある
- 小さな変化が積み重なる
- 過去と今が重なる瞬間がある
- 話をしているうちに、その人の生き方が見えてくる

この空気感に触れる言葉を探してほしい。

【こういう方向もアリ】
- 場所の名前っぽいもの（カフェの名前、通りの名前、駅の名前みたいな語感）
- 動詞（「話す」「待つ」「集まる」「見る」「賭ける」「帰る」）
- 時間帯（「土曜の午後」「日曜の帰り道」「金曜の夜」）
- 感覚（「予感」「残像」「余韻」「手触り」）
- 関係性（「隣」「距離」「沈黙」「合図」）
- 好きな作品のタイトルや台詞で、この企画に通じるもの

自由に書いてくれ。
ここから何かが生まれるかもしれないし、生まれないかもしれない。
でも言葉は出しておきたい。
```
