# ビジュアルブラッシュアップ — 完成版プロンプト集

まず全体の設計意図を言っておく。

V1のプロンプトは「その人がどういう人か」を説明していた。V2で顔面スペックを足した。でもまだ足りなかった。今回のV3では **「この人の一番いい瞬間を切り取る」** ことに集中した。キャラクター紹介ではなく、1枚の写真として成立する画を作る。

SeeArt（SeedDream 4.5）の文字数制限を意識して、プロンプトは400字以内に収めた。スタイルリファレンスとしてソラの既存画像を添付する前提で書いている。

# 使用手順まとめ

## 生成手順

1. SeeArt（SeedDream 4.5）を開く
2. ソラの既存ベスト画像をスタイルリファレンスとして添付
3. 上記プロンプト（メイン）を貼り付け
4. 生成 → 3〜5枚生成して選定
5. 不満な場合はバリエーションAで別構図を試す

## 7人の差別化クロスチェック

生成したら、7人を横に並べて以下を確認：

| チェック項目 | 基準 |
|------------|------|
| 輪郭が全員違うか | ゲン/カズ(四角), フジ(面長), マキ(シャープ卵型), ノグチ(卵型), ジン(卵型穏), イイヅカ(四角ごつ) |
| 目が全員違うか | ゲン(奥二重細め), フジ(切れ長二重), カズ(奥二重腫れ), マキ(切れ長奥二重), ノグチ(奥二重落ち着き), ジン(一重寄り奥二重), イイヅカ(一重鋭い) |
| 年齢感が伝わるか | 63/51/49/34/38/32/49 — 全員が正しい年齢に見えるか |
| ソラと並べて全員別人か | 困り眉・伏し目・大きい瞳のソラと誰も被っていないか |

---

以上、7人分。プロンプトは全部貼り付けるだけで使える形にした。

ユキ原則の追記：**「この7人が並んだとき、1人1人の人生が見えるかどうか」が判断基準**。顔が違うだけでは足りない。その顔に、その人の25年や30年や49年が見えること。それができたら、このメンバーで行ける。
