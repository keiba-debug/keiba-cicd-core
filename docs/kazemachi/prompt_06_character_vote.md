# kazemachi プロンプト⑥ — キャラクターオーディション最終投票

> **正式名称**: メインキャスト決定投票
> **目的**: 76人分のキャラ素材から、kazemachiのメインキャスト（5〜7名）を決定する
> **前提**: ビジュアルレビュー（prompt_05）が完了していること

---

## プロンプト本文（全作家共通）

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 宿題⑥ — 最終オーディション投票
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

これまで集まった全キャラクター素材（76名＋ソラ固定）を全て読んだ上で、**kazemachiのメインキャストを選んでください**。

### 選定条件（重要度順）
1. **根拠の違い＝人格の違い**が最も豊かに表現できるか
2. **対立構造**を作りやすいか（特にソラとの相性）
3. **kazemachiという居場所に必然性がある**か
4. **ビジュアルの魅力**と設定の整合性
5. **物語としてのバランス**（年齢・性別・価値観・バックグラウンドの多様性）

### 投票形式
- **メインキャスト**: 5〜7名を選んでください（6名推奨）
- **役割も合わせて提案**してください
  - 入口役（マキ的なポジション）
  - 中心（ソラは固定）
  - 論理派
  - 経験派／沈黙派
  - 記録・記憶役
  - 若手／新世代
  - ダークサイド／外側からの目（任意）

### 出力してほしい項目

**【投票結果】**
- 選んだキャラ一覧（名前＋誰の回答から出たキャラか）
- 各キャラの役割と選んだ理由（特にソラとの関係性）

**【落選したが惜しかったキャラ】**（3名程度）
- 名前と惜しかった理由

**【最終的なコアメンバー構成案】**
- 6人編成の場合の理想的な組み合わせを1案提示
- 各人の役割と、期待される「化学反応」を記述

**【構成コメント】**（特にカイ・レイ向け）
- このメンバーだとどんな会話のテンポ・対立軸が生まれるか
- 弱点となりそうな点と、その補強案

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 出力ルール
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
- 冒頭に必ず【作家名】を付ける（例: 【カイ】）
- 必ず最後まで書ききること
- 推測がある場合は「※推測」と明記
- ソラは固定メンバーとして必ず含める
```

---

## 使い方

### 投票の進め方
1. `writers/` 内の全作家のシステムプロンプトを前置き
2. 上記プロンプトを投げる
3. 各作家の投票結果を回収
4. 多数決＋調整で最終コアメンバーを決定
