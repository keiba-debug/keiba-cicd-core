番組自体は競馬予想がメインですが、キャラクターのバックストーリーが

## 設計してほしいもの

### 1. シーズン1（全12回）のストーリーアーク
各回で明かされるバックストーリーの断片を設計してください。
- 第1回：顔合わせ。表面的な自己紹介
- 第4回あたり：最初の伏線が置かれる
- 第8回あたり：関係性に変化が起きる出来事
- 第12回：シーズン1の「あっ」という発見（でも全ては明かさない）

### 2. 各回に1つ入れる「匂わせ」リスト
12回分、毎回1つずつ入れるバックストーリーの断片。
視聴者が後から見返すと「あの時の一言はこういう意味だったのか」となるもの。

### 3. キャラクター成長の方向性
12回を通じて各キャラがどう変わるか（または変わらないか）。
予想スタイルの変化と、人間的な成長を分けて書いてください。

### 4. 「名場面」候補を5つ
視聴者が切り抜きたくなるような、印象的なシーンの設計。
予想が当たる/外れるだけでなく、キャラの関係性が動く瞬間。

### 5. シーズン2への布石
シーズン1の最終回で、シーズン2に向けた伏線を1つ仕込む。

## 制約
- 各回の予想パートは実際のレースデータに基づくので、ストーリーは予想結果に依存しない形で設計すること
- 「予想が当たったから仲直り」のような安易な展開は避ける
- キャラのバックストーリーは会話の中で自然に出る一言で匂わせる（ナレーションで説明しない）
