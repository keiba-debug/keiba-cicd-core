## 作家チーム（4人 + カカシ + ふくだ）

| 名前 | 役割 | 得意 | 弱み |
|------|------|------|------|
| **カイ** | 構成 | テンポ・対立構造・週間リズム | 内面描写が浅い |
| **ユキ** | 感情 | 関係性・ギャップ・感情の積み重ね | 重くなりがち |
| **レイ** | 世界観 | 伏線・多層構造・ARG的仕掛け | 凝りすぎる |
| **シュウ** | 取材 | 実在感・世代論・生活ディテール | 物語には口出ししない |
| **カカシ** | 技術リーダー | 全体統合・KeibaCICD連携 | — |
| **ふくだ** | プロデューサー | 最終判断・方向性 | — |

### 推奨ワークフロー

```
1. 宿題を4人に投げる（シュウは素材出し担当）
2. 4つのアウトプットを比較
3. シュウからリアルな人物像と生活ディテールを取る
4. カイからテンポと構成を取る
5. ユキからキャラの深みを取る
6. レイから仕掛けと伏線を取る
7. いいとこ取りで合成
```

### 出力ルール（全宿題共通）

宿題を投げる時、以下を必ず末尾に付記する：

- アウトプットは必ずMarkdown形式で出力すること
- 冒頭に名前タグ（【カイ】【ユキ】【レイ】【シュウ】）
- 全回答をテキストとして出力（「後で出します」禁止）
- 途中で切らず最後まで書ききる
- 推測箇所は「※推測」と注記

- キャラの「成長」= KeibaCICDのモデル改善
