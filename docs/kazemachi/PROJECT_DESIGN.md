## 1. プロジェクト概要

### 勝ち条件

「競馬×キャラクター×物語」で、作品として成立する世界を作り、
その世界に人が集まった結果として収益が生まれる状態。

- まず「作品」として面白いこと
- その結果として人が集まること
- その結果として収益が生まれること

**順序が大事。収益は結果であって目的ではない。**

### 一行コンセプト

**「競馬の話をしていたら、この6人の秘密が少しずつ見えてきた。」**

### 内部コードネーム

**kazemachi**（確定）
- 風待ち＝何かを待っている状態。まだ始まっていない
- 風街＝風が吹く街、人が集う場所。海街diary的な響き
- 漢字表記は揺らがせる。読み手に委ねる

### タイトル候補（寝かせ中）

| 候補 | 推し | 特徴 |
|------|------|------|
| **来週も隣で** | ユキ1位、全員TOP3入り | 関係と継続。唯一の全員TOP3 |
| **外れたまま、また来週** | レイ1位 | 行為と感情。二層構造に一致。的中自慢との差別化 |
| **常連の余韻** | カイ1位 | 場と結果。バランス最良 |
| 風街の帰り道 | レイ2位 | コードネームと連動 |
| 競馬で生きてる | カイ原案 | シンプル最強。広く届く |

### 作品の軸

**「根拠の違いが、そのまま人格の違いであること」**

競馬の予想における根拠——データ、直感、経験、記録、ロマン——は、
その人が何を信じ、何を怖がり、何を美しいと思うかの表れ。
競馬の話をすると、その人の生き方が見えてしまう。

## 2. 物語の構造

### 二層構造

| 層 | 内容 | 比率 | 機能 |
|---|---|---|---|
| **表の物語** | 小さな関係性の変化の積み重ね | 95% | 毎週の満足感、キャラへの愛着 |
| **裏の物語** | 「15年前のあの日」の秘密 | 5% | 長期の引力、考察班の喜び |

- 表の物語だけで十分楽しめる
- 裏の物語に気づいた人には全部がつながる
- **「謎が進むから関係が変わる」のではなく、「関係が変わったから謎の見え方が変わる」**（ユキ原則）

### 物語の定義

この企画における「物語」とは：

- 一本の大河ドラマではない
- **日常の中に、ちょっとした気まずさ、ちょっとした理解、ちょっとした救いが増えていくこと**
- 先週言えなかったことが、今週は少し言える
- 単発投稿のはずなのに、「前より近い」がある
- 半年後に見返すと変化がわかる

### 現実との接続（最大の独自性）

**実際のレース結果が物語を動かす。**

- フィクション連載は「確定した物語」を追うだけ
- この企画は「毎週現実が起きる」
- 現実のレースが、毎週のエピソードのオチになる
- 誰も結末を知らない。それが偶然なのか計算なのか、読者にはわからない

## 3. キャラクター設計

### 三層構造

```
マキ（入口）
  ↓ 読者が世界に入ってくる
ソラ（感情の中心）
  ↓ ソラを見ているうちに他のキャラも好きになる
フジさん（謎の中心）
  ↓ フジさんの謎が気になって離れられなくなる
```

### コアキャスト（6人）

#### マキ（進行役 / つなぎ役 ♀34）

- **役割**: コミュニティの主催者。場を作り、人を迎え、空気を整える
- **核心**: 「平凡だけど、みんなが集まってくる」
- **人格**: MCではなく「迎える人」。優しいだけではない——場を守りたい人は少しだけコントロールしたい人でもある
- **記号**: 書類、クリップボード
- **ビジュアル**: 書類まみれで半笑い（ベース確定済み）

#### ソラ（直感派 ♀26）

- **役割**: 感情の中心。見ていたい存在。周囲を映す鏡
- **核心**: 「合ってるかな…」（脆い直感）
- **人格**: 根拠のない直感に毎週身を晒す。外れた時が一番美しい。少し足りないまま、でも本気で見ている
- **守るべきもの**:
  - 根拠の不完全さ（全部をうまく説明できない方がいい）
  - 外れた時の美しさ（悔しがり方、黙り方、次にまた見ようとする姿勢）
  - 他人の言葉に揺れる柔らかさ
- **引力の本質**: 恋愛ではなく「放っておけなさ、話を最後まで聞きたさ、見失いたくなさ」
- **記号**: 双眼鏡、色ペンだらけの出馬表（赤=第一直感、青=比較、緑=説明できない印）
- **ビジュアル**: 黒無造作ボブ、くすんだ青灰シャツ、汚れたジャケット（ほぼ確定）

#### ナルミ（データ派 ♂30）

- **役割**: 冷たい論理。数字で自分を安心させたい人
- **核心**: 「数字は嘘をつかない」（冷たい確信）
- **人格**: 数字の人でありながら、数字に出ないものに揺れる瞬間が物語になる
- **記号**: PC、スプレッドシート、眼鏡
- **ビジュアル**: 紺カーディガン、寒色系（設計済み、未生成）

#### ゲンさん（経験派 ♂60代前半）

- **役割**: 蓄積された経験。40年分の記憶。過去を道具として使う人
- **核心**: 「この展開、前にも見た」（静かな既視感）
- **人格**: 笑っている口元と、笑っていない目。引退した刑事が競馬場に通っている感じ
- **記号**: 手書きノート（赤丸が1つだけ）、スマホ裏向き、眼鏡おでこ
- **ビジュアル**: グレーグリーンのチェックシャツ、ゴツい体格（v3プロンプト作成済み）

#### テツ（ロマン派）

- **役割**: 過去とロマン。馬やレースに自分の感情を重ねる人
- **核心**: 「あのレースを思い出している」（遠い過去）
- **人格**: 勝敗ではなく、馬やレースに重ねる感情そのものがコンテンツ
- **記号**: 古い馬券
- **ビジュアル**: 暗色コート、1人で立つ（設計済み、未生成）

#### フジさん（記録係 ♀48）

- **役割**: 数字と秩序。深層の重み。全員の成績を最も長く見てきた人
- **核心**: 記録することで世界を整理したい。実は一番知っている可能性がある
- **人格**: 淡々。無言の記録係だが、たまに一言で全員を止める。「昔はもっと喋る人だった」
- **記号**: タブレット+スタイラス（成績管理）
- **ビジュアル**: チャコール/紺のカーディガン、ティール/スレート青の1色（プロンプト作成済み）

### ゲストキャラクター

**コア6人は「場」として固定。ゲストが不定期に訪れる。**

- **頻度**: 月1〜2回（特別回、テーマ回に合わせる）
- **役割**: コア6人の「普段見えない面」を引き出す触媒。謎への無自覚な情報提供者
- **設計原則**:
  - ゲストは主役ではなく「コア6人の輪郭を変える光」
  - ゲストが直接謎を語るのではなく、ゲストの言葉へのコア6人の反応が謎になる
  - 1人に必ず「この人にしかない1つのこと」を設計する
  - ゲストが去った後の「コア6人だけの余韻」を必ず入れる

#### ゲスト候補リスト

| ゲスト | 概要 | 引き出すもの |
|--------|------|-------------|
| 初めて来た大学生 | 読者の分身。素朴な質問が鋭い | 全員の「なぜ競馬をやるか」 |
| ソラと同じ馬を推したおばさん | ソラの直感が「共鳴」として返る | ソラの孤独が少し薄くなる |
| ゲンさんの30年ぶりの飲み仲間 | 昔話が「まだ終わっていない話」に変わる | ゲンさんの過去と現在のズレ |
| フジさんの前職の後輩 | 「昔は予想もしてたんですよね」 | フジさんの沈黙が「経た結果の静けさ」に見える |
| ナルミを批判するSNS論客 | 知識はあるが言い方が嫌な人 | ナルミの防御反応、テツの仲裁力 |
| テツが毎週見かける男 | 話したことがない。ある日だけ声をかける | 競馬場の孤独と連帯 |
| ソラの元同級生 | 「ソラって昔からこうだったよね」 | ソラの過去と現在をつなぐ |

## 4. コンテンツ設計

### 週間リズム

| 曜日 | 温度 | 内容 |
|---|---|---|
| 月 | 低 | 余韻、振り返り、今週の入口 |
| 火 | 中 | グルメ、旅、馬の話、雑談 |
| 水 | 中 | キャラの趣味や生活、パドック |
| 木 | 上がる | 今週の注目馬、視点が噛み合い始める |
| 金 | 高 | 予想前夜、全員の視点が揃う |
| 土日 | 最大 | 予想、実況、結果、感情の爆発 |

**「日常で好きにさせて、週末で信用させる」**（カイ）

### コンテンツの比率

- **予想 3〜4 : 関係性と生活 6〜7**
- 予想回は濃くする。頻度は少なめでも感情のピークを担う
- 平日は「競馬のある生活」。すべての投稿に競馬との接点が必要

### コンテンツ種別

#### 毎日（X投稿）
- 1枚絵+一言
- キャラ同士の短いやり取り
- パドック、馬、グルメ、収支、小ネタ

#### 毎週（X + ショート動画）
- 金曜：今週の論点整理
- 土曜：予想前の会話
- 日曜：実況 or 結果後の反応
- 週明け：振り返り

#### 月1（Note + 漫画）
- Note長文（ゲンさんの昔話完全版、フジさんの講座、マキの日記）
- 漫画（世界観の定着装置）
- テーマ回（依存、名馬、血統、初心者講座）

#### ショート動画（インスタ / TikTok / YouTube Shorts）
- 30秒〜1分。キャラ×物語×競馬を切り取る
- ソラのパドック感想、ゲンさんの昔話、フジさんの一言収支報告
- 新規入口として最強。アルゴリズムがフォロワー0でもリーチを出す

### 平日コンテンツ案

| 企画名 | 内容 | 担当キャラ |
|--------|------|-----------|
| 今日の競馬場メシ | 食べ物写真+一言 | ソラ中心、ゲンさん脱線 |
| パドックで何を見る？ | 同じ写真に各キャラが別コメント | 全員 |
| フジさんの収支ノート | 淡々とした数字の話 | フジさん |
| ゲンさんの昔話 | 名馬、消えた血統、地方の競馬場 | ゲンさん |
| 帰り道の一言 | レース後の気持ちを一言だけ | 各キャラ |
| マキの初心者ガイド | そもそも馬券ってどう買うの？ | マキ |
| 馬の話だけする日 | 美しさ、癖、歩き方 | ソラ、テツ |

## 5. 「15年前のあの日」の設計ルール

### 位置づけ

- 物語の前面ではなく**関係性の地下水脈**
- 主菜ではなく出汁
- 全投稿が伏線に見える状態にはしない
- 月に1〜2回、妙に引っかかる発言や一致が起きる程度

### ルール

1. 表の連載はあくまで日常群像劇
2. 特定条件（同じコース、同じ脚質、特定の天候）の時だけ深層が反応する
3. 反応しても全部は説明しない（「何かを思い出したんだな」で止める）
4. 真相の前進は関係性の前進と同期させる
5. 秘密は答えではなく、現在の関係を歪ませている圧として使う
6. 伏線はキャラの行動の結果として自然に生まれるもの。伏線のためにキャラを動かさない（ユキ原則）

### 全キャラの「あの日」への関与

| キャラ | 関与の仕方 |
|--------|-----------|
| ゲンさん | 一番長く知っている。記憶の語り部 |
| テツ | 別視点で知っている。詩的に断片を語る |
| ナルミ | データで検証しようとしている |
| ソラ | 感覚で覚えている（自覚なし） |
| フジさん | 全員の記録を持っている。実は一番知っている可能性 |
| マキ | 何を知っているか不明。視聴者の目線代理 |

## 6. 絶対に守ること

1. **「根拠が面白い」が核** — 的中率ではなく「なぜその馬を選んだか」
2. **外れ予想は消さない** — 外れた時にこそその人が出る
3. **仲良しにしすぎない** — 見方の違い、言い方の違い、温度差を残す
4. **すべての投稿に競馬との接点** — ただの雑談にしない
5. **KeibaCICDのチラ見せは自然な埋め込み** — 前面に出しすぎない
6. **ソラの脆さを守る** — 勝ち続けるソラ、説明が上手いソラにしない

## 7. 絶対にやらないこと

1. 最初から壮大な本筋を背負わせること
2. 予想精度を前面に出しすぎること（外れた時に人格まで下がる）
3. 全員を仲良くしすぎること
4. 平日投稿を単なる雑談にすること
5. 謎のためにキャラを動かすこと
6. ソラの引力を恋愛に回収すること
7. 予想を売ること（売るのは「この世界に居たい権利」）

## 9. 画像テイスト

- **画風**: 日常アニメ系。スキップとローファー〜ふつうの軽音部のライン
- **線**: スケッチ、手描きペン、はみ出し、不完全
- **色**: 低彩度、フラット、ウォーターカラーウォッシュ
- **彩度ルール**: 各キャラ1色だけ鮮やかな色を持つ（ソラ=出馬表の色ペン、ゲンさん=ノートの赤丸）
- **背景**: 白背景（キャラ単体）、競馬場（シーン）
- **統一感**: 全キャラが同じアーティストに描かれたように見えること
- **生成ツール**: SeeArt (SeedDream 4.0)
