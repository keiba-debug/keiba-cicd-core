## Web UI 改善

### レース詳細画面にJRDB由来データ統合

- JRDB SED/KYI由来のデータ（前走IDM、総合指数等）
- レース画面で市場情報との比較が可能に

- 過去走ごとのIDM推移
- JRDB独自指標のトレンド表示
- 能力変化の可視化

### IDM比較画面の拡張
- **コース/距離フィルタ**: 同一コース条件のIDMだけを抽出して比較
- **ペースフィルタ**: Hペース/Mペース/Sペース別のIDM傾向
- **IDM比較 → 過去同一重賞比較**: 例えば弥生賞のIDM推移を過去5年分重ねて「今年のメンバーレベル」を把握
  - X軸: 各年のレース日、Y軸: IDM → 年ごとの線で比較
  - 同一レース名でdata3から過去開催を自動検索

### ラップ比較画面（IDM比較のアーキテクチャ転用）
- **同一コース過去レースのラップ重ね比較**: X軸=ハロン区間、Y軸=ラップタイム
- 過去の弥生賞ラップと今年の想定ペースを重ねてレースイメージを掴む
- 凡例ON/OFF + ハイライト連動はIDM比較と同一パターン
- JRDB SED通過順位データとの連携で馬別ラップも可能に

### レース比較ビューアー（コース別レース検索 + 映像連動）
- **同一コース条件のレースを一覧比較**: コース/距離/馬場/クラスでフィルタ → カード形式で一覧表示
- 各カードに表示する情報:
  - ラップタイム（前半3F/後半3F）+ ラップグラフ
  - ペース分析（H/M/S + RPCI）
  - レースランクレベル（IDM平均/ARd平均でメンバーの質を数値化）
  - 着順Top3 + 脚質 + 着差
  - 通過順位（JRDB SED由来）
- **レーシングビューアー一括再生**: 選択したレースの映像を連続再生（JRA公式リンク or ローカル映像連携）
- **活用シーン**:
  - 「中山1200m外A 良」の過去10Rを並べて逃げ/差し有利の傾向を把握
  - 同一重賞の過去5年分を比較してメンバーレベル・ペース傾向を分析
  - 特定馬の同コース過去走を並べて成長/衰退を映像で確認
- 既存のラップ比較画面・IDM比較画面のアーキテクチャを転用可能

### 予測確信度 (Prediction Confidence)
- レースごとに「モデルがどれだけ自信を持っているか」を数値化
- 活用例:
  - 確信度低いレースは見送り推奨 → 無駄な馬券購入を回避
  - 確信度高いレースに資金を集中 → バンクロール効率の最大化
  - Bank画面に確信度カラム追加、フィルタ条件として使用
- 指標候補:
  - P/Wモデルの予測確率の分散（Top1とTop2の差が大きい = 確信度高）
  - 出走馬の過去走データ充実度（初昇級馬が多い = 確信度低）
  - ARd標準偏差（馬間の差が小さい = 確信度低）
  - 同一クラス・条件の過去モデル精度（未勝利→1勝クラス昇級戦は精度低い傾向）
- 背景: 前走未勝利勝ち上がり組ばかりのレースなど、比較データが乏しいケースでモデルの限界を明示する武器になる
