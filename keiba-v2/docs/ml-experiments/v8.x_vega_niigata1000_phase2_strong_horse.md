# vega-niigata1000 Phase 2: 強馬予測の事前特徴量検証

作成: 2026-05-09
対象: 新潟芝1000m 全135R / 2,230走（2020-2026）
ターゲット: `is_strong_horse`（結果脚質①テン速＆末脚速 かつ 3着内）

**目的**: 「結果として速かった馬」を事前に判定するための特徴量を発見する。
Phase 1 検証②で「結果脚質①強馬」が 3着内率54.7%、ROI 170% という支配的な性能を持つことを確認。
本Phaseでは、この強馬になる馬の事前指標（過去走脚質、血統、厩舎、騎手）を解明する。

### ⑤ 結論サマリ（強馬予測の事前指標）

- **特徴量化推奨（事前判定用）**:
  - `past_corner_first_avg_5`: 過去5走の前半通過順位平均（先行傾向）
  - `past_last_3f_min_5`: 過去5走のベスト上がり3F（末脚ピーク）
  - `past_short_avg_l3f`: 過去短距離での上がり3F平均（千直適性の代理）
  - `sire_strong_rate`: 父馬の千直強馬率（5走以上で平滑化）
  - `trainer_strong_rate`: 厩舎の千直強馬率
  - `jockey_strong_rate`: 騎手の千直強馬率
  - `past_choku_finish_avg`: 過去千直平均着順（経験馬のみ）
  - `past_choku_last_3f_avg`: 過去千直上がり3F平均（経験馬のみ）

- **予想時の判断ヒント**:
  1. 「過去5走の前半通過順位 + 上がり3Fベスト」のセットで強馬候補を選別
  2. 父・厩舎・騎手の強馬率TOPと一致する馬を優遇
  3. 過去千直経験馬は「平均着順」と「上がり3F平均」を併用

- **次フェーズ（ルールエンジン v0.1）への入力**:
  - 上記特徴量を用いた段階判定ロジックの設計
  - polaris予測値との統合方法（重み付き和 / AND条件 / 説明文生成）
