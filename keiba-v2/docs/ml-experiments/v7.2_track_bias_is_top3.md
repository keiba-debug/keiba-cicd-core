## is_top3修正

JRA複勝ルールに従い、出走頭数で3着内判定を変更:
- 8頭以上: 3着以内 = 的中
- 5-7頭: 2着以内 = 的中
- 4頭以下: 2着以内 = 的中

修正箇所: experiment.py の2箇所 (血統PIT統計 + 特徴量/ターゲット計算)
