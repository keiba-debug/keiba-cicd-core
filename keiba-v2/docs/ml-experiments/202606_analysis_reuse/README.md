# 202606 分析系機能 強化・再利用案

このフォルダは、`202606_*.md` で作成した分析レビューを  
「実装可能な強化案」として継続育成するための作業ハブです。

## 目的

- 分析ページの知見を、ML再学習とは別に運用改善へ接続する
- 買い目最終判断のプラスオプションを、段階導入で検証する
- レース表画面の表示改善を、実装しやすい粒度に分解する

## 運用方針

- 元の `202606_*.md` は履歴として残す（原本保全）
- このフォルダで「横断統合」「優先順位」「実装チケット化」を進める
- 採否は AUC だけでなく `gap>=k` 条件の ROI / CI / 件数で判断する

## ドキュメント構成

- `01_master_index.md` : 元レビューと再利用テーマの索引
- `02_enhancement_backlog.md` : 実装バックログ（優先度・完了条件）
- `03_change_log.md` : 更新履歴（いつ/何を/なぜ変えたか）
