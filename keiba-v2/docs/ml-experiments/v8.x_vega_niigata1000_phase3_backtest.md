# Phase 3c バックテスト結果: vega-niigata1000 RuleEngine v0.2

実施日: 2026-05-09
評価期間: 2020-2025 (新潟芝1000m直線、135R / 2,230走)
比較対象: polaris単独 vs polaris + rule (v0_2)
モデル: polaris v7.1 (`pred_proba_p`, predictions.json から取得)
ルール: `rules/v0_2.yaml` (28ルール)

## サマリー

| 指標 | polaris単独 | polaris+rule | 差分 |
|---|---:|---:|---:|
| top1 勝率 | 33.3% (45/135) | **40.0% (54/135)** | **+6.7pt** |
| top1 単勝ROI | 146.4% | **155.6%** | +9.2pt |
| top3 ヒット率 (race-level) | 97.8% | 94.8% | -3.0pt |
| top3 単勝ROI | 147.2% | 123.7% | -23.5pt |

**結論**: ルール適用で **本命1点軸の精度・ROI が改善** (45勝→54勝、+9勝)。一方、3頭買い時の総ROI は polaris単独に劣後。

## 詳細

### 1. 本命単勝の改善

ルール適用により top1 (推奨1頭) の的中数が +9勝 (45→54)。
- 1点 ¥100 で 135R 投資 → polaris単独 ¥19,762 / polaris+rule **¥21,000** (+¥1,238)
- ROI も +9.2pt と一貫

### 2. 3頭買いでの ROI 後退

top3 ROI は 147% → 124% に低下。原因仮説:
- top3 hit_rate (1頭でも入る率) が 97.8% → 94.8% (-3.0pt)。3着内 0頭のレース が +4 増えた
- ルールが 1頭目を強く再ランクするものの、2-3頭目の選択は polaris の方が安定
- 本命1点を狙う運用なら +、3頭ボックスを狙うなら polaris単独優位

### 3. 除外推奨 (STEP F) の精度

| 区分 | 馬数 | top3率 |
|---|---:|---:|
| 除外推奨 (is_rejected) | 75 | **10.7%** |
| 非除外 | 2,155 | 18.4% |

除外推奨馬の top3率 10.7% は非除外 18.4% より **7.7pt 低い** → STEP F は妥当に飛ぶ馬を抽出できている。
ただし 10.7% (約9頭中1頭) はゼロではないので「絶対除外」ではなく「割引」運用が現実的。

## 注意点 (バイアス)

- **lookahead bias**: polaris の `pred_proba_p` は 2020-2025 全期間で訓練済モデルの予測。過去レースには間接的にデータが含まれる。
  - polaris単独 / polaris+rule **両方に同じバイアス** → 相対比較は妥当。絶対値は楽観的。
- **千直の特殊性**: 18頭立てで 3着内 = 3頭。素朴 3頭買いでも 16.7% は当たる。本評価の top3_hit_rate (race-level) は 95%超で **飽和**気味。
- **F 単独のROI差**: 除外運用に切り替えた場合の収支インパクトは別途要検証。
