# v7.3 dam_top3_rate除外実験

> Session 84 | 2026-03-04

## 背景

v7.2 の特徴量重要度分析で `dam_top3_rate`（母産駒複勝率）が全体の **22.7%** を占め、
2位の `prev_finish`（5.6%）の4倍の支配的重要度を持つことが判明。

### dam_top3_rate のリーク問題

1. **時間軸リーク**: 2020年の訓練データに対して2025年2月までの母産駒成績を使用
   - PIT（point-in-time）補正は無効化されていた
   - コメント: "Sire stats are population-level, not individual-level leakage"
   - 実際には dam は産駒数が少なく（中央値13走）、個馬の影響が大きい
2. **自己参照リーク**: 対象馬自身の成績が母の統計に含まれている
   - 産駒数が少ないほど影響大（例: 産駒5頭で3着内3回 → top3_rate=60%）
3. **jrdb_jockey_idx（Session 81）と同パターン**: 1特徴量が支配 → 他が学習不足

### 母馬統計の分布（13,109母馬）

- 産駒出走数: mean=18.7, median=13.0, max=149
- 3走以下: 17.3%（統計的に不安定）
- 5走以下: 27.1%
- 10走以上: 58.4%（やっとstd=0.094で安定）

## 変更内容

- `PEDIGREE_FEATURES` から `dam_top3_rate` を除外（170→169特徴量）
- 他のdam特徴量（maturity_index, fresh_advantage, tight_penalty等）は維持
  - これらは「傾向」を表し、直接的な成績率よりリスク小

## 結論

- **dam_top3_rate はリーク特徴量**: 除外でTest AUC +0.04~+0.07、ROI +21pt
- **Session 81のjrdb_jockey_idx除外と同パターン**: 支配的特徴量のリーク除外で汎化性向上
- **教訓**: sire_stats_index のdam統計は産駒数が少なく、PIT補正なしではリークリスク大
- 今後: 他のdam_*特徴量も将来的にPIT化を検討（現時点では重要度が低いので影響小）

## 学習事項

- **★★★★ 1特徴量が22%超 → リーク疑え**: dam_top3_rate除外でAUC+0.07、Win ROI+21pt
- **★★★ 血統PIT未対応のdam統計はリーク源**: 産駒数中央値13走で自己参照+時間軸リーク
- **★★ Val AUC低下+Test AUC上昇 = 過学習削減**: リーク特徴量は見かけの適合度を上げるが汎化しない
