# v5.11 母馬(dam)統計量追加

> Session 54 | 2026-02-26
> Status: **DEPLOYED** - dam特徴量で全モデル大幅AUC改善

## 背景

### Session 53の発見: 母馬効果 > 父馬効果

v5.8/v5.9で「sire_top3_rateがModel B #1 (227,370)」と報告していたが、
UM_DATAオフセットバグにより実際には**母馬(dam)の産駒成績**を
sireとして集計していたことが判明。

v5.10（オフセット修正後）では正しいsire/bmsデータでAUC改善は控えめだった。
しかし、バグデータで母馬効果の強さが偶然証明されたため、
**意図的にdam統計量を追加する実験**を実施。

### 母馬効果が強い理由

- 遺伝: 50%（父馬と同等）
- 子宮環境: 母体の健康・栄養状態
- 育成環境: 出生後の初期発育

父馬は遺伝50%のみ。母馬はそれに加え環境要因も寄与するため、
**産駒成績に対する予測力が高い**。

## 教訓

1. **母馬効果は父馬効果の10倍以上**: Model Bでdam_top3_rate=234K vs sire/bms圏外
2. **バグから正しい発見へ**: v5.8のオフセットバグが母馬効果の発見につながった
3. **ベイズ平滑化が高カーディナリティを制御**: 14K種類の母馬でもprior meanへの収縮が効く
4. **環境要因の重要性**: 遺伝だけでなく子宮環境・育成環境が産駒成績に大きく影響

## 注意点

### Point-in-Time リーク

sire_stats_indexは全期間(2020-2026)で集計されており、
テスト期間(2025-2026)のデータも含まれる。
damは産駒数が少ないため、sire/bmsよりリーク影響が大きい可能性がある。
今後Optuna + PiT化で検証予定。
