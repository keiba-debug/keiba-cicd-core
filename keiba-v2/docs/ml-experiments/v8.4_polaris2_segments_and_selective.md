### 4. 戦略原則の定立 (Phase 4.3 で確立)
**「市場と一致するシグナルは ROI を下げる、乖離するシグナルこそ収益源」**
- Sel_v2 (モデル信頼度高フィルタ = 市場一致) → ROI 半減 (203% → 108%)
- Sel_v3 (vb_gap フィルタ = モデル vs 市場の乖離) → ROI 倍増 (203% → 381%)
- → Session 119「視点独立性は ROI 源泉」原則がモデル内部でも観測

## その他の発見

- **polaris 2.0 の苦手領域**: 新馬・1勝クラス (1,229 bets / ROI 76-83%) — 特化モデル候補 #1
- **重賞は予想外に得意**: G1 ROI +385%、G2 +118%、G3 +265%、Listed +120% (当初仮説と逆)
- **W単独は Sharpe 30.88 だが累計 -¥37,770** — Sharpe の罠 (単独判定NG)
- **Hybrid-Concur 失敗**: P==W 一致レースは「市場でも当たり前」な本命に集中 → ROI 低下
