## 2. 現在の設計の限界

### 推定指数（現在の能力R）が持つ根本的な問題

```
現在のability_score → 能力R は
「過去の実績パターンから推定した現在の能力水準」

= 「今この馬はどのくらい走れるか」の推定値
```

しかし以下の情報が完全に欠落している。

```
同じ 能力R = 60 の馬でも:

  ケースA: 30戦・8歳馬
    → 既にキャリアのピークを過ぎている
    → 今後の成長は見込めない
    → 能力R通りかそれ以下の走りが大半

  ケースB: 5戦・3歳馬・初重賞挑戦
    → 成長曲線の上昇途中
    → 潜在能力はまだ引き出されていない
    → 能力R以上の走りが十分にあり得る
    → オッズは低い実績を反映 = VB候補
```

**市場はこの差を正確に評価できていない。**  
これが構造的な市場の歪みであり、AIが優位に立てる領域。

## 3. 将来ビジョン：2種類の指数

### 指数①: 推定指数（Estimated Ability Rating）

```
定義: 過去の実績から推定した現在の能力水準
単位: レーティングスケール（KeibaBook互換）
現在の実装: ability_score → 能力R（Phase A-6）

高い = 実績が豊富で安定した能力
低い = 実績が低い or 条件が合っていない
```

### 指数②: 限界突破指数（Breakthrough Potential Index）

```
定義: 潜在能力と現在の発揮率のギャップ
     「この馬はまだ伸びしろがあるか」

高い = 未知数・期待値が大きい・成長途中
低い = 既に能力を出し切っている・ピーク過ぎ

計算イメージ:
  推定上限能力（血統・牧場・厩舎から推定）
  - 現在の推定指数
  = 限界突破指数
```

### 2軸での表示イメージ

```
馬名: ○○○○
  能力R（推定）: 58.2  ← 現在の水準
  限界R（潜在）: 64.0  ← 推定上限
  伸びしろ:      +5.8  ← 限界突破指数

解釈:
  「現在58だがポテンシャルは64まである馬」
  「市場オッズは58水準 = 明確なVB候補」
```

## 4. 成長曲線モデルの設計

### 必要な学習要素

```
Input（馬の背景情報）:
  血統
    父の成熟パターン（早熟/晩成）
    母父の影響（スタミナ/スピード系統）

  生産牧場
    牧場別の成長カーブの傾向
    育成方針（早期仕上げ vs じっくり育成）

  厩舎
    調教師の育て方のパターン
    使い方（たたき台 vs 仕上げ型）

  個体履歴
    出走数（経験値）
    デビュー時期・年齢
    成績変化のトレンド（上昇/安定/下降）
    前走着差の推移
```

### 成長フェーズの分類

```
┌──────────────────────────────────────────┐
│  上昇期  │  全盛期  │  安定期  │  下降期  │
└──────────────────────────────────────────┘

上昇期:
  出走数が少ない、年齢が若い
  能力Rが走るたびに改善している
  限界突破指数: 高

全盛期:
  能力Rがピーク付近で安定
  パフォーマンスが一定
  限界突破指数: 低〜中

安定期:
  能力Rは維持されているが向上しない
  条件次第で変動
  限界突破指数: 低

下降期:
  能力Rが徐々に低下
  高齢・消耗の影響
  限界突破指数: 極低
```

## 6. VB判断への組み込み

### 現在（Phase A）

```
VB条件: gap >= 6 かつ ability >= -1.2（能力R >= 56）
→ 市場乖離 × 現在の能力水準
```

### 将来（Phase D以降）

```
VB条件:
  gap >= 6                    （市場乖離）
  かつ 能力R >= 56             （現在の能力水準）
  かつ 限界突破指数 >= threshold （伸びしろ）

→ 「市場が気づいていない成長途中の馬」を
  定量的に検出できる
```

### 期待される効果

```
現在の課題:
  gap条件は「市場の見落とし」を捉えるが
  「なぜ市場が見落としているか」は分からない

限界突破指数追加後:
  「経験が浅く過去実績が低いが潜在能力は高い」
  という構造的な市場の歪みを明示的に捉えられる
  → VBの質が向上
  → 特に若馬・昇級馬・条件変更馬でROI改善が期待
```
