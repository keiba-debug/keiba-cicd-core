## 凡例

| 項目 | 説明 |
|:---|:---|
| **基準値** | そのコースの平均的なPCI3値。この値を「M（平均ペース）」の中心とする |
| **H境界** | ハイペース判定の閾値。この値未満でH判定 |
| **S境界** | スローペース判定の閾値。この値以上でS判定 |
| **n** | 算出に使用したレース数（サンプル数） |
| **-** | 未算出（データ収集中） |

### ペース分類記号

| 記号 | 意味 | 条件 |
|:---:|:---|:---|
| **H** | ハイペース | PCI3 < H境界 |
| **h** | やや前傾 | H境界 ≤ PCI3 < 基準値-1.4 |
| **M** | 平均ペース | 基準値±1.4 |
| **s** | ややスロー | 基準値+1.4 ≤ PCI3 < S境界 |
| **S** | スローペース | PCI3 ≥ S境界 |

#### 📊 中山ダート1200m 含水率別分析

**分析対象**: 383レース

##### 含水率区分別PCI3

| 含水率 | レース数 | PCI3平均 | 標準偏差 | 傾向 |
|:---:|:---:|:---:|:---:|:---:|
| 0-5% | 178 | 42.99 | 2.34 | 平均 |
| **5-10%** | 131 | **42.53** | 2.83 | **最もハイ傾向** |
| 10-15% | 69 | 43.15 | 2.27 | ややスロー |
| 15-20% | 5 | 45.29 | 1.67 | スロー傾向 |

##### 含水率細分化（2%刻み）

| **4-6%** | 52 | **42.35** | ハイ傾向 |
| **6-8%** | 62 | **42.20** | **最もハイ** |
| 8-10% | 38 | 43.52 | |

| 12-14% | 15 | 42.63 | |

##### 馬場状態別PCI3

| 馬場 | レース数 | PCI3平均 | 標準偏差 |
|:---:|:---:|:---:|:---:|
| 良 | 246 | 42.96 | 2.39 |
| **稍重** | 79 | **42.51** | 2.94 |
| 重 | 36 | 43.02 | 2.60 |
| 不良 | 22 | 43.25 | 2.07 |

##### 💡 考察

1. **含水率 4-8%（5-10%区分）が最もハイペースになりやすい**
   - 乾きすぎず、重すぎない適度な状態でペースが上がる
   
2. **含水率 15%以上でスロー傾向が顕著**
   - PCI3平均: 45.29（基準値42.4から+2.9）
   - 馬場が重くなり、逃げ・先行馬が抑えるためか

3. **馬場状態では「稍重」が最もハイペース**
   - 適度な湿り気がスピードを出しやすい状態

4. **含水率によるPCI差: 2.76** → ペースへの影響は「やや影響あり」レベル
